<p><b>生物分類學</b>（英語：<span lang="en">biotaxonomy</span>）通常直接稱<b>分類学</b>（英語：<span lang="en">taxonomy</span>；法語：<span lang="fr">taxonomie</span>；西班牙語：<span lang="es">taxonomía</span>），是关于生物个体的鉴定（identification）、归类（classification）和命名（nomenclature）的原理和方法的学科；此三要件间有相互依存的关系。生物分類學範圍包括：研究<a href="/wiki/%E7%94%9F%E7%89%A9" title="生物">生物</a><a class="mw-redirect" href="/wiki/%E9%A1%9E%E7%BE%A4" title="類群">类群</a>间的异同程度，阐明生物间的亲缘关系、演化过程和发展规律。<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>
</p><p>要將生物分類，首先要知道<a href="/wiki/%E7%94%9F%E7%89%A9" title="生物">生物</a>與非生物的定義，但是我們似乎沒有辦法準確定義，以病毒來說，雖然可在其他生物體內寄生並複製，但在生物體外卻沒有一般生物的特徵如製造或攝取營養，<a class="mw-redirect" href="/wiki/%E7%94%9F%E6%AE%96" title="生殖">生殖</a>等現象。又如引起<a class="mw-redirect" href="/wiki/%E7%98%8B%E7%89%9B%E7%97%85" title="瘋牛病">瘋牛病</a>的<a class="mw-redirect" href="/wiki/%E6%9C%8A%E7%B2%92" title="朊粒">朊粒</a>（prion）可以造成感染卻無<a class="mw-redirect" href="/wiki/DNA" title="DNA">DNA</a>成分，一直以來，DNA被視為生命<a href="/wiki/%E9%81%BA%E5%82%B3%E7%89%A9%E8%B3%AA" title="遺傳物質">遺傳物質</a>，經由與<a class="mw-redirect" href="/wiki/RNA" title="RNA">RNA</a>的<a class="mw-redirect" href="/wiki/%E8%BD%89%E9%8C%84" title="轉錄">轉錄</a><a class="mw-redirect" href="/wiki/%E8%BD%89%E8%AD%AF" title="轉譯">轉譯</a>過程，形成<a class="mw-redirect" href="/wiki/%E8%9B%8B%E7%99%BD%E8%B3%AA" title="蛋白質">蛋白質</a>，再進一步形成組成細胞的各個部分，如<a class="mw-redirect" href="/wiki/%E7%B4%B0%E8%83%9E%E8%86%9C" title="細胞膜">細胞膜</a>、<a class="mw-redirect" href="/wiki/%E8%83%9E%E5%99%A8" title="胞器">胞器</a>等，而<a class="mw-redirect" href="/wiki/%E7%B4%B0%E8%83%9E" title="細胞">細胞</a>則是我們長久以來所認為組成生命體的最小單位。
</p>

<p>关于如何为生物命名的原则有很多国际协议，例如《<a class="mw-redirect" href="/wiki/%E5%9C%8B%E9%9A%9B%E6%A4%8D%E7%89%A9%E5%91%BD%E5%90%8D%E6%B3%95%E8%A6%8F" title="國際植物命名法規">国际植物命名法规</a>》（<span lang="en">International Code of Botanical Nomenclature</span>，简称ICBN）、《<a href="/wiki/%E5%9B%BD%E9%99%85%E5%8A%A8%E7%89%A9%E5%91%BD%E5%90%8D%E6%B3%95%E8%A7%84" title="国际动物命名法规">国际动物命名法规</a>》（International Code of Zoological Nomenclature，简称ICZN）以及《<a class="new" href="/w/index.php?title=%E5%9B%BD%E9%99%85%E7%BB%86%E8%8F%8C%E5%91%BD%E5%90%8D%E6%B3%95%E8%A7%84&amp;action=edit&amp;redlink=1" title="国际细菌命名法规（页面不存在）">国际细菌命名法规</a>》（International Code of Nomenclature of Bacteria，简称ICNB）。第四版的生物命名法规（BioCode）草案在2000年出版，它试图在5个领域标准化命名，但现在还没有被正式采纳。《<a class="new" href="/w/index.php?title=%E5%9B%BD%E9%99%85%E7%97%85%E6%AF%92%E5%91%BD%E5%90%8D%E5%92%8C%E5%88%86%E7%B1%BB%E6%B3%95%E8%A7%84&amp;action=edit&amp;redlink=1" title="国际病毒命名和分类法规（页面不存在）">国际病毒命名和分类法规</a>》（International Code of Virus Classification and Nomenclature，简称ICVCN）是属于生物命名法规的。
</p>
<h2><span id=".E7.94.9F.E7.89.A9.E5.88.86.E7.95.8C.28kingdom.29.E5.92.8C.E5.9F.9F.28domain.29"></span><span class="mw-headline" id="生物分界(kingdom)和域(domain)">生物分界(kingdom)和域(domain)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%E7%94%9F%E7%89%A9%E5%88%86%E7%B1%BB%E5%AD%A6&amp;action=edit&amp;section=2" title="编辑章节：生物分界(kingdom)和域(domain)">编辑</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>传统上，生物分为五界(kingdom)，
</p>
<ul><li><a href="/wiki/%E5%8E%9F%E6%A0%B8%E7%94%9F%E7%89%A9%E7%95%8C" title="原核生物界">原核生物界</a>(bacteria) -- <a class="mw-redirect" href="/wiki/%E5%8E%9F%E7%94%9F%E7%94%9F%E7%89%A9%E7%95%8C" title="原生生物界">原生生物界</a>(protista) -- <a class="mw-redirect" href="/wiki/%E7%9C%9F%E8%8F%8C%E7%95%8C" title="真菌界">真菌界</a>(fungus) -- <a class="mw-redirect" href="/wiki/%E6%A4%8D%E7%89%A9%E7%95%8C" title="植物界">植物界</a>(plant) -- <a class="mw-redirect" href="/wiki/%E5%8A%A8%E7%89%A9%E7%95%8C" title="动物界">动物界</a>(animal)</li></ul>
<p>Copeland提出过四界说：
</p>
<ul><li><a href="/wiki/%E5%8E%9F%E6%A0%B8%E7%94%9F%E7%89%A9%E7%95%8C" title="原核生物界">原核生物界</a> -- <a class="mw-redirect" href="/wiki/%E5%8E%9F%E7%94%9F%E7%94%9F%E7%89%A9%E7%95%8C" title="原生生物界">原生生物界</a> -- <a class="mw-redirect" href="/wiki/%E6%A4%8D%E7%89%A9%E7%95%8C" title="植物界">植物界</a> -- <a class="mw-redirect" href="/wiki/%E5%8A%A8%E7%89%A9%E7%95%8C" title="动物界">动物界</a></li></ul>
<p>也有人使用<a href="/wiki/%E4%B8%89%E5%9F%9F%E7%B3%BB%E7%B5%B1" title="三域系統">三域系統</a>说。这种分类方法是依據细胞是否有<a href="/wiki/%E7%BB%86%E8%83%9E%E6%A0%B8" title="细胞核">核</a>以及<a href="/wiki/%E7%BB%86%E8%83%9E%E8%86%9C" title="细胞膜">细胞膜</a>和<a class="mw-redirect" href="/wiki/%E7%BB%86%E8%83%9E%E5%A3%81" title="细胞壁">细胞壁</a>的差异。
</p>
<ul><li><a class="mw-redirect" href="/wiki/%E5%8F%A4%E7%B4%B0%E8%8F%8C" title="古細菌">古菌域</a>(archaea) -- <a class="mw-redirect" href="/wiki/%E7%B4%B0%E8%8F%8C" title="細菌">细菌域</a>(bacteria) -- <a href="/wiki/%E7%9C%9F%E6%A0%B8%E7%94%9F%E7%89%A9" title="真核生物">真核域</a>(eukaryota)</li></ul>
<p>另有一些不属于上述所有分类的介于生物和非生物之间的生命形式，即只能<a href="/wiki/%E5%AF%84%E7%94%9F" title="寄生">寄生</a>于细胞内的<a href="/wiki/%E7%97%85%E6%AF%92" title="病毒">病毒</a>，而它们在细胞外并不表现出活跃的生命形式。
</p>
