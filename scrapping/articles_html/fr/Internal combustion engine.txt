<p>Un <b>moteur à combustion interne</b> ou <b>MCI</b> (en <a href="/wiki/Anglais" title="Anglais">anglais</a> : <span class="lang-en" lang="en"><i>internal combustion engine</i></span> ou ICE) est un type de <span class="page_h"><a class="mw-disambig" href="/wiki/Moteur_%C3%A0_combustion" title="Moteur à combustion">moteur à combustion</a></span>, c'est-à-dire un <a href="/wiki/Moteur" title="Moteur">moteur</a> permettant d'obtenir un travail mécanique à partir d'un gaz en surpression, cette dernière étant obtenue à l'aide d'un processus de <a href="/wiki/Combustion" title="Combustion">combustion</a>. Dans le cas d'un moteur à combustion interne, cette combustion a lieu à l'intérieur du moteur.
</p><p>Il existe deux grands types de moteurs à combustion interne : les moteurs produisant un <a href="/wiki/Couple_(physique)" title="Couple (physique)">couple</a> sur un <a href="/wiki/Arbre_(m%C3%A9canique)" title="Arbre (mécanique)">arbre mécanique</a> et les <a href="/wiki/Moteur_%C3%A0_r%C3%A9action" title="Moteur à réaction">moteurs à réaction</a> éjectant rapidement un <a href="/wiki/Fluide_(mati%C3%A8re)" title="Fluide (matière)">fluide</a> par une <a href="/wiki/Tuy%C3%A8re" title="Tuyère">tuyère</a>.
</p>

<li>moteur à allumage par compression : <a href="/wiki/Moteur_Diesel" title="Moteur Diesel">moteur Diesel</a> et <a href="/wiki/Moteur_%C3%A0_boule_chaude" title="Moteur à boule chaude">semi-diesel</a>, à deux temps ou à quatre temps ;</li>
<li><a href="/wiki/Moteur_%C3%A0_gaz" title="Moteur à gaz">moteur à gaz</a>. Voir aussi <a href="/wiki/Gazog%C3%A8ne" title="Gazogène">gazogène</a>, <a href="/wiki/GPL_carburant" title="GPL carburant">gaz de pétrole liquéfié</a>, <a href="/wiki/Gaz_naturel_pour_v%C3%A9hicules" title="Gaz naturel pour véhicules">gaz naturel pour véhicules</a> (GNV).</li></ul>

</p>
<h2><span id="Moteurs_.C3.A0_r.C3.A9action_.28fournissant_une_pouss.C3.A9e.29"></span><span class="mw-headline" id="Moteurs_à_réaction_(fournissant_une_poussée)">Moteurs à réaction (fournissant une poussée)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Moteur_%C3%A0_combustion_interne&amp;veaction=edit&amp;section=2" title="Modifier la section : Moteurs à réaction (fournissant une poussée)">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Moteur_%C3%A0_combustion_interne&amp;action=edit&amp;section=2" title="Modifier la section : Moteurs à réaction (fournissant une poussée)">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a href="/wiki/Moteur_%C3%A0_r%C3%A9action" title="Moteur à réaction">Moteur à réaction</a>.</div></div>
<p>Selon le type de <a href="/wiki/Comburant" title="Comburant">comburant</a> utilisé, il existe deux types de moteur à réaction :
</p>
<ul><li><a href="/wiki/A%C3%A9robie" title="Aérobie">aérobies</a> (dont le comburant est le <a href="/wiki/Dioxyg%C3%A8ne" title="Dioxygène">dioxygène</a> de l'<a href="/wiki/Air" title="Air">air</a>) :
<ul><li><a href="/wiki/Turbor%C3%A9acteur" title="Turboréacteur">turboréacteur</a>,</li>
<li><a href="/wiki/Stator%C3%A9acteur" title="Statoréacteur">statoréacteur</a>,</li>
<li><a href="/wiki/Superstator%C3%A9acteur" title="Superstatoréacteur">superstatoréacteur</a>,</li>
<li><a href="/wiki/Pulsor%C3%A9acteur" title="Pulsoréacteur">pulsoréacteur</a> et <a href="/wiki/Moteur_%C3%A0_ondes_de_d%C3%A9tonation_puls%C3%A9es" title="Moteur à ondes de détonation pulsées">moteur à ondes de détonation pulsées</a> ;</li></ul></li>
<li><a href="/wiki/Ana%C3%A9robie" title="Anaérobie">anaérobies</a> (utilisation d'un comburant embarqué) :
<ul><li><a href="/wiki/Moteur-fus%C3%A9e" title="Moteur-fusée">moteur-fusée</a> dont <a href="/wiki/Moteur-fus%C3%A9e_%C3%A0_ergols_liquides" title="Moteur-fusée à ergols liquides">moteur-fusée à ergols liquides</a>.</li></ul></li></ul>
