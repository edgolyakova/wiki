<div id="contentSub"></div>
<div class="mw-body-content mw-content-ltr" dir="ltr" id="mw-content-text" lang="fr"><div class="mw-parser-output"><p>Une <b>idéologie</b> est un système prédéfini d'idées, appelées aussi catégories, à partir desquelles la <a href="/wiki/R%C3%A9alit%C3%A9" title="Réalité">réalité</a> est analysée, par opposition à une connaissance intuitive de la réalité sensible perçue. De tels systèmes considérés comme idéologiques existent dans les domaines politique, social, économique, culturel et religieux. Une idéologie est souvent la dimension culturelle d'une <a class="mw-redirect" href="/wiki/Institution_(sociologie)" title="Institution (sociologie)">institution sociale</a> ou d'un système de pouvoir. Une idéologie <i>dominante</i> est diffuse et omniprésente, mais généralement invisible pour celle ou celui qui la partage, du fait même que cette idéologie fonde la <a href="/wiki/Weltanschauung" title="Weltanschauung">façon de voir le monde</a>.
</p><p>On peut distinguer dans une idéologie les dimensions :
</p>
<ul><li><a href="/wiki/Cognition" title="Cognition">cognitive</a> : <a href="/wiki/Dogme" title="Dogme">dogmes</a>, <a href="/wiki/Croyance" title="Croyance">croyances</a> (<span class="citation">« c'est ainsi »</span>) ;</li>
<li><a href="/wiki/Morale" title="Morale">morale</a> : <a href="/wiki/Jugement_(philosophie)" title="Jugement (philosophie)">jugements</a>, <a href="/wiki/Valeur_morale" title="Valeur morale">valeurs</a> (<span class="citation">« c'est bien ; c'est mal »</span>) ;</li>
<li><a href="/wiki/%C3%89thique_normative" title="Éthique normative">normative</a> : <a class="extiw" href="https://fr.wiktionary.org/wiki/norme" title="wikt:norme">normes</a> (<span class="citation">« il faut ; on doit »</span>).</li></ul>
<p>À l'origine, le terme d'idéologie fut créé par <a href="/wiki/Antoine_Destutt_de_Tracy" title="Antoine Destutt de Tracy">Destutt de Tracy</a> pour tenter de fonder une discipline qui étudie les idées pour elles-mêmes (la <a href="/wiki/M%C3%A9m%C3%A9tique" title="Mémétique">mémétique</a>, si ses axiomes sont corrects, pourrait être une branche ou dimension de cette étude). Mais ce sens s'est perdu en faveur de la notion de système d'idées doctrinaire. Le terme tend à prendre un sens de plus en plus large, et est parfois employé pour sa seule connotation péjorative en vue de dénigrer une école de pensée adverse, qu'elle soit ou non dogmatique.
</p>

<h2><span class="mw-headline" id="Origines">Origines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=1" title="Modifier la section : Origines">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=1" title="Modifier la section : Origines">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".C3.89tymologie"></span><span class="mw-headline" id="Étymologie">Étymologie</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=2" title="Modifier la section : Étymologie">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=2" title="Modifier la section : Étymologie">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Du <a href="/wiki/Grec_ancien" title="Grec ancien">grec ancien</a> <i>ἰδέα</i> (<i>idea</i>), « idée », et de <i>λόγος</i> (<i>logos</i>), « science, discours ». L'idéologie est donc, étymologiquement, un <a href="/wiki/Discours" title="Discours">discours</a> sur les idées. En grec ancien, le nom <i>ἰδέα</i> apparenté au verbe ἰ<i>δεῖν</i>, « voir », suggèrerait plutôt le sens d'« image ».  L'idéologie est communément interprétée comme :
</p>
<ul><li>la logique d'une idée par rapport à sa contrainte ;</li>
<li>la logique d'une vision ;</li>
<li>la logique d'une image développée pour la <a href="/wiki/Pens%C3%A9e_de_groupe" title="Pensée de groupe">pensée de groupe</a>.</li></ul>
<h3><span class="mw-headline" id="Terme">Terme</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=3" title="Modifier la section : Terme">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=3" title="Modifier la section : Terme">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Logique (du <a href="/wiki/Grec_ancien" title="Grec ancien">grec ancien</a> : λόγος) : c'est à la fois le <a href="/wiki/Discours" title="Discours">discours</a> et la <a href="/wiki/Rh%C3%A9torique" title="Rhétorique">rhétorique</a> de l'homme, animal politique selon <a href="/wiki/Aristote" title="Aristote">Aristote</a><sup class="reference" id="cite_ref-Aristote,_La_Politique_1-0"><a href="#cite_note-Aristote,_La_Politique-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup> ; grâce à la parole et donc à la <a href="/wiki/Rh%C3%A9torique" title="Rhétorique">rhétorique</a>, l'homme n'est plus un animal comme les autres<sup class="reference" id="cite_ref-Aristote,_La_Politique_1-1"><a href="#cite_note-Aristote,_La_Politique-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>. Dans l'idéologie, il s'agit d'une logique par les mots, d'un discours, c'est-à-dire de rhétorique incluant la <a class="mw-redirect" href="/wiki/Logique_fallacieuse" title="Logique fallacieuse">logique fallacieuse</a> ; depuis la logique dépasse le <a href="/wiki/Langage_humain" title="Langage humain">langage humain</a>. (voir <a href="/wiki/Langage" title="Langage">Langage</a>, <a href="/wiki/Logique" title="Logique">Logique</a> et <a href="/wiki/V%C3%A9rit%C3%A9" title="Vérité">Vérité</a>)
</p><p>Le terme d'idéologie apparaît à la fin du <abbr class="abbr" title="18ᵉ siècle"><span class="romain">XVIII</span><sup style="font-size:72%">e</sup></abbr> siècle : il fut forgé en 1796 par <a href="/wiki/Antoine_Destutt_de_Tracy" title="Antoine Destutt de Tracy">Destutt de Tracy</a> (<i>Mémoire sur la faculté de penser</i>), pour désigner l'étude des idées, de leur caractère, de leur origine et de leurs lois, ainsi que leurs rapports avec les signes qui les expriment.
</p><p>Dans la continuation des <a href="/wiki/Lumi%C3%A8res_(philosophie)" title="Lumières (philosophie)">Lumières</a>, les <a href="/wiki/Soci%C3%A9t%C3%A9_des_id%C3%A9ologues" title="Société des idéologues">idéologues</a>, groupe animé par <a href="/wiki/Antoine_Destutt_de_Tracy" title="Antoine Destutt de Tracy">Destutt de Tracy</a>, (<a href="/wiki/Pierre_Jean_Georges_Cabanis" title="Pierre Jean Georges Cabanis">Cabanis</a>, <a href="/wiki/Volney" title="Volney">Volney</a>, <a class="mw-redirect" href="/wiki/Dominique-Joseph_Garat" title="Dominique-Joseph Garat">Garat</a>, <a href="/wiki/Pierre_Daunou" title="Pierre Daunou">Daunou</a>), voulaient instaurer une science des idées. Leur projet était de traiter les idées comme des phénomènes naturels exprimant la relation de l'homme avec son milieu naturel de vie. Dans la lignée du <a href="/wiki/Sensualisme" title="Sensualisme">sensualisme</a> de <a href="/wiki/%C3%89tienne_Bonnot_de_Condillac" title="Étienne Bonnot de Condillac">Condillac</a>, qui cherchait déjà l'origine des idées, ils voulaient faire une analyse scientifique de la pensée. L'idéologie est alors considérée comme un système de pensée cohérent et naturel, indépendant de son cadre historique.
</p><p>Cependant, selon <a href="/wiki/Georges_Canguilhem" title="Georges Canguilhem">Georges Canguilhem</a>, ces idéologues étaient des <a href="/wiki/Positivisme" title="Positivisme">positivistes</a> avant la lettre, <a href="/wiki/Lib%C3%A9ralisme" title="Libéralisme">libéraux</a>, <a href="/wiki/Th%C3%A9ologie" title="Théologie">anti-théologiens</a>, et <a href="/wiki/M%C3%A9taphysique" title="Métaphysique">anti-métaphysiciens</a>, cherchant à dissiper les mythes et l'<a href="/wiki/Obscurantisme" title="Obscurantisme">obscurantisme</a>. Ils ont cru en <a href="/wiki/Napol%C3%A9on_Ier" title="Napoléon Ier">Bonaparte</a> comme continuateur de la <a href="/wiki/R%C3%A9volution_fran%C3%A7aise" title="Révolution française">Révolution Française</a>, pour devenir anti-napoléoniens. Et c'est Napoléon <abbr class="abbr" title="premier">I<sup>er</sup></abbr> qui a renversé leur image, au nom du réalisme politique et social, en les dénonçant eux-mêmes comme des métaphysiciens à la pensée creuse<sup class="reference" id="cite_ref-:0_2-0"><a href="#cite_note-:0-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>Cette problématique accompagne tout le <a href="/wiki/XIXe_si%C3%A8cle" title="XIXe siècle"><abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle</a>, en parallèle au déploiement de la pensée scientifique et à la <a href="/wiki/R%C3%A9volution_industrielle" title="Révolution industrielle">Révolution industrielle</a>. Ce qui guide les penseurs, c'est la recherche d'un système global et cohérent, qui s'articule autour de l'application des lois scientifiques aux <a href="/wiki/Sociologie" title="Sociologie">phénomènes sociaux</a>. Dans certains cas, la polarisation se fait entre <a class="mw-redirect" href="/wiki/Id%C3%A9ologie_scientifique" title="Idéologie scientifique">idéologie scientifique</a> et <a href="/wiki/Croyance" title="Croyance">croyances</a> religieuses.
</p><p>Au <abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle, <a href="/wiki/Karl_Marx" title="Karl Marx">Marx</a> propose de cesser de considérer l'idéologie comme un système neutre et donne un éclairage critique au concept originel de l'idéologie de l'époque : il voit l'utilisation de l'idéologie comme un système d'opinions servant les intérêts de <a href="/wiki/Classe_sociale" title="Classe sociale">classes sociales</a> (voir <a href="#Analyse_marxiste">Analyse marxiste</a>). Il reprend le concept du renversement du rapport de la connaissance à la chose. L'idéologie désigne désormais un système d'idées issue d'une situation qui méconnaît son véritable rapport au réel<sup class="reference" id="cite_ref-:0_2-1"><a href="#cite_note-:0-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>
Selon Georges Canguilhem :</p><blockquote><p>« La fortune, aujourd'hui, de la notion d'idéologie a des origines non douteuses. Elle tient à la vulgarisation de Karl Marx. Idéologie est un concept épistémologique à fonction polémique, appliqué à ces systèmes de représentations qui s'expriment dans la langue de la politique, de la religion et de la métaphysique. Ces langues se donnent pour l'expression de ce que sont les choses mêmes, alors qu'elles sont des moyens de protection et de défense d'une situation, c'est-à-dire d'un système de rapports des hommes entre eux et des hommes aux choses.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>»</p></blockquote><p>À ce propos, Canguilhem pose la question « Qu'est-ce qu'une idéologie scientifique ? » et de sa pertinence en épistémologie et histoire des sciences<sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>.
</p><h3><span class="mw-headline" id="Acceptions_actuelles">Acceptions actuelles</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=4" title="Modifier la section : Acceptions actuelles">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=4" title="Modifier la section : Acceptions actuelles">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Une idéologie est un ensemble d'idées sur la structure de la société, sur les forces qui agissent dans la société, sur les sources de conflit qui y sont présentes, et aussi sur les modalités qui permettent de résoudre ces conflits, ensemble d'idées partagées par un groupe, communément appelé parti politique. Une définition dérivée de l'idéologie est celle d'une doctrine <a href="/wiki/Politique" title="Politique">politique</a> qui fournit un principe unique à l'explication du réel. Celle-ci est susceptible d'inspirer rapidement un programme d'action et constitue un ensemble cohérent d'idées imposées et parfois acceptées sans réflexion critique et sans discernement. L'idéologie offre des notions beaucoup plus larges que celles des doctrines qui sont la dimension intellectualisée d'une idée imaginée. Les doctrines font appel à la dimension culturelle des « <a href="/wiki/Psychologie" title="Psychologie">comportements psychologiques</a> » et s'inscrivent dans un processus collectif important : la notion d'idéal remplace alors l'idéologie en encadrant une « société de masses ».
</p><p>L'idéologie peut être vue sous l'angle <a href="/wiki/Sociologie" title="Sociologie">sociologique</a> : l'idéologie a été définie par <a href="/wiki/Guy_Rocher" title="Guy Rocher">Guy Rocher</a> comme un <span class="citation">« système d'<a href="/wiki/Id%C3%A9e" title="Idée">idées</a> et de <a href="/wiki/Jugement_(philosophie)" title="Jugement (philosophie)">jugements</a>, explicite et généralement organisé, qui sert à décrire, expliquer, interpréter ou justifier la situation d'un groupe ou d'une collectivité et qui, s'inspirant largement de valeurs, propose une orientation précise à l'action historique de ce groupe ou de cette collectivité »</span><sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>. Un autre auteur, <a href="/wiki/Jean_Baechler" title="Jean Baechler">Jean Baechler</a>, donne cependant une définition plus fine et plus complète de l'idéologie.
</p>
<ol><li>Au départ, l’idéologie est l’ensemble des représentations mentales qui apparaissent dès lors que des hommes nouent entre eux des liens, des associations.</li>
<li>Ces représentations forment ensuite un ensemble d’états de la conscience liés à l’action politique, autrement dit à la façon conflictuelle ou non dont les humains organisent leur vie sociale. Le noyau de ces états de conscience est non verbal, c’est-à-dire composé de pulsions affectives ; ces états idéels s’actualisent dans différents types de registre et peuvent être inférés à partir des manifestations objectives et matérielles auxquelles ils donnent lieu.</li>
<li>L’idéologie se trouve dans le contenu et non dans le contenant. Il n’existe pas de genre discursif qui puisse être décrété idéologique en tant que tel.</li>
<li>Au total, pour cet auteur, une idéologie est une formation discursive polémique, ni vraie ni fausse, efficace ou inefficace, cohérente ou incohérente, élaborée ou non, normale ou pathologique, grâce à laquelle une passion cherche à réaliser une valeur par l’exercice du pouvoir dans une société<sup class="reference" id="cite_ref-6"><a href="#cite_note-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.</li></ol>
<p>Les analyses épistémologiques amènent une reformulation un peu plus nuancée de l'idéologie : celle-ci, ayant permis la conceptualisation des sciences, est également analysée quant à sa neutralité, sa construction et ses fondements. Et la critique marxiste n'est qu'un angle possible d'étude de ceux-ci.
</p><p>Le philosophe allemand Christian Duncker invoque la nécessité d'« une réflexion critique du concept d'idéologie » (2006). Dans son travail, il tâche d'introduire le concept de l'idéologie dans le premier plan, comme les soucis étroitement reliés de l'épistémologie et de l'histoire. Le terme <i>idéologie</i> est défini en termes de système de représentation qui explicitement ou implicitement clame la vérité absolue. Dans le <span class="citation">« système totalitaire »</span>, <a href="/wiki/Hannah_Arendt" title="Hannah Arendt">Hannah Arendt</a> écrit que l'idéologie est consubstantielle au phénomène totalitaire et qu'elle présente plusieurs caractéristiques indissociables. D’une part, elle forme un système d’interprétation définitive du monde, elle affiche une prétention omnisciente et « omni-explicative » de celui-ci, qu’il s’agisse des événements passés ou futurs. D’autre part, elle affirme son caractère irrécusable, infalsifiable. Elle n’est jamais prise en défaut et s’émancipe de la réalité. Une autre caractéristique de l’idéologie est son « logicisme », son aptitude à se doter d’une cohérence interne, à intégrer en permanence la contradiction dans un processus logique. L’idéologie, de ce point de vue, est exactement ce qu’elle prétend être : la logique d’une idée.
</p><p>L'idéologie est une <a href="/wiki/Pens%C3%A9e_de_groupe" title="Pensée de groupe">pensée de groupe</a>, le discours, la vision, et la logique s'adresse au groupe les soutenant et à la totalité de la société afin d'y faire adhérer le plus de monde. Autrement dit, l'idéologie est un moyen pour un groupe d'accroître son pouvoir par l'accumulation de force <a href="/wiki/Politique" title="Politique">politique</a>, de soutien, au sein de la société. L'idéologie est pourtant une vision tout à fait partiale qui peut se tromper lourdement (voir <a href="/wiki/Nazisme" title="Nazisme">nazisme</a>), cependant ce qui la définit c'est qu'elle cherche à devenir majoritaire, et par là même elle s'impose suivant un énoncé (discours d'une personne et de son groupe, sa minorité) et avec une logique comme structure la soutenant (voir <a href="/wiki/Totalitarisme" title="Totalitarisme">totalitarisme</a>). C'est la <a href="/wiki/Tyrannie_de_la_majorit%C3%A9" title="Tyrannie de la majorité">tyrannie de la majorité</a>, mais cette majorité dans l'idéologie est une force majoritaire instrumentalisée : pourtant là où il y a influence d'un groupe sur un autre ou sur la politique de la société, il ne s'agit donc pas à proprement parler de <a href="/wiki/D%C3%A9mocratie" title="Démocratie">démocratie</a>, mais d'un autre <a class="mw-redirect" href="/wiki/Type_de_gouvernement" title="Type de gouvernement">type de gouvernement</a>.
</p><p>Il existe également des cas d'idéologie se voulant sans chef ni organisation (voir <a href="/wiki/Anarchisme" title="Anarchisme">anarchisme</a>) : mais par le fait même que cette 'logique d'une vision' refuse le discours imposé verticalement, elle refuse donc la pensée de groupe, et son statut d'idéologie en devient dès lors discutable.
</p>
<h2><span id="Id.C3.A9ologies_politiques"></span><span class="mw-headline" id="Idéologies_politiques">Idéologies politiques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=5" title="Modifier la section : Idéologies politiques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=5" title="Modifier la section : Idéologies politiques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Analyse_marxiste">Analyse marxiste</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=6" title="Modifier la section : Analyse marxiste">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=6" title="Modifier la section : Analyse marxiste">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Pour <a href="/wiki/Karl_Marx" title="Karl Marx">Karl Marx</a>, l'idéologie est l'ensemble des idées, des valeurs et des normes servant à légitimer la division en classes de la société. L’idéologie au sens marxiste décrit donc l'idéologie dominante en tant que <span class="citation">« <a href="/wiki/Vision_du_monde" title="Vision du monde">vision du monde</a> »</span> imposée par la <a class="mw-redirect" href="/wiki/Classe_dominante" title="Classe dominante">classe dominante</a>. C'est la construction intellectuelle qui expliquerait et justifierait un ordre social existant à partir de raisons naturelles ou religieuses. Cette vision ne serait en réalité qu'un voile destiné à cacher la poursuite d'intérêts matériels égoïstes que la classe dominante utiliserait pour renforcer ou étendre sa domination : ainsi pour renforcer le pouvoir en place, l'idéologie de la classe dominante se présenterait de manière que les intérêts de la classe dominante paraissent être les intérêts de tous. L'idéologie devient une <a href="/wiki/Infrastructure_et_superstructure" title="Infrastructure et superstructure">superstructure</a> de la société dont elle émane et qu'elle soutient. Selon <a href="/wiki/Friedrich_Engels" title="Friedrich Engels">Friedrich Engels</a>, « l’idéologie est un processus que le soi-disant penseur accomplit sans doute consciemment, mais avec une conscience fausse. Les forces motrices véritables qui le mettent en mouvement lui restent inconnues, sinon ce ne serait point un processus idéologique<sup class="reference" id="cite_ref-7"><a href="#cite_note-7"><span class="cite_crochet">[</span>7<span class="cite_crochet">]</span></a></sup> ».
</p><p>La critique de <a href="/wiki/Karl_Marx" title="Karl Marx">Karl Marx</a> de l'idéologie est d'abord une critique de la <a href="/wiki/Mis%C3%A8re" title="Misère">misère</a> que cette idéologie cache, misère qui réside dans les rapports sociaux à la fois résultat et moteur de cette misère. La première misère est l' <i>obligation au travail</i> impliquée dans l'organisation de la société par <i><a href="/wiki/Le_Capital" title="Le Capital">Le Capital</a></i> dans laquelle toute personne dépourvue d'une part de ce capital se voit dans l'obligation de vendre sa <a href="/wiki/Force_de_travail" title="Force de travail">force de travail</a>. Des auteurs comme <a href="/wiki/J%C3%BCrgen_Habermas" title="Jürgen Habermas">Habermas</a>, <a href="/wiki/Louis_Althusser" title="Louis Althusser">Althusser</a>, <a class="new" href="/w/index.php?title=John_Thompson_(sociologue)&amp;action=edit&amp;redlink=1" title="John Thompson (sociologue) (page inexistante)">Thompson</a> <a class="extiw" href="https://en.wikipedia.org/wiki/John_Thompson_(sociologist)" title="en:John Thompson (sociologist)"><span class="indicateur-langue" title="Article en anglais : « John Thompson (sociologist) »">(en)</span></a>, vont développer cette conception critique de l'idéologie.
</p><p><a href="/wiki/Jean-Paul_Sartre" title="Jean-Paul Sartre">Jean-Paul Sartre</a> définit une idéologie comme « une conception globale du monde »<sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>8<span class="cite_crochet">]</span></a></sup>, sans en dédouaner le marxisme malgré son appartenance à ce courant.
</p><p><a href="/wiki/Louis_Althusser" title="Louis Althusser">Louis Althusser</a> utilise le concept d'« <a href="/wiki/Appareil_id%C3%A9ologique_d%27%C3%89tat" title="Appareil idéologique d'État">appareils idéologiques d'État</a> » (scolaire, famille, religion, information, syndical, juridique, culturel et politique), par distinction avec les « appareils répressifs » d'État (armée, gouvernement, administration).
</p><p>Les études de John B. Thompson concernant l'idéologie dans notre société moderne abordent les dimensions culturelles et politiques de l'idéologie en regard de la communication de masse, caractéristique de notre monde contemporain. L'idéologie concerne le comment le « sens » établit et maintient systématiquement des relations asymétriques de <a class="mw-redirect" href="/wiki/Pouvoir_(sociologie)" title="Pouvoir (sociologie)">pouvoir</a>.
</p>
<h3><span class="mw-headline" id="Analyse_situationniste">Analyse situationniste</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=7" title="Modifier la section : Analyse situationniste">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=7" title="Modifier la section : Analyse situationniste">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>L'idéologie a aussi trouvé ses critiques dans l'<a href="/wiki/Internationale_situationniste" title="Internationale situationniste">Internationale situationniste</a>, qui fait de la critique de l'idéologie la condition <i><a href="/wiki/Sine_qua_non" title="Sine qua non">sine qua non</a></i> des relations de ses membres entre eux<sup class="reference" id="cite_ref-9"><a href="#cite_note-9"><span class="cite_crochet">[</span>9<span class="cite_crochet">]</span></a></sup> : la représentation du monde répondant à celle de soi (« le monde du rêve est le rêve du monde » <a href="/wiki/Raoul_Vaneigem" title="Raoul Vaneigem">Raoul Vaneigem</a>) chacun est responsable de l'ensemble d'un projet dans lequel il se retrouve ; en l'occurrence, en finir avec le <i>spectacle</i>, organisation sociale où « tout ce qui était directement vécu s'est éloigné dans une représentation » (<i><a href="/wiki/La_Soci%C3%A9t%C3%A9_du_spectacle_(livre)" title="La Société du spectacle (livre)">La Société du spectacle</a></i>, <a href="/wiki/Guy_Debord" title="Guy Debord">Guy Debord</a>), spectacle qui est ici considéré comme la forme achevée du Capital.
</p><p>Plus clairement, pour l'Internationale situationniste, toute organisation révolutionnaire se doit de critiquer « radicalement toute idéologie en tant que pouvoir séparé des idées et idées du pouvoir séparé ».
</p>
<h2><span id="Id.C3.A9ologie_et_sciences_sociales"></span><span class="mw-headline" id="Idéologie_et_sciences_sociales">Idéologie et sciences sociales</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=8" title="Modifier la section : Idéologie et sciences sociales">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=8" title="Modifier la section : Idéologie et sciences sociales">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Saint-Simon">Saint-Simon</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=9" title="Modifier la section : Saint-Simon">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=9" title="Modifier la section : Saint-Simon">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a class="mw-redirect" href="/wiki/Claude_Henri_de_Rouvroy,_comte_de_Saint-Simon" title="Claude Henri de Rouvroy, comte de Saint-Simon">Claude Henri de Rouvroy, comte de Saint-Simon</a>.</div></div>
<p><a href="/wiki/Claude-Henri_de_Rouvroy_de_Saint-Simon" title="Claude-Henri de Rouvroy de Saint-Simon">Claude-Henri de Rouvroy de Saint-Simon</a> (1760-1825), lointain cousin du célèbre mémorialiste <a class="mw-redirect" href="/wiki/Louis_de_Rouvroy,_duc_de_Saint-Simon" title="Louis de Rouvroy, duc de Saint-Simon">Louis de Rouvroy, duc de Saint-Simon</a>, fut l'un des premiers à récupérer le concept d'idéologie afin d'en faire un système philosophique complet, entièrement fondé sur les sciences, en excluant tout apport des <a href="/wiki/Religion" title="Religion">religions</a>, puisqu'il était <a href="/wiki/Ath%C3%A9isme" title="Athéisme">athée</a>. Il joua un rôle tout particulier dans la diffusion de l'idéologie.
</p><p>Saint-Simon, très influencé par les idéologues, notamment le docteur <a href="/wiki/Jean_Burdin" title="Jean Burdin">Jean Burdin</a>, bâtit entre 1801 et 1825 un système global que <a href="/wiki/Pierre_Musso" title="Pierre Musso">Pierre Musso</a> qualifie de <a href="/wiki/Philosophie_des_r%C3%A9seaux" title="Philosophie des réseaux">philosophie des réseaux</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10"><span class="cite_crochet">[</span>10<span class="cite_crochet">]</span></a></sup>. Pour Saint-Simon, les relations des individus en société sont, par <a href="/wiki/M%C3%A9taphore" title="Métaphore">métaphore</a> avec la <a href="/wiki/Physiologie" title="Physiologie">physiologie</a>, qui était en plein développement à ce moment, assimilables aux réseaux organiques des êtres humains (réseaux sanguins, système nerveux). Il introduit aussi la notion de capacité du réseau. L'appellation de <span class="citation">« nouveau christianisme »</span> fut en réalité trompeuse pour un système qui, prenant <a href="/wiki/Isaac_Newton" title="Isaac Newton">Isaac Newton</a> comme référence suprême, prétendait remplacer <a href="/wiki/Dieu" title="Dieu">Dieu</a> par la <a href="/wiki/Loi_universelle_de_la_gravitation" title="Loi universelle de la gravitation">gravitation universelle</a>. Sur le plan spirituel, les sciences se substituent à la religion. Sur le plan temporel, les économistes remplacent les politiques<sup class="reference" id="cite_ref-11"><a href="#cite_note-11"><span class="cite_crochet">[</span>11<span class="cite_crochet">]</span></a></sup>.
</p><p>Le système de gouvernement doit comprendre trois chambres (chambre des inventeurs, chambre de savants, chambre d'exécution composée d'industriels)<sup class="reference" id="cite_ref-12"><a href="#cite_note-12"><span class="cite_crochet">[</span>12<span class="cite_crochet">]</span></a></sup>. Saint-Simon introduit la croyance exclusive dans le <a href="/wiki/Progr%C3%A8s" title="Progrès">progrès</a> industriel. Il est l'un des tenants de l'industrialisme, mot qu'il forgea en 1824 dans son <i>catéchisme des industriels</i>. Son système était très empreint de <a href="/wiki/Religiosit%C3%A9" title="Religiosité">religiosité</a><sup class="reference" id="cite_ref-13"><a href="#cite_note-13"><span class="cite_crochet">[</span>13<span class="cite_crochet">]</span></a></sup>.
</p>
<h3><span id="Recherche_d.27un_syst.C3.A8me_coh.C3.A9rent_.281825.29"></span><span class="mw-headline" id="Recherche_d'un_système_cohérent_(1825)">Recherche d'un système cohérent (1825)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=10" title="Modifier la section : Recherche d'un système cohérent (1825)">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=10" title="Modifier la section : Recherche d'un système cohérent (1825)">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La préoccupation de la recherche d'un système cohérent que l'on trouvait déjà dans l'école des idéologues, un moment oubliée par les guerres de l'Empire et par la Restauration, ressurgit vers 1825<sup class="reference" id="cite_ref-14"><a href="#cite_note-14"><span class="cite_crochet">[</span>14<span class="cite_crochet">]</span></a></sup>, dans le contexte du début du règne de <a class="mw-redirect" href="/wiki/Charles_X_de_France" title="Charles X de France">Charles X</a>.
</p><p>La fin de l'année 1825 et l'année 1826 furent ainsi, en France, un moment de réflexion sur un système philosophique global. On peut considérer que c'est une période charnière dans l'histoire des idées. Les penseurs qui participèrent à cette réflexion furent principalement <a href="/wiki/Auguste_Comte" title="Auguste Comte">Auguste Comte</a>, <a class="mw-redirect" href="/wiki/Barth%C3%A9lemy_Prosper_Enfantin" title="Barthélemy Prosper Enfantin">Barthélemy Prosper Enfantin</a>, <a href="/wiki/Charles_Fourier" title="Charles Fourier">Charles Fourier</a>… et probablement <a class="mw-redirect" href="/wiki/F%C3%A9licit%C3%A9_Robert_de_Lamennais" title="Félicité Robert de Lamennais">Lamennais</a>, qui fut engagé dans la réflexion des catholiques.
</p><p>Cette période initia un grand nombre de mouvements de différentes natures : idéologies, <a href="/wiki/Utopie" title="Utopie">utopies</a>, qui donneront naissance par la suite aux grandes théories sur le <a href="/wiki/Lib%C3%A9ralisme" title="Libéralisme">libéralisme</a>, ainsi qu'aux différentes formes de <a href="/wiki/Socialisme" title="Socialisme">socialisme</a>.
</p>
<h3><span class="mw-headline" id="Saint-Simonisme_:_continuation">Saint-Simonisme : continuation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=11" title="Modifier la section : Saint-Simonisme : continuation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=11" title="Modifier la section : Saint-Simonisme : continuation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a href="/wiki/Saint-simonisme" title="Saint-simonisme">Saint-simonisme</a>.</div></div>
<p>À la mort de <a class="mw-redirect" href="/wiki/Claude_Henri_de_Rouvroy,_comte_de_Saint-Simon" title="Claude Henri de Rouvroy, comte de Saint-Simon">Saint-Simon</a> (1825), un polytechnicien, <a class="mw-redirect" href="/wiki/Barth%C3%A9lemy_Prosper_Enfantin" title="Barthélemy Prosper Enfantin">Barthélemy Prosper Enfantin</a> reprend sa doctrine. Très intéressé par le système de Saint-Simon, il publie avec <a href="/wiki/Saint-Amand_Bazard" title="Saint-Amand Bazard">Saint-Amand Bazard</a>, l'<i>Exposition de la doctrine de Saint-Simon</i> (1829). Ces idées sont ainsi diffusées par le mouvement dit <a href="/wiki/Saint-simonisme" title="Saint-simonisme">saint-simonien</a>, sous des formes transformées au cours du temps. En 1831, <a href="/wiki/Saint-Amand_Bazard" title="Saint-Amand Bazard">Saint-Amand Bazard</a> se détache du groupe libéral d'Enfantin (schisme) et fonde une branche de sensibilité socialiste, qui influence notamment <a href="/wiki/Karl_Marx" title="Karl Marx">Marx</a> ; <a class="mw-redirect" href="/wiki/Lazare_Hippolyte_Carnot" title="Lazare Hippolyte Carnot">Lazare Hippolyte Carnot</a>, second fils de <a class="mw-redirect" href="/wiki/Lazare_Nicolas_Marguerite_Carnot" title="Lazare Nicolas Marguerite Carnot">Lazare Carnot</a>, collabore à l'un des journaux ; <a href="/wiki/Michel_Chevalier_(homme_politique)" title="Michel Chevalier (homme politique)">Michel Chevalier</a>, saint-simonien de sensibilité libérale, est un proche conseiller de <a href="/wiki/Napol%C3%A9on_III" title="Napoléon III">Napoléon III</a> ; les idées <a href="/wiki/Saint-simonisme" title="Saint-simonisme">saint-simoniennes</a> se développent dans certains cercles de l'école polytechnique.
</p><p>Les idées saint-simoniennes ont ainsi une forte influence en <a href="/wiki/France" title="France">France</a> dans la phase de développement industriel du <a href="/wiki/Second_Empire" title="Second Empire">Second Empire</a>, puis de la <a href="/wiki/Troisi%C3%A8me_R%C3%A9publique_(France)" title="Troisième République (France)"><abbr class="abbr" title="Troisième">III<sup>e</sup></abbr> République</a> (<a href="/wiki/R%C3%A9volution_industrielle" title="Révolution industrielle">révolution industrielle</a>). Elles se répandent aussi hors des frontières à travers la <a href="/wiki/Colonisation" title="Colonisation">colonisation</a> en <a href="/wiki/Afrique" title="Afrique">Afrique</a> et au <a href="/wiki/Moyen-Orient" title="Moyen-Orient">Moyen-Orient</a> dont <a class="mw-redirect" href="/wiki/Barth%C3%A9lemy_Prosper_Enfantin" title="Barthélemy Prosper Enfantin">Enfantin</a> fut l'initiateur (pour plus de détails, voir l'article sur <a class="mw-redirect" href="/wiki/Barth%C3%A9lemy_Prosper_Enfantin" title="Barthélemy Prosper Enfantin">Barthélemy Prosper Enfantin</a>). C'est ainsi que l'on parle d'une <a href="/wiki/Id%C3%A9ologie_coloniale_fran%C3%A7aise" title="Idéologie coloniale française">idéologie coloniale française</a> (voir aussi dans l'article <a href="/wiki/Cercle_Saint-Simon" title="Cercle Saint-Simon">cercle Saint-Simon</a> les liens avec la colonisation et la propagation de la <a class="mw-redirect" href="/wiki/Langue_fran%C3%A7aise" title="Langue française">langue française</a>). Elles trouvent des applications pratiques dans la construction des <a href="/wiki/Chemin_de_fer" title="Chemin de fer">chemins de fer</a> (étoile de Belgrand), de routes, de canaux, et encore aujourd'hui dans les réseaux de <a href="/wiki/T%C3%A9l%C3%A9communications" title="Télécommunications">télécommunications</a><sup class="reference" id="cite_ref-15"><a href="#cite_note-15"><span class="cite_crochet">[</span>15<span class="cite_crochet">]</span></a></sup>.
</p>
<h3><span id="Positivisme_d.27Auguste_Comte"></span><span class="mw-headline" id="Positivisme_d'Auguste_Comte">Positivisme d'Auguste Comte</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=12" title="Modifier la section : Positivisme d'Auguste Comte">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=12" title="Modifier la section : Positivisme d'Auguste Comte">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/Auguste_Comte" title="Auguste Comte">Auguste Comte</a> fut secrétaire de <a class="mw-redirect" href="/wiki/Claude_Henri_de_Rouvroy,_comte_de_Saint-Simon" title="Claude Henri de Rouvroy, comte de Saint-Simon">Saint-Simon</a> de 1817 à 1824. Il quitta Saint-Simon pour fonder son propre mouvement philosophique.
</p><p>L'idéologie de Comte se subdivise en deux parties :
</p>
<ul><li>Le <a href="/wiki/Auguste_Comte#Le_positivisme_scientifique" title="Auguste Comte">positivisme scientifique</a> : les causes premières sont oubliées. Dans le <i>Cours de philosophie positive</i> (1830-1842), Comte expose la <a href="/wiki/Loi_des_trois_%C3%A9tats" title="Loi des trois états">loi des trois états</a> : l'humanité passe par trois états : l'état théologique (les dieux gouvernent le monde), l'état <a href="/wiki/M%C3%A9taphysique" title="Métaphysique">métaphysique</a> (des entités abstraites déterminent le monde), et l'état positif (les sciences parviennent à l'état positif).</li>
<li>Le <a href="/wiki/%C3%89glise_positiviste" title="Église positiviste">positivisme religieux</a> : Dans cette phase, Auguste Comte définit les relations en société à partir de trois fondements : l'<a href="/wiki/Altruisme" title="Altruisme">altruisme</a>, l'ordre et le <a href="/wiki/Progr%C3%A8s" title="Progrès">progrès</a>. La <a href="/wiki/Sociologie" title="Sociologie">sociologie</a> (il reprend le terme de <a class="mw-redirect" href="/wiki/Siey%C3%A8s" title="Sieyès">Sieyès</a>) couronne les sciences dites positives : mathématiques, physique, chimie, astronomie, biologie (<i><a href="/wiki/Syst%C3%A8me_de_politique_positive" title="Système de politique positive">Système de politique positive</a></i>, 1851-1854).</li></ul>
<p>Le monde est gouverné par les morts. L'humanité est un Grand-Être, sorte de continuation du culte de l'<a class="mw-redirect" href="/wiki/Culte_de_la_Raison_et_de_l%27%C3%8Atre_supr%C3%AAme" title="Culte de la Raison et de l'Être suprême">Être Suprême</a>, dont il est le <span class="citation">« grand-prêtre »</span><sup class="reference" id="cite_ref-16"><a href="#cite_note-16"><span class="cite_crochet">[</span>16<span class="cite_crochet">]</span></a></sup>. Le <a href="/wiki/Positivisme" title="Positivisme">positivisme</a> aura une influence déterminante à partir du milieu du <abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle sur de nombreuses personnalités et dans de nombreux domaines : le <a class="mw-redirect" href="/wiki/Positivisme_logique" title="Positivisme logique">positivisme logique</a>, le <a href="/wiki/Positivisme_juridique" title="Positivisme juridique">positivisme juridique</a>, qui se fonde sur le système de politique positive de la phase religieuse, et le <a href="/wiki/Positivisme#Le_positivisme_aujourd'hui" title="Positivisme">néopositivisme</a>.
</p>
<h3><span id="Caract.C3.A9ristiques_des_premi.C3.A8res_id.C3.A9ologies"></span><span class="mw-headline" id="Caractéristiques_des_premières_idéologies">Caractéristiques des premières idéologies</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=13" title="Modifier la section : Caractéristiques des premières idéologies">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=13" title="Modifier la section : Caractéristiques des premières idéologies">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Les idées de <a class="mw-redirect" href="/wiki/Claude_Henri_de_Rouvroy,_comte_de_Saint-Simon" title="Claude Henri de Rouvroy, comte de Saint-Simon">Saint-Simon</a> et d'<a href="/wiki/Auguste_Comte" title="Auguste Comte">Auguste Comte</a> ont en commun une certaine <a href="/wiki/Religiosit%C3%A9" title="Religiosité">religiosité</a> et une foi absolue dans le <a href="/wiki/Progr%C3%A8s" title="Progrès">progrès</a> des sociétés humaines par les sciences, la technique, et l'industrie. Tous deux excluent la <a href="/wiki/M%C3%A9taphysique" title="Métaphysique">métaphysique</a>, et remplacent la <a href="/wiki/T%C3%A9l%C3%A9ologie" title="Téléologie">finalité</a> par l'<a href="/wiki/Explication" title="Explication">explication</a> scientifique des <a href="/wiki/Ph%C3%A9nom%C3%A8ne" title="Phénomène">phénomènes</a>. Ils ignorent les <a href="/wiki/Litt%C3%A9rature_antique" title="Littérature antique">auteurs classiques de l'antiquité</a> <a href="/wiki/Gr%C3%A8ce_antique" title="Grèce antique">grecque</a> et <a href="/wiki/Rome_antique" title="Rome antique">romaine</a>, qui avaient été redécouverts dès le <a href="/wiki/Moyen_%C3%82ge" title="Moyen Âge">Moyen Âge</a> et la <a class="mw-redirect" href="/wiki/Renaissance_(p%C3%A9riode_historique)" title="Renaissance (période historique)">Renaissance</a>. Ils ne s'appuient ni sur les <a href="/wiki/Pr%C3%A9socratiques" title="Présocratiques">présocratiques</a>, ni sur la <a href="/wiki/Philosophie_antique" title="Philosophie antique">philosophie antique</a>.
</p>
<h2><span id="Id.C3.A9ologie_et_sciences"></span><span class="mw-headline" id="Idéologie_et_sciences">Idéologie et sciences</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=14" title="Modifier la section : Idéologie et sciences">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=14" title="Modifier la section : Idéologie et sciences">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>La notion d'<i>idéologie scientifique</i> peut apparaître de prime abord comme un « monstre logique » puisque la science s'oppose catégoriquement aux idéologies politiques, juridiques, économiques et religieuses. Marx ne cite pas la science au nombre des idéologies, bien que la science dépende de l'activité matérielle des hommes<sup class="reference" id="cite_ref-:0_2-2"><a href="#cite_note-:0-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>Canguilhem utilise l'expression pour désigner les discours, à prétention de théorie scientifique, qui apparaissent et disparaissent dans l'évolution historique des connaissances. Il distingue l'<i>idéologie scientifique</i> (domaine de l'<a href="/wiki/%C3%89pist%C3%A9mologie" title="Épistémologie">épistémologie</a>) de l'<i>idéologie des scientifiques</i><sup class="reference" id="cite_ref-:1_17-0"><a href="#cite_note-:1-17"><span class="cite_crochet">[</span>17<span class="cite_crochet">]</span></a></sup>, et qui serait plutôt du domaine de la <a href="/wiki/Sociologie_des_sciences" title="Sociologie des sciences">sociologie des sciences</a>.
</p>
<h3><span id="Id.C3.A9ologie_des_scientifiques"></span><span class="mw-headline" id="Idéologie_des_scientifiques">Idéologie des scientifiques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=15" title="Modifier la section : Idéologie des scientifiques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=15" title="Modifier la section : Idéologie des scientifiques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>C'est l'ensemble des discours tenus par les scientifiques sur leur méthodes, leur objet, leur place relative dans la culture et la société.
</p><p>Les idéologies <i>des</i> scientifiques sont des idéologies philosophiques<sup class="reference" id="cite_ref-:1_17-1"><a href="#cite_note-:1-17"><span class="cite_crochet">[</span>17<span class="cite_crochet">]</span></a></sup>. Par exemple, au <abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle est apparu le <a href="/wiki/Scientisme" title="Scientisme">scientisme</a> postulant que la connaissance scientifique doit permettre d'échapper à l'ignorance dans tous les domaines et donc d'organiser scientifiquement l'humanité. De même décréter que la science, la technologie sont neutres, peut faire partie intégrante d'une idéologie philosophique (comme d'ailleurs le point de vue opposé : <a href="/wiki/Relativisme" title="Relativisme">relativisme</a>, voire la réduction de la science à une croyance sociologique déterminée par des intérêts).
</p><p>Canguilhem donne comme exemples de concepts idéologiques de scientifiques au <abbr class="abbr" title="18ᵉ siècle"><span class="romain">XVIII</span><sup style="font-size:72%">e</sup></abbr> siècle, ceux de <i>Nature</i> et d'<i>Expérience</i><sup class="reference" id="cite_ref-:1_17-2"><a href="#cite_note-:1-17"><span class="cite_crochet">[</span>17<span class="cite_crochet">]</span></a></sup>.
</p>
<h3><span id="Id.C3.A9ologie_scientifique"></span><span class="mw-headline" id="Idéologie_scientifique">Idéologie scientifique</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=16" title="Modifier la section : Idéologie scientifique">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=16" title="Modifier la section : Idéologie scientifique">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Selon Canguilhem, l'idéologie scientifique serait plutôt une idéologie de philosophes à prétention scientifique, ou des scientifiques « présomptifs ou présomptueux », souvent considérés comme des précurseurs. Il donne les exemples de <a href="/wiki/Pierre_Louis_Moreau_de_Maupertuis" title="Pierre Louis Moreau de Maupertuis">Maupertuis</a> (avec son « atome séminal »), de <a href="/wiki/Georges-Louis_Leclerc_de_Buffon" title="Georges-Louis Leclerc de Buffon">Buffon</a> (avec sa « molécule organique »), de <a href="/wiki/Charles_Bonnet_(naturaliste)" title="Charles Bonnet (naturaliste)">Charles Bonnet</a> (« échelle des êtres ») et de <a href="/wiki/Denis_Diderot" title="Denis Diderot">Diderot</a> (dans <i><a href="/wiki/Le_R%C3%AAve_de_D%27Alembert" title="Le Rêve de D'Alembert">Le Rêve de d'Alembert</a></i> pour l'idée d'évolution du vivant) qui sont des idéologies scientifiques dans le domaine des sciences naturelles.
</p><p>
L'idéologie scientifique n'est ni une fausse conscience (comme Marx l'entendait de l'idéologie), ni une fausse science, car la fausse science ne rencontre jamais le faux et ne renonce à rien. La fausse science n'a pas d'histoire, alors que toute science passe par un état préscientifique. L'idéologie scientifique est alors « évidemment »<sup class="reference" id="cite_ref-:2_18-0"><a href="#cite_note-:2-18"><span class="cite_crochet">[</span>18<span class="cite_crochet">]</span></a></sup> :</p><blockquote><p>« La méconnaissance des exigences méthodologiques et des possibilités opératoires de la science dans le secteur de l'expérience qu'elle cherche à investir, mais elle n'est pas l'ignorance, ou le mépris ou le refus de la science. »</p></blockquote><p>L'idéologie scientifique n'est pas une superstition, car si elle occupe une place usurpée dans l'espace de la connaissance, elle n'est pas dans l'espace des croyances religieuses. L'idéologie scientifique est sur-située (historiquement en amont) par rapport à la science. Elle est aussi dé-portée, car la science constituée se place dans un autre cadre que l'idéologie lui assignait<sup class="reference" id="cite_ref-:2_18-1"><a href="#cite_note-:2-18"><span class="cite_crochet">[</span>18<span class="cite_crochet">]</span></a></sup>.
</p><p>Canguilhem donne l'exemple de l'<a href="/wiki/Atomisme" title="Atomisme">atomisme</a>, comme idéologie scientifique jusqu'au <abbr class="abbr" title="18ᵉ siècle"><span class="romain">XVIII</span><sup style="font-size:72%">e</sup></abbr> siècle. Lorsque la chimie et la physique constituent la connaissance scientifique de l'atome, le mot persiste mais dans un cadre différent de l'atomisme grec, qui n'est plus celui de l'indivisible : « ce que la science trouve n'est pas ce que l'idéologie donnait à chercher »<sup class="reference" id="cite_ref-:2_18-2"><a href="#cite_note-:2-18"><span class="cite_crochet">[</span>18<span class="cite_crochet">]</span></a></sup>.
</p><p>De même, la <a href="/wiki/Th%C3%A9orie_de_la_d%C3%A9g%C3%A9n%C3%A9rescence" title="Théorie de la dégénérescence">théorie de la dégénérescence</a> à la fin du <abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle est considérée comme une « idéologie scientifique majeure » dans le domaine de l'<a href="/wiki/H%C3%A9r%C3%A9dit%C3%A9" title="Hérédité">hérédité</a> pathologique<sup class="reference" id="cite_ref-19"><a href="#cite_note-19"><span class="cite_crochet">[</span>19<span class="cite_crochet">]</span></a></sup>.
</p><p>Dans l'épistémologie des sciences même, chez <a href="/wiki/Thomas_Samuel_Kuhn" title="Thomas Samuel Kuhn">Kuhn</a>, le concept de <a href="/wiki/Paradigme" title="Paradigme">paradigme</a> dominant explique la stagnation et la discontinuité de l'évolution des théories scientifiques.
</p>
<h3><span id="Id.C3.A9ologie_et_v.C3.A9rit.C3.A9"></span><span class="mw-headline" id="Idéologie_et_vérité">Idéologie et vérité</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=17" title="Modifier la section : Idéologie et vérité">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=17" title="Modifier la section : Idéologie et vérité">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span class="mw-headline" id="Histoire_des_sciences">Histoire des sciences</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=18" title="Modifier la section : Histoire des sciences">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=18" title="Modifier la section : Histoire des sciences">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Canguilhem distingue une histoire des sciences qui s'articule selon une succession de <i>faits de vérité</i>, et qui dès lors n'a pas à rendre compte des <i>idéologies scientifiques</i>. Et aussi une histoire des sciences « qui traite une science dans son histoire comme une purification élaborée de <i>normes de vérification</i> », et qui doit <i>a contrario</i> s'en occuper<sup class="reference" id="cite_ref-:3_20-0"><a href="#cite_note-:3-20"><span class="cite_crochet">[</span>20<span class="cite_crochet">]</span></a></sup>.
</p><p>L'idéologie et la science devraient être distinguées, mais aussi entrelacées. Distinguées, par exemple pour ne pas projeter ou mettre en continuité des concepts scientifiques modernes avec des concepts antiques ou médiévaux, ou chercher chez Diderot ce qui se trouvera chez Darwin ; entrelacées « pour empêcher de réduire l'histoire d'une science à la platitude d'un historique, c'est-à-dire sans ombres de relief »<sup class="reference" id="cite_ref-:3_20-1"><a href="#cite_note-:3-20"><span class="cite_crochet">[</span>20<span class="cite_crochet">]</span></a></sup>.
</p><p>Selon Canguilhem, la spécificité de l'idéologie scientifique doit être reconnue en lui faisant une place « sur différents plans de scientificité », sans quoi l'histoire des sciences risquerait d'être une idéologie dans son sens péjoratif de fausse conscience. « À ne vouloir faire que l'histoire de la vérité on fait une histoire illusoire »<sup class="reference" id="cite_ref-:3_20-2"><a href="#cite_note-:3-20"><span class="cite_crochet">[</span>20<span class="cite_crochet">]</span></a></sup>.
</p>
<h4><span id=".C3.89pist.C3.A9mologie"></span><span class="mw-headline" id="Épistémologie">Épistémologie</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=19" title="Modifier la section : Épistémologie">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=19" title="Modifier la section : Épistémologie">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>La constatation de l'origine sociologique d'une théorie scientifique n'implique pas qu'elle soit de valeur limitée. Ainsi l'origine religieuse du concept d'<a href="/wiki/%C3%89nergie" title="Énergie">énergie</a> n'enlève rien à la valeur scientifique de la notion d'<a href="/wiki/%C3%89nergie_nucl%C3%A9aire" title="Énergie nucléaire">énergie atomique</a><sup class="reference" id="cite_ref-:4_21-0"><a href="#cite_note-:4-21"><span class="cite_crochet">[</span>21<span class="cite_crochet">]</span></a></sup>.
</p><p>Des théories scientifiques peuvent être idéologisées, ce qui ne les invalide pas forcément. Des théories scientifiques valables peuvent s'intégrer dans des ensembles idéologiques, et une approche idéologique peut préparer le terrain à une approche scientifique et déclencher des études de valeur scientifiques<sup class="reference" id="cite_ref-:4_21-1"><a href="#cite_note-:4-21"><span class="cite_crochet">[</span>21<span class="cite_crochet">]</span></a></sup> (par exemple dans le domaine des <a class="mw-redirect" href="/wiki/Sciences_de_l%27environnement" title="Sciences de l'environnement">sciences de l'environnement</a>).
</p><p>Une théorie, parmi d'autres théories concurrentes de valeur comparable, peut être sélectionnée par « choix idéologique », mais ceci ne garantit ni la validité, ni le caractère erroné de la théorie choisie.
</p><p>Selon <a href="/wiki/Joseph_Gabel" title="Joseph Gabel">Joseph Gabel</a> « l'identification scientifique vise à simplifier des réalités compliquées, afin de les mettre à la portée de la science ». Alors que l'identification idéologique simplifie encore plus des réalités parfois simples, « pour gagner, en échange du confort intellectuel ainsi offert, l'adhésion des foules ».
</p><p>Il y aurait une analogie positive de la connaissance qui vise à connaître quelque chose en l'<i>assimilant à quelque chose de déjà connu</i>, et une analogie négative de l'idéologie qui tend à faire <i>détester quelque chose en l'assimilant à quelque chose de déjà détesté</i><sup class="reference" id="cite_ref-:4_21-2"><a href="#cite_note-:4-21"><span class="cite_crochet">[</span>21<span class="cite_crochet">]</span></a></sup><i>.</i>
</p>
<h2><span class="mw-headline" id="Approches_psychologiques">Approches psychologiques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Id%C3%A9ologie&amp;veaction=edit&amp;section=20" title="Modifier la section : Approches psychologiques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Id%C3%A9ologie&amp;action=edit&amp;section=20" title="Modifier la section : Approches psychologiques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Cette partie est une traduction du <a class="new" href="/w/index.php?title=Passage_sur_l%27id%C3%A9ologie&amp;action=edit&amp;redlink=1" title="Passage sur l'idéologie (page inexistante)">passage sur l'idéologie</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Psychological_research" title="en:Psychological research"><span class="indicateur-langue" title="Article en anglais : « Psychological research »">(en)</span></a>. Certaines recherches en psychologie <sup class="reference" id="cite_ref-jost2008_22-0"><a href="#cite_note-jost2008-22"><span class="cite_crochet">[</span>22<span class="cite_crochet">]</span></a></sup> suggèrent que les idéologies reflètent les procédés des besoins et désirs, contrairement à la pensée que les convictions politiques dérivent toujours d’une réflexion indépendante et objective. En 2008<sup class="reference" id="cite_ref-jost2008_22-1"><a href="#cite_note-jost2008-22"><span class="cite_crochet">[</span>22<span class="cite_crochet">]</span></a></sup>, une recherche a suggéré que les idéologies pourraient fonctionner comme des éléments d’interprétation qui se répandent pour répondre aux besoins de comprendre le monde, d’éviter l’angoisse existentielle et de maintenir des relations d’estime entre les personnes. Les auteurs ont conclu que de tels besoins pourraient conduire de façon disproportionnée à l’adoption de systèmes de justification des visions du monde (voir l'étymologie d'idéologie).
</p><p>Les psychologues ont découvert que des <a class="new" href="/w/index.php?title=Trait_de_personnalit%C3%A9&amp;action=edit&amp;redlink=1" title="Trait de personnalité (page inexistante)">traits de personnalité</a>, diverses particularités individuelles, besoins et croyances idéologiques pourraient être liés. Par exemple, une méta-analyse de Jost, Glaser, Kruglanski et Sulloway en 2003 a confronté 88 études originaires de 12 pays différents, comportant plus de 22 000 sujets et a trouvé que l’angoisse de la mort (présente dans le terrorisme dans les médias, le marketing de la peur ; <a href="/wiki/Th%C3%A9orie_de_la_gestion_de_la_peur" title="Théorie de la gestion de la peur">théorie de la gestion de la peur</a>, les intransigeances/intolérance face à l’ambiguïté, le manque d’ouverture aux nouvelles expériences (<i>lack of openness to experience</i>), le fait d’éviter l’incertitude (<a class="mw-redirect" href="/wiki/Aversion_%C3%A0_l%27incertitude" title="Aversion à l'incertitude">aversion à l'incertitude</a>), le besoin de se réduire à l’aspect cognitif, le besoin d’une structure identitaire personnelle, et la crainte de perdre sa position ou son <a href="/wiki/Estime_de_soi" title="Estime de soi">estime personnelle</a>, tous contribuent au degré de conservatisme politique<sup class="reference" id="cite_ref-23"><a href="#cite_note-23"><span class="cite_crochet">[</span>23<span class="cite_crochet">]</span></a></sup> chez l’individu.
</p><p>Selon les chercheurs, ces résultats montreraient que les conservateurs en politique mettent l’accent sur la <a class="mw-redirect" href="/wiki/Peur_du_changement" title="Peur du changement">résistance au changement</a> et qu’ils sont mus par des besoins qui visent à réduire la peur et l’incertitude. Selon <a class="new" href="/w/index.php?title=Robert_Altemeyer&amp;action=edit&amp;redlink=1" title="Robert Altemeyer (page inexistante)">Robert Altemeyer</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Robert_Altemeyer" title="en:Robert Altemeyer"><span class="indicateur-langue" title="Article en anglais : « Robert Altemeyer »">(en)</span></a> ainsi que d’autres chercheurs, les individus conservateurs en politique ont tendance à se placer très haut sur l’échelle d'autoritarisme de droite.
</p>
<ul><li><a class="new" href="/w/index.php?title=Right-wing_Authoritarianism&amp;action=edit&amp;redlink=1" title="Right-wing Authoritarianism (page inexistante)">Right-wing Authoritarianism</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Right-wing_Authoritarianism" title="en:Right-wing Authoritarianism"><span class="indicateur-langue" title="Article en anglais : « Right-wing Authoritarianism »">(en)</span></a> (RWA) : Échelle mesurant la soumission d’un individu aux autorités établies, son agressivité contre les opposants des autorités établies et son adhérence aux normes sociales. En dépit du terme <i>right-wing</i> (de droite), il a été montré en <a class="mw-redirect" href="/wiki/URSS" title="URSS">URSS</a> que des individus communistes pouvaient donner des scores élevés<sup class="reference" id="cite_ref-24"><a href="#cite_note-24"><span class="cite_crochet">[</span>24<span class="cite_crochet">]</span></a></sup>. La psychologue Felicia Pratto et ses collègues ont obtenu des données soutenant l'idée qu’une grande orientation vers la domination sociale est fortement liée à des visées politiques conservatrices.</li>
<li><a class="new" href="/w/index.php?title=Social_Dominance_Orientation_Social&amp;action=edit&amp;redlink=1" title="Social Dominance Orientation Social (page inexistante)">Social Dominance Orientation Social</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Social_Dominance_Orientation_Social" title="en:Social Dominance Orientation Social"><span class="indicateur-langue" title="Article en anglais : « Social Dominance Orientation Social »">(en)</span></a> (SDO) : Échelle mesurant la préférence d’un individu pour un système hiérarchique.</li></ul>
<p>Il est donc avéré que le conservatisme de droite ou de gauche, défini par une politique et une idéologie rigide et fermée, risque de conduire à choisir — souvent inconsciemment — une idéologie caractérisée par l’autoritarisme (pouvant aller jusqu'au fascisme ou au totalitarisme), et favorisant donc ses représentants. En rapport dans le domaine psychologique et sociologique : <a href="/wiki/Propagande" title="Propagande">Propagande</a>, <a href="/wiki/Langue_de_bois" title="Langue de bois">langue de bois</a>, <a href="/wiki/Sophisme" title="Sophisme">sophisme</a>, <a href="/wiki/Scientisme" title="Scientisme">scientisme</a>, <a class="mw-redirect" href="/wiki/Pouvoir_(sociologie)" title="Pouvoir (sociologie)">pouvoir (sociologie)</a>, <a href="/wiki/Organisation_sociale" title="Organisation sociale">organisation sociale</a>, <a href="/wiki/H%C3%A9g%C3%A9monie" title="Hégémonie">hégémonie</a>, <a href="/wiki/Manipulation_mentale" title="Manipulation mentale">manipulation mentale</a>, <a href="/wiki/Norme" title="Norme">norme</a>, <a class="mw-redirect" href="/wiki/Valeur_(sociologie)" title="Valeur (sociologie)">valeur (sociologie)</a>, <a href="/wiki/Biais_%C3%A9motionnel" title="Biais émotionnel">biais émotionnel</a>, <a href="/wiki/Amalgame_(communication)" title="Amalgame (communication)">amalgame (communication)</a>, <a href="/wiki/Conformisme" title="Conformisme">conformisme</a>, <a href="/wiki/Ob%C3%A9issance" title="Obéissance">obéissance</a>, <a href="/wiki/Normalit%C3%A9_(comportement)" title="Normalité (comportement)">normalité</a>, <a href="/wiki/Statu_quo_ante_bellum" title="Statu quo ante bellum">statu quo</a>, <a href="/wiki/Effet_de_simple_exposition" title="Effet de simple exposition">effet de simple exposition</a>, <a href="/wiki/Effet_de_halo" title="Effet de halo">effet de halo</a>, <a href="/wiki/Lieu_commun" title="Lieu commun">lieu commun</a>, <a href="/wiki/Pens%C3%A9e_de_groupe" title="Pensée de groupe">pensée de groupe</a>, <a class="mw-redirect" href="/wiki/Hyperstimulus" title="Hyperstimulus">hyperstimulus</a>, et <a href="/wiki/Ancrage_(psychologie)" title="Ancrage (psychologie)">ancrage</a>.
