</p>
<div class="bandeau-container homonymie plainlinks hatnote" style=""><div class="bandeau-cell bandeau-icone metadata" style="display:table-cell;padding-right:0.5em"><a href="/wiki/Aide:Homonymie" title="Aide:Homonymie"><img alt="Page d’aide sur l’homonymie" class="noviewer" data-file-height="375" data-file-width="512" decoding="async" height="15" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a9/Logo_disambig.svg/20px-Logo_disambig.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a9/Logo_disambig.svg/30px-Logo_disambig.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a9/Logo_disambig.svg/40px-Logo_disambig.svg.png 2x" width="20"/></a></div><div class="bandeau-cell" style="display:table-cell;padding-right:0.5em">
<p>Pour les articles homonymes, voir <a class="mw-disambig" href="/wiki/E" title="E">E</a> et <a class="mw-redirect" href="/wiki/Liste_de_sujets_nomm%C3%A9s_d%27apr%C3%A8s_Leonhard_Euler#Nombres" title="Liste de sujets nommés d'après Leonhard Euler">Nombres d'Euler</a>.
</p>

<div class="thumb tright"><div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/Fichier:Hyperbola_E.svg"><img alt="" class="thumbimage" data-file-height="652" data-file-width="609" decoding="async" height="182" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Hyperbola_E.svg/170px-Hyperbola_E.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Hyperbola_E.svg/255px-Hyperbola_E.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Hyperbola_E.svg/340px-Hyperbola_E.svg.png 2x" width="170"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Hyperbola_E.svg" title="Agrandir"></a></div>L’aire sous l’hyperbole est égale à 1 sur l’intervalle <span class="texhtml">[1, e]</span>.</div></div></div>
<p>Le <b>nombre <span class="texhtml">e</span></b><sup class="reference" id="cite_ref-1"><a href="#cite_note-1"><span class="cite_crochet">[</span>a<span class="cite_crochet">]</span></a></sup> est la base des <a href="/wiki/Logarithme_naturel" title="Logarithme naturel">logarithmes naturels</a>, c'est-à-dire le <a href="/wiki/Nombre" title="Nombre">nombre</a> défini par <span class="texhtml">ln(e) = 1</span>. Cette <a href="/wiki/Table_de_constantes_math%C3%A9matiques" title="Table de constantes mathématiques">constante mathématique</a>, également appelée nombre d'Euler<sup class="reference" id="cite_ref-2"><a href="#cite_note-2"><span class="cite_crochet">[</span>b<span class="cite_crochet">]</span></a></sup> ou <b>constante de Néper</b> en référence aux mathématiciens <a href="/wiki/Leonhard_Euler" title="Leonhard Euler">Leonhard Euler</a> et <a href="/wiki/John_Napier" title="John Napier">John Napier</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>c<span class="cite_crochet">]</span></a></sup>, vaut <a href="#Décimales_connues">environ 2,71828</a>.
</p><p>Ce nombre est défini à la fin du <a href="/wiki/XVIIe_si%C3%A8cle" title="XVIIe siècle"><abbr class="abbr" title="17ᵉ siècle"><span class="romain">XVII</span><sup style="font-size:72%">e</sup></abbr> siècle</a>, dans une correspondance entre <a href="/wiki/Gottfried_Wilhelm_Leibniz" title="Gottfried Wilhelm Leibniz">Leibniz</a> et <a href="/wiki/Christian_Huygens" title="Christian Huygens">Christian Huygens</a>, comme étant la <a class="mw-redirect" href="/wiki/Base_(analyse)" title="Base (analyse)">base</a> du <a href="/wiki/Logarithme_naturel" title="Logarithme naturel">logarithme naturel</a>. Autrement dit, il est caractérisé par la relation <span class="texhtml">ln(e) = 1</span> ou de façon équivalente il est l'image de 1 par la <a href="/wiki/Fonction_exponentielle" title="Fonction exponentielle">fonction exponentielle</a>, d'où la notation <span class="texhtml">exp(<i>x</i>) = e<sup><i>x</i></sup></span>. La décomposition de cette fonction en <a href="/wiki/S%C3%A9rie_enti%C3%A8re" title="Série entière">série entière</a> mène à la définition de <span class="texhtml">e</span> par Euler comme somme de la <a href="/wiki/S%C3%A9rie_(math%C3%A9matiques)" title="Série (mathématiques)">série</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup> :
<span style="display: block; margin-left:1.6em;"><span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle \mathrm {e} =1+{\frac {1}{1}}+{\frac {1}{1\times 2}}+{\frac {1}{1\times 2\times 3}}+{\frac {1}{1\times 2\times 3\times 4}}+\cdots =\sum _{n=0}^{+\infty }{\dfrac {1}{n!}}.}" xmlns="http://www.w3.org/1998/Math/MathML">

</p>

<h2><span id="D.C3.A9finition_par_les_logarithmes"></span><span class="mw-headline" id="Définition_par_les_logarithmes">Définition par les logarithmes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=1" title="Modifier la section : Définition par les logarithmes">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=1" title="Modifier la section : Définition par les logarithmes">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="Logarithme_n.C3.A9p.C3.A9rien"></span><span class="mw-headline" id="Logarithme_népérien">Logarithme népérien</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=2" title="Modifier la section : Logarithme népérien">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=2" title="Modifier la section : Logarithme népérien">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/Fichier:John_Napier.jpg"><img alt="" class="thumbimage" data-file-height="400" data-file-width="317" decoding="async" height="215" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e3/John_Napier.jpg/170px-John_Napier.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e3/John_Napier.jpg/255px-John_Napier.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/e/e3/John_Napier.jpg 2x" width="170"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:John_Napier.jpg" title="Agrandir"></a></div><a href="/wiki/John_Napier" title="John Napier">John Napier</a>.</div></div></div>
<p>Au début du <a href="/wiki/XVIIe_si%C3%A8cle" title="XVIIe siècle"><abbr class="abbr" title="17ᵉ siècle"><span class="romain">XVII</span><sup style="font-size:72%">e</sup></abbr> siècle</a>, le mathématicien écossais <a href="/wiki/John_Napier" title="John Napier">John Napier</a> construit les premières <a href="/wiki/Table_de_logarithmes" title="Table de logarithmes">tables de logarithmes</a>, qui permettent de simplifier des calculs de <a href="/wiki/Produit_(math%C3%A9matiques)" title="Produit (mathématiques)">produits</a> et <a href="/wiki/Division" title="Division">quotients</a> mais aussi <a href="/wiki/Racine_carr%C3%A9e" title="Racine carrée">racines carrées</a>, <a href="/wiki/Racine_cubique" title="Racine cubique">cubiques</a> et autres. Elles consistent à associer à chaque nombre d'une liste un autre nombre (appelé <i>logarithme</i>), de façon qu'une relation de <a href="/wiki/Proportionnalit%C3%A9" title="Proportionnalité">proportionnalité</a> entre quatre termes de la première liste se traduise par des différences égales entre les termes correspondants de la seconde liste<sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup> : si <span class="texhtml"><i>a</i></span>, <span class="texhtml"><i>b</i></span>, <span class="texhtml"><i>c</i></span> et <span class="texhtml"><i>d</i></span> ont pour logarithmes respectifs <span class="texhtml"><i>A</i></span>, <span class="texhtml"><i>B</i></span>, <span class="texhtml"><i>C</i></span> et <span class="texhtml"><i>D</i></span>, alors la relation <span class="texhtml"><span class="frac nowrap"><sup style="font-size: 70%; vertical-align: 0.4em;"><i>a</i></sup>⁄<sub style="font-size: 70%; vertical-align: 0em;"><i>b</i></sub></span> = <span class="frac nowrap"><sup style="font-size: 70%; vertical-align: 0.4em;"><i>c</i></sup>⁄<sub style="font-size: 70%; vertical-align: 0em;"><i>d</i></sub></span></span> est équivalente à la relation <span class="texhtml"><i>A </i>− <i>B</i> = <i>C </i>− <i>D</i></span>.
</p>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a href="/wiki/Table_de_logarithmes" title="Table de logarithmes">Table de logarithmes</a>.</div></div>
<p>Plus précisément, Napier fixe un <i>rayon</i> initial de dix millions<sup class="reference" id="cite_ref-6"><a href="#cite_note-6"><span class="cite_crochet">[</span>d<span class="cite_crochet">]</span></a></sup> et construit une liste dans laquelle chaque nombre permet de calculer le suivant en lui soustrayant un dix-millionième de sa valeur. Ces opérations successives sont donc des multiplications itérées par <span class="texhtml">1 – 10<sup>–7</sup></span> et la liste constitue une <a href="/wiki/Suite_g%C3%A9om%C3%A9trique" title="Suite géométrique">suite géométrique</a> de premier terme 10<sup>7</sup>. Le logarithme de chaque nombre de la liste étant son rang d'apparition, la formule du logarithme ainsi obtenu par Napier s'écrit alors :

<h3><span id="Logarithme_d.C3.A9cimal"></span><span class="mw-headline" id="Logarithme_décimal">Logarithme décimal</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=3" title="Modifier la section : Logarithme décimal">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=3" title="Modifier la section : Logarithme décimal">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>En 1624, <a href="/wiki/Henry_Briggs" title="Henry Briggs">Henry Briggs</a>, correspondant avec Napier, modifie les paramètres de construction des tables de logarithmes. D'abord il fixe à 0 le logarithme de 1, ce qui revient à choisir un rayon unitaire. Ses tables transforment alors les produits en sommes, ce qui s'écrit en formulation moderne : <span class="texhtml">log(<i>ab</i>) = log(<i>a</i>) + log(<i>b</i>)</span>. Ensuite, il fixe à 1 le logarithme de 10, ce qui fait qu'une multiplication par 10 d'un nombre se traduit par un ajout d'une unité à son logarithme.
</p><p>Briggs obtient ainsi une table de valeurs du <a href="/wiki/Logarithme_d%C3%A9cimal" title="Logarithme décimal">logarithme décimal</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>, fondé sur le système de numération en base 10, mais la notion de <a href="/wiki/Fonction_(math%C3%A9matiques)" title="Fonction (mathématiques)">fonction</a> n'a pas encore émergé à l'époque. En particulier, il n'y a pas de trace d'une évaluation d'un <a class="mw-redirect" href="/wiki/Taux_d%27accroissement" title="Taux d'accroissement">taux d'accroissement</a> en 1, qui aurait pu faire apparaître une approximation de <span class="texhtml">log<sub>10</sub>(e)</span>.
</p>
<h3><span class="mw-headline" id="Logarithme_naturel">Logarithme naturel</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=4" title="Modifier la section : Logarithme naturel">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=4" title="Modifier la section : Logarithme naturel">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>En 1647, <a href="/wiki/Gr%C3%A9goire_de_Saint-Vincent" title="Grégoire de Saint-Vincent">Grégoire de Saint-Vincent</a> met en évidence une relation analogue à celle du logarithme entre les aires de domaines délimités par une branche d'<a href="/wiki/Hyperbole_(math%C3%A9matiques)" title="Hyperbole (mathématiques)">hyperbole</a> et son <a href="/wiki/Asymptote" title="Asymptote">asymptote</a>. En 1661, <a href="/wiki/Christian_Huygens" title="Christian Huygens">Christian Huygens</a> fait le lien entre les logarithmes et la <a href="/wiki/Quadrature_(math%C3%A9matiques)" title="Quadrature (mathématiques)">quadrature</a> de l'hyperbole, et en particulier celle d'équation <span class="texhtml"><i>x</i> <i>y</i> = 1</span>. Le <a href="/wiki/Logarithme_naturel" title="Logarithme naturel">logarithme naturel</a> est donc mis en évidence, mais sa base (<span class="texhtml">e</span>) n'est pas identifiée.
</p><p>C’est dans une lettre de <a href="/wiki/Gottfried_Wilhelm_Leibniz" title="Gottfried Wilhelm Leibniz">Leibniz</a> à Huygens que ce nombre est enfin identifié comme la base du logarithme naturel<sup class="reference" id="cite_ref-9"><a href="#cite_note-9"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>, vers 1690, mais Leibniz le note <i>b</i>.
</p>
<h3><span class="mw-headline" id="Nouvelle_notation">Nouvelle notation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=5" title="Modifier la section : Nouvelle notation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=5" title="Modifier la section : Nouvelle notation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>Euler, dans un article écrit en 1727 ou 1728<sup class="reference" id="cite_ref-10"><a href="#cite_note-10"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>, est le premier à noter <i>e</i> <span class="citation">« le nombre dont le logarithme est l'unité »</span>. Il utilise cette notation, avec la même définition, dans une lettre à <a href="/wiki/Christian_Goldbach" title="Christian Goldbach">Goldbach</a> en 1731<sup class="reference" id="cite_ref-11"><a href="#cite_note-11"><span class="cite_crochet">[</span>7<span class="cite_crochet">]</span></a></sup>.
</p><p>Le choix de la lettre <i>e</i> comme un hommage au nom d'Euler lui-même<sup class="reference" id="cite_ref-12"><a href="#cite_note-12"><span class="cite_crochet">[</span>8<span class="cite_crochet">]</span></a></sup> étant par conséquent peu probable, d'autres suppositions ont été avancées : première voyelle ou première lettre non utilisée dans un <a class="mw-redirect" href="/wiki/Calcul_litt%C3%A9ral" title="Calcul littéral">calcul littéral</a>, initiale de « exponentielle », <abbr class="abbr" title="et cetera">etc.</abbr><sup class="reference" id="cite_ref-13"><a href="#cite_note-13"><span class="cite_crochet">[</span>9<span class="cite_crochet">]</span></a></sup>.
</p>

<h3><span id="Irrationalit.C3.A9"></span><span class="mw-headline" id="Irrationalité">Irrationalité</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=9" title="Modifier la section : Irrationalité">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=9" title="Modifier la section : Irrationalité">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La première preuve de l'<a href="/wiki/Nombre_irrationnel" title="Nombre irrationnel">irrationalité</a> de <span class="texhtml">e</span> est due à Euler (<span title="Voir la section Fraction continue"><a href="#Fraction_continue">voir <i>infra</i></a></span>). <a href="/wiki/Joseph_Fourier" title="Joseph Fourier">Fourier</a> donna la preuve plus simple suivante<sup class="reference" id="cite_ref-19"><a href="#cite_note-19"><span class="cite_crochet">[</span>15<span class="cite_crochet">]</span></a></sup><sup class="reference cite_virgule">,</sup><sup class="reference" id="cite_ref-20"><a href="#cite_note-20"><span class="cite_crochet">[</span>16<span class="cite_crochet">]</span></a></sup>, en utilisant la décomposition de <span class="texhtml">e</span> par la série exponentielle et en <a href="/wiki/Raisonnement_par_l%27absurde" title="Raisonnement par l'absurde">raisonnant par l'absurde</a>.
</p><p>Il s'agit de prouver que pour tout entier <span class="texhtml"><i>n</i> &gt; 0</span>, le nombre <span class="texhtml"><i>n</i>e</span> n'est pas entier. Pour cela, il montre que <span class="texhtml"><i>n</i>!e</span> lui-même n'est pas entier, en le décomposant sous la forme

Cette conclusion étant valable quel que soit l'entier <span class="texhtml"><i>n</i> &gt; 0</span>, <span class="texhtml">e</span> est irrationnel.
</p>

</p>
<h3><span class="mw-headline" id="Transcendance">Transcendance</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=11" title="Modifier la section : Transcendance">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=11" title="Modifier la section : Transcendance">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Pour aller plus loin, c’est-à-dire que pour montrer que <span class="texhtml">e</span> n’est solution d’aucune équation du troisième degré à coefficients rationnels puis, qu’il est <a href="/wiki/Nombre_transcendant" title="Nombre transcendant">transcendant</a>, ce qui signifie qu’il n’est solution d’aucune équation polynomiale à coefficients rationnels, de nouvelles idées sont nécessaires.
</p><p>La transcendance de <span class="texhtml">e</span> fut établie par <a href="/wiki/Charles_Hermite" title="Charles Hermite">Charles Hermite</a> en 1873<sup class="reference" id="cite_ref-24"><a href="#cite_note-24"><span class="cite_crochet">[</span>h<span class="cite_crochet">]</span></a></sup>, par une méthode préfigurant la théorie des <a href="/wiki/Approximant_de_Pad%C3%A9" title="Approximant de Padé">approximants de Padé</a>, développée en 1892 dans la thèse de son élève <a href="/wiki/Henri_Pad%C3%A9" title="Henri Padé">Henri Padé</a>. Les différents <a href="/wiki/Approximant_de_Pad%C3%A9_de_la_fonction_exponentielle" title="Approximant de Padé de la fonction exponentielle">approximants de Padé de la fonction exponentielle</a> fournissent en effet de nombreuses expressions de <span class="texhtml">e</span> sous forme de fractions continues généralisées.
</p><p>Puisque <span class="texhtml">e</span> est transcendant, <span class="texhtml">e<sup><i>r</i></sup></span> l'est aussi, pour tout rationnel <span class="texhtml mvar" style="font-style:italic;">r</span> non nul (et plus généralement : <span class="texhtml"><i>f</i>(e)</span>, pour toute <a href="/wiki/Fonction_alg%C3%A9brique" title="Fonction algébrique">fonction algébrique</a> <span class="texhtml mvar" style="font-style:italic;">f</span> non constante).
</p><p>Le <a href="/wiki/Th%C3%A9or%C3%A8me_de_Gelfond-Schneider" title="Théorème de Gelfond-Schneider">théorème de Gelfond-Schneider</a> permet de démontrer également que, par exemple, <span class="texhtml">e<sup>π</sup></span> est transcendant, mais on ne sait pas encore, en 2020, si <span class="texhtml">e<sup>e</sup></span> et <span class="texhtml">π<sup>e</sup></span> sont transcendants ou non (il est cependant conjecturé que tous les nombres de cette forme le sont).
</p><p>Il est également <a href="/wiki/Conjecture" title="Conjecture">conjecturé</a> que <span class="texhtml">e</span> est un <a href="/wiki/Nombre_normal" title="Nombre normal">nombre normal</a>.
</p>

<h3><span id="Probl.C3.A8me_des_int.C3.A9r.C3.AAts_compos.C3.A9s"></span><span class="mw-headline" id="Problème_des_intérêts_composés">Problème des intérêts composés</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=13" title="Modifier la section : Problème des intérêts composés">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=13" title="Modifier la section : Problème des intérêts composés">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>En 1685, <a href="/wiki/Jacques_Bernoulli" title="Jacques Bernoulli">Jacques Bernoulli</a> étudie le problème des <a href="/wiki/Int%C3%A9r%C3%AAts_compos%C3%A9s" title="Intérêts composés">intérêts composés</a> en progression continue : si un montant <span class="texhtml"><i>a</i></span> rapporte un montant <span class="texhtml"><i>b</i></span> d'intérêts au bout d'un temps fini, on peut considérer que ces intérêts s'acquièrent linéairement en fonction du temps. Mais sur l'intervalle de temps considéré, ces intérêts devraient eux-mêmes produire des intérêts, et ainsi de suite. Bernoulli obtient ainsi une expression qui évoque le développement en série exponentielle<sup class="reference" id="cite_ref-25"><a href="#cite_note-25"><span class="cite_crochet">[</span>17<span class="cite_crochet">]</span></a></sup>.
</p>
<h2><span id="D.C3.A9cimales_connues"></span><span class="mw-headline" id="Décimales_connues">Décimales connues</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=14" title="Modifier la section : Décimales connues">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=14" title="Modifier la section : Décimales connues">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Parmi les rationnels de numérateur et dénominateur inférieurs à 1 000, le plus proche de <span class="texhtml">e</span> est<sup class="reference" id="cite_ref-26"><a href="#cite_note-26"><span class="cite_crochet">[</span>18<span class="cite_crochet">]</span></a></sup> <span class="texhtml"><span style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">878</span><span style="position:absolute;left:-10000px;top:auto;width:1px;height:1px;overflow:hidden">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">323</span></span></span> ≈ <span style="white-space: nowrap">2,718<span style="margin-left: 0.25em">27</span></span>.
</p><p>La valeur numérique de <span class="texhtml">e</span> tronquée à 15 <a href="/wiki/Syst%C3%A8me_d%C3%A9cimal" title="Système décimal">décimales</a> est<sup class="reference" id="cite_ref-OEIS_27-0"><a href="#cite_note-OEIS-27"><span class="cite_crochet">[</span>19<span class="cite_crochet">]</span></a></sup> <span style="white-space: nowrap">2,718<span style="margin-left: 0.25em">281</span><span style="margin-left: 0.25em">828</span><span style="margin-left: 0.25em">459</span><span style="margin-left: 0.25em">045</span></span>.
</p><p>Le nombre de décimales connues de la constante <span class="texhtml">e</span> a beaucoup augmenté au cours des dernières décennies. Cette précision est due à l’augmentation des performances des ordinateurs ainsi qu’au perfectionnement des algorithmes<sup class="reference" id="cite_ref-28"><a href="#cite_note-28"><span class="cite_crochet">[</span>20<span class="cite_crochet">]</span></a></sup><sup class="reference cite_virgule">,</sup><sup class="reference" id="cite_ref-29"><a href="#cite_note-29"><span class="cite_crochet">[</span>21<span class="cite_crochet">]</span></a></sup>.
</p>

<h2><span class="mw-headline" id="Dans_la_culture_informatique">Dans la culture informatique</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=E_(nombre)&amp;veaction=edit&amp;section=15" title="Modifier la section : Dans la culture informatique">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=E_(nombre)&amp;action=edit&amp;section=15" title="Modifier la section : Dans la culture informatique">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Le nombre <span class="texhtml">e</span> fait l'objet de nombreux hommages dans le milieu informatique.
</p><p>Pour son <a href="/wiki/Introduction_en_bourse" title="Introduction en bourse">introduction en bourse</a> en 2004, <a href="/wiki/Google" title="Google">Google</a> a annoncé vouloir lever non pas un chiffre rond comme c'est généralement le cas, mais 2 718 281 828 <abbr class="abbr" title="dollar">$</abbr>, soit <span class="texhtml">e</span> milliards de dollars (au dollar près)<sup class="reference" id="cite_ref-41"><a href="#cite_note-41"><span class="cite_crochet">[</span>33<span class="cite_crochet">]</span></a></sup>. Google est aussi à l'origine d'une campagne de recrutement originale en juillet 2004 : des panneaux mentionnant « {first 10-digit prime found in the consecutive digits of e}.com » ({premier nombre premier à 10 chiffres trouvé dans les décimales successives de e}.com) affichés dans un premier temps dans la <a href="/wiki/Silicon_Valley" title="Silicon Valley">Silicon Valley</a>, puis à <a href="/wiki/Cambridge_(Massachusetts)" title="Cambridge (Massachusetts)">Cambridge</a>, <a href="/wiki/Seattle" title="Seattle">Seattle</a> et <a href="/wiki/Austin_(Texas)" title="Austin (Texas)">Austin</a> incitaient les curieux à se rendre sur le site aujourd'hui disparu 7427466391.com. Là, le visiteur devait résoudre un problème encore plus difficile, qui lui-même le renvoyait sur le site <a href="/wiki/Google_Labs" title="Google Labs">Google Labs</a> où il était invité à soumettre un CV<sup class="reference" id="cite_ref-42"><a href="#cite_note-42"><span class="cite_crochet">[</span>34<span class="cite_crochet">]</span></a></sup><sup class="reference cite_virgule">,</sup><sup class="reference" id="cite_ref-43"><a href="#cite_note-43"><span class="cite_crochet">[</span>35<span class="cite_crochet">]</span></a></sup>. Le premier nombre premier à dix chiffres dans les décimales de <span class="texhtml">e</span> est 7 427 466 391, qui commence à la <abbr class="abbr" title="Quatre-vingt-dix-neuvième (nonante-neuvième)">99<sup>e</sup></abbr> décimale<sup class="reference" id="cite_ref-OEIS_27-1"><a href="#cite_note-OEIS-27"><span class="cite_crochet">[</span>19<span class="cite_crochet">]</span></a></sup>.
</p><p>L'informaticien <a href="/wiki/Donald_Knuth" title="Donald Knuth">Donald Knuth</a> a numéroté les différentes versions de son programme <a href="/wiki/Metafont" title="Metafont">Metafont</a> d'après les décimales de <span class="texhtml">e</span> : 2, 2,7, 2,71, 2,718, et ainsi de suite. De la même façon, les numéros de versions de son programme <a href="/wiki/TeX" title="TeX">TeX</a> approchent <a href="/wiki/Pi" title="Pi"><span class="texhtml">π</span></a><sup class="reference" id="cite_ref-44"><a href="#cite_note-44"><span class="cite_crochet">[</span>36<span class="cite_crochet">]</span></a></sup>.
</p>
