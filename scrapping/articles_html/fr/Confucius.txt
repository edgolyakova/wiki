<p><b>Confucius</b> (<small>en <a href="/wiki/Fran%C3%A7ais" title="Français">français</a> : </small><span class="API" style="font-family:'DejaVu Sans','Doulos SIL','Lucida Grande','Segoe UI','Arial Unicode MS','Adobe Pi Std','Lucida Sans Unicode','Chrysanthi Unicode',Code2000,Gentium,GentiumAlt,'TITUS Cyberbit Basic','Bitstream Vera Sans','Bitstream Cyberbit','Hiragino Kaku Gothic Pro','Matrix Unicode',sans-serif;"><a href="/wiki/Aide:Alphabet_phon%C3%A9tique_international" title="Aide:Alphabet phonétique international">/kɔ̃.fy.sjys/</a></span> ; <a href="/wiki/Sinogramme_simplifi%C3%A9" title="Sinogramme simplifié">chinois simplifié</a> : <span class="lang-zh-hans" lang="zh-hans">孔子</span> ; <a href="/wiki/Hanyu_pinyin" title="Hanyu pinyin">pinyin</a> : <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">Kǒng Zǐ</span></span> ; <a href="/wiki/Romanisation_Wade-Giles" title="Romanisation Wade-Giles">Wade</a> : <i><span class="lang-zh-latn-wadegile" lang="zh-latn-wadegile">K'ung³-tzu³</span></i>), né le <time class="nowrap date-lien bday" data-sort-value="U-0550-09-22" datetime="U-0550-09-22"><a href="/wiki/28_septembre" title="28 septembre">28 septembre</a> <a class="mw-redirect" href="/wiki/551_av._J.-C." title="551 av. J.-C.">551 <abbr class="abbr" title="551 avant Jésus-Christ">av. J.-C.</abbr></a></time> à Zou (陬) et mort le <time class="nowrap date-lien dday" data-sort-value="U-0478-04-06" datetime="U-0478-04-06"><a href="/wiki/11_avril" title="11 avril">11 avril</a> <a href="/wiki/479_av._J.-C." title="479 av. J.-C.">479 <abbr class="abbr" title="479 avant Jésus-Christ">av. J.-C.</abbr></a></time> à <a href="/wiki/Qufu" title="Qufu">Qufu</a> (曲阜) dans l’actuelle province du <a href="/wiki/Shandong" title="Shandong">Shandong</a>, est un <a href="/wiki/Philosophie_chinoise" title="Philosophie chinoise">philosophe chinois</a>. Son patronyme est Kong, son prénom Qiu, et son <a href="/wiki/Pr%C3%A9nom_social" title="Prénom social">prénom social</a> Zhongni. Il est le personnage historique qui a le plus marqué la civilisation chinoise, et est considéré comme le premier « éducateur » de la <a href="/wiki/Chine" title="Chine">Chine</a>.
</p><p>Son enseignement a donné naissance au <a href="/wiki/Confucianisme" title="Confucianisme">confucianisme</a>, doctrine politique et sociale érigée en <a href="/wiki/Religion_d%27%C3%89tat" title="Religion d'État">religion d'État</a> dès la <a href="/wiki/Dynastie_Han" title="Dynastie Han">dynastie Han</a> et qui ne fut officiellement bannie qu'au début du <abbr class="abbr" title="20ᵉ siècle"><span class="romain">XX</span><sup style="font-size:72%">e</sup></abbr> siècle, avec une résurgence en 1973 (voir <a href="/wiki/Critique_de_Lin_Piao_et_de_Confucius" title="Critique de Lin Piao et de Confucius">Critique de Lin Piao et de Confucius</a>). Ses principaux disciples sont nommés les <a href="/wiki/Douze_Philosophes" title="Douze Philosophes">Douze Philosophes</a> et honorés dans les <a href="/wiki/Temple_confuc%C3%A9en" title="Temple confucéen">temples confucéens</a>.
</p><p>Il est généralement appelé <i>Kǒngzǐ</i> (孔子) ou <i>Kǒng Fūzǐ</i> (孔夫子) par les Chinois, ce qui signifie « Maître Kong » et a été <a href="/wiki/Romanisation_des_langues_chinoises#Romanisations_des_premiers_missionnaires" title="Romanisation des langues chinoises">latinisé</a> en <i>Confucius</i> par les <a href="/wiki/Compagnie_de_J%C3%A9sus" title="Compagnie de Jésus">Jésuites</a>. Les <a href="/wiki/Cor%C3%A9ens" title="Coréens">Coréens</a> l'appellent <i>Kong-ja</i>, les <a href="/wiki/Japonais_(peuple)" title="Japonais (peuple)">Japonais</a>, <i>Kôshi, </i>et les <a href="/wiki/Vietnamiens" title="Vietnamiens">Vietnamiens</a>, <i>Khổng Tử</i>.
</p>

<h2><span class="mw-headline" id="Sa_vie">Sa vie</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=1" title="Modifier la section : Sa vie">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=1" title="Modifier la section : Sa vie">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Sa figure mi-historique, mi-légendaire est retracée dans sa première <a href="/wiki/Biographie" title="Biographie">biographie</a> issue de <i><a href="/wiki/Shiji" title="Shiji">Shiji</a></i> (<span class="lang-zh-hant" lang="zh-hant">史記</span> / <span class="lang-zh-hans" lang="zh-hans">史记</span>, <i>Shǐjì</i>), œuvre de l'<a href="/wiki/Historien" title="Historien">historien</a> chinois <a href="/wiki/Sima_Qian" title="Sima Qian">Sima Qian</a> écrite de <a href="/wiki/109_av._J.-C." title="109 av. J.-C.">109</a> à <a href="/wiki/91_av._J.-C." title="91 av. J.-C.">91 av. J.-C.</a>, soit plus de quatre siècles après sa disparition<sup class="reference" id="cite_ref-1"><a href="#cite_note-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>.
</p>
<h3><span class="mw-headline" id="Noms">Noms</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=2" title="Modifier la section : Noms">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=2" title="Modifier la section : Noms">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Il avait pour nom personnel Kong Qiu (孔丘), et pour nom social Zhongni (仲尼). Aussi appelé respectueusement Kongfuzi (孔夫子 : « maître Kong »), ou simplement Kongzi (孔子)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>Le nom latinisé « <b>Confucius</b> », dérivé de Kongfuzi, a été créé au <a href="/wiki/XVIe_si%C3%A8cle" title="XVIe siècle"><abbr class="abbr" title="16ᵉ siècle"><span class="romain">XVI</span><sup style="font-size:72%">e</sup></abbr> siècle</a> par des <a href="/wiki/Missionnaire_chr%C3%A9tien" title="Missionnaire chrétien">missionnaires</a> <a href="/wiki/J%C3%A9suitisme" title="Jésuitisme">jésuites</a> en Chine, probablement par <a href="/wiki/Matteo_Ricci" title="Matteo Ricci">Matteo Ricci</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>.
</p>
<h3><span class="mw-headline" id="Origines">Origines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=3" title="Modifier la section : Origines">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=3" title="Modifier la section : Origines">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La famille Kong était originaire de l'<a href="/wiki/Song_(%C3%89tat)" title="Song (État)">État de Song</a>. Kong Fu Jia, son arrière-grand-père, y était le ministre de la guerre. Après que celui-ci fut assassiné, son fils Fang Shu se réfugia dans l'<a href="/wiki/Lu_(%C3%89tat)" title="Lu (État)">État de Lu</a>, où il mena une carrière militaire. Son fils, <a href="/wiki/Shu_Lianghe" title="Shu Lianghe">Shu Lianghe</a>, allait suivre ses traces et faire lui aussi une brillante carrière militaire. La famille Kong était une famille de grands guerriers. Confucius, fils de Shu Lianghe, fut le premier de sa lignée à abandonner la voie des armes.
</p>
<h3><span class="mw-headline" id="Naissance">Naissance</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=4" title="Modifier la section : Naissance">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=4" title="Modifier la section : Naissance">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Confucius est né le 28 septembre 551 avant notre ère à Zou (陬), ville dont son père était le gouverneur, non loin de la ville de <a href="/wiki/Qufu" title="Qufu">Qufu</a> (曲阜), pays de Lu, actuelle province de <a href="/wiki/Shandong" title="Shandong">Shandong</a>. Sa mère Zheng Zai étant allée prier sur le mont Qiū (丘) le prénomma ainsi.
</p>
<h4><span id="Une_naissance_l.C3.A9gendaire"></span><span class="mw-headline" id="Une_naissance_légendaire">Une naissance légendaire</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=5" title="Modifier la section : Une naissance légendaire">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=5" title="Modifier la section : Une naissance légendaire">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>D'après la légende, des événements extraordinaires auraient précédé sa naissance ; une <a href="/wiki/Licorne" title="Licorne">licorne</a> aurait en outre prédit sa naissance. Elle vomit une tablette de <a href="/wiki/Jade" title="Jade">jade</a> qui prédisait la naissance d'un enfant qui soutiendrait la déclinante <a href="/wiki/Dynastie_Zhou" title="Dynastie Zhou">dynastie Zhou</a>. Au cours de la nuit de sa naissance, deux <a href="/wiki/Dragon" title="Dragon">dragons</a> se seraient posés sur le toit de sa maison. Cinq vieillards, <span class="need_ref" style="cursor:help;" title="Ce passage est confus.">qui restituaient les essences des Cinq Planètes</span><sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Wikip%C3%A9dia:Style_encyclop%C3%A9dique#Clair" title="Wikipédia:Style encyclopédique">[pas clair]</a></sup>, arrivèrent dans sa cour. Des chants célestes se seraient fait entendre. Puis finalement, des voix prophétisèrent : <i>Le Ciel favorisera la naissance d'un fils saint</i>.
</p><p>Les historiens chinois, depuis deux mille ans, parlent de ce temps très ancien comme étant celui des « <a href="/wiki/P%C3%A9riode_des_Printemps_et_Automnes" title="Période des Printemps et Automnes">Printemps et des Automnes</a> » (春秋), faisant ainsi référence à une chronique racontant ce qui advint entre 771 et 481 <abbr class="abbr nowrap" title="avant Jésus-Christ">av. J.-C.</abbr> précisément dans cette région que l'on nommait alors le pays de <a href="/wiki/Lu_(%C3%89tat)" title="Lu (État)">Lu</a>.
</p>
<h3><span class="mw-headline" id="Famille">Famille</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=6" title="Modifier la section : Famille">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=6" title="Modifier la section : Famille">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Selon la tradition, son père, <a href="/wiki/Shu_Lianghe" title="Shu Lianghe">Shu Lianghe</a> (<span class="lang-zh" lang="zh">叔梁紇</span>), était un descendant de <a class="new" href="/w/index.php?title=Yi_Yin&amp;action=edit&amp;redlink=1" title="Yi Yin (page inexistante)">Yi Yin</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Yi_Yin" title="en:Yi Yin"><span class="indicateur-langue" title="Article en anglais : « Yi Yin »">(en)</span></a> (<span class="lang-zh" lang="zh">伊尹</span>), premier ministre de <a href="/wiki/Cheng_Tang" title="Cheng Tang">Cheng Tang</a> (<span class="lang-zh" lang="zh">湯</span>), le fondateur de la <a href="/wiki/Dynastie_Shang" title="Dynastie Shang">dynastie Shang</a> (<span class="lang-zh" lang="zh">商朝</span>).
</p><p>Gouverneur de la principauté de <a href="/wiki/Lu_(%C3%89tat)" title="Lu (État)">Lu</a> (<span class="lang-zh-hans" lang="zh-hans">鲁国</span> / <span class="lang-zh-hant" lang="zh-hant">魯國</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">lǔguó</span></span>), dans le sud-est de l’actuelle province du <a href="/wiki/Shandong" title="Shandong">Shandong</a>, il épousa en secondes noces à l'âge de 65 ans la jeune Zheng Zai, alors âgée de 15 ans.
</p><p>Il mourut alors que Confucius n’avait que trois ans, laissant sa famille dans la pauvreté.
</p>
<h3><span class="mw-headline" id="Descendance">Descendance</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=7" title="Modifier la section : Descendance">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=7" title="Modifier la section : Descendance">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La famille de Confucius, les Kong, possède l'<a href="/wiki/Arbre_g%C3%A9n%C3%A9alogique" title="Arbre généalogique">arbre généalogique</a> le plus grand du monde. Les filiations de père en fils, qui comptent aujourd'hui 83 générations, ont été enregistrés depuis la mort de Confucius. Selon le Comité de Compilation de la Généalogie de Confucius, il compte 2 millions de descendants connus et enregistrés, sur un total de descendants estimés de 3 millions<sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>.
</p>
<h2><span class="mw-headline" id="Apparence_physique">Apparence physique</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=8" title="Modifier la section : Apparence physique">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=8" title="Modifier la section : Apparence physique">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Selon le <a href="/wiki/Shiji" title="Shiji">Shiji</a>, Confucius mesurait six pieds neuf pouces de haut, ce qui équivaut à 2,06 mètres. Cette haute stature lui venait de son père qui mesurait lui-même plus de 2,20 mètres.
</p>
<h2><span class="mw-headline" id="Enfance_et_jeunesse">Enfance et jeunesse</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=9" title="Modifier la section : Enfance et jeunesse">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=9" title="Modifier la section : Enfance et jeunesse">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Dès l’âge de dix-sept ans, grâce à un goût précoce pour les livres et les <a href="/wiki/Rite" title="Rite">rites</a>, Confucius serait devenu <a href="/wiki/Pr%C3%A9cepteur" title="Précepteur">précepteur</a>. Il se maria à dix-neuf ans et eut son premier fils, Kong Li (<span class="lang-zh" lang="zh">孔鯉</span>) un an plus tard. Celui-ci fut suivi de deux filles. Pour vivre, il effectuait probablement des tâches administratives pour le chef de province.
</p>
<h2><span id=".C3.82ge_adulte"></span><span class="mw-headline" id="Âge_adulte">Âge adulte</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=10" title="Modifier la section : Âge adulte">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=10" title="Modifier la section : Âge adulte">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Relations_avec_le_duc_Jing_de_Qi">Relations avec le duc Jing de Qi</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=11" title="Modifier la section : Relations avec le duc Jing de Qi">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=11" title="Modifier la section : Relations avec le duc Jing de Qi">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Confucius qui séjournait au <a href="/wiki/Qi_(%C3%89tat)" title="Qi (État)">pays de Qi</a> (<span class="lang-zh-hans" lang="zh-hans">齐国</span> / <span class="lang-zh-hant" lang="zh-hant">齊國</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">qíguó</span></span>) fut reçu à cette occasion par <a href="/wiki/Jiang_Ziya" title="Jiang Ziya">Jiang Ziya</a> (<span class="lang-zh" lang="zh">姜子牙</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">jiāng zǐyá</span></span>), duc Jiang de Qi, qui le fit mander. Les deux hommes dialoguèrent à propos d'affaires d'État, mais aussi sur des sujets comme la <a href="/wiki/Morale" title="Morale">morale</a> et les us et <a href="/wiki/Coutume" title="Coutume">coutumes</a>. Ils se parlèrent régulièrement pendant tout le séjour de Confucius. Le duc Jing, qui aimait les idées de Confucius, était décidé à lui confier des charges dans son gouvernement mais en fut finalement dissuadé par un de ses conseillers, qui craignait probablement pour son poste. Finalement, quand le duc Jiang rencontra une dernière fois Confucius, il lui dit « Je suis trop vieux pour vous employer. » Par la suite, Confucius retourna au pays de Lu<sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Aide:R%C3%A9f%C3%A9rence_n%C3%A9cessaire" title="Aide:Référence nécessaire">[<abbr class="abbr" title="référence">réf.</abbr> souhaitée]</a></sup>.
</p>
<h3><span class="mw-headline" id="Rencontre_avec_Lao_Tseu">Rencontre avec <a href="/wiki/Lao_Tseu" title="Lao Tseu">Lao Tseu</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=12" title="Modifier la section : Rencontre avec Lao Tseu">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=12" title="Modifier la section : Rencontre avec Lao Tseu">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La <a href="/wiki/L%C3%A9gende" title="Légende">légende</a> veut qu’il ait rencontré <a href="/wiki/Lao_Tseu" title="Lao Tseu">Lao Zi</a> (老子), père du <a href="/wiki/Tao%C3%AFsme" title="Taoïsme">taoïsme</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>. Il serait allé le trouver, à <a href="/wiki/Luoyang" title="Luoyang">Luoyang</a>, pour s'enrichir de connaissances concernant les <a href="/wiki/Rite_fun%C3%A9raire" title="Rite funéraire">rites de deuil</a>. Ils auraient eu un long échange et, au moment où Confucius allait le quitter, Lao Dan lui aurait dit :
</p><p><span class="citation">« Selon les traditions, les gens fortunés donnent des présents à leur hôte et les gens pauvres donnent des mots. N'étant pas aisé, je puis néanmoins vous donner des mots : un homme intelligent, grand observateur, se trouvera toujours en danger de mort, car il se plaît à parler des autres. Par son vaste savoir et son solide jugement, il en vient à découvrir ce que les autres ont de plus méprisable. Être fils comme être un simple sujet dépossède du soi. »</span><sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Aide:R%C3%A9f%C3%A9rence_n%C3%A9cessaire" title="Aide:Référence nécessaire"><span title="Ce passage nécessite une référence ; voir l'aide.">[réf. nécessaire]</span></a></sup>
</p><p>Après, Confucius resta sidéré et renonça à parler pendant trois jours - ou un mois - tellement Lao Zi l'avait troublé.
</p>

<h4><span id=".C3.80_la_cour_de_Lu"></span><span class="mw-headline" id="À_la_cour_de_Lu">À la cour de Lu</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=15" title="Modifier la section : À la cour de Lu">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=15" title="Modifier la section : À la cour de Lu">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Après la mort de sa mère en -530, il enseigna sa connaissance des textes anciens au petit groupe de disciples qui le suivait. Après quelques emplois subalternes à la cour du duc de Lu, il devint Grand Ministre de la Justice de Lu à l'âge de 53 ans. <a href="/wiki/Yuan_Xian" title="Yuan Xian">Yuan Xian</a> était le chef de la maisonnée de Confucius quand celui-ci exerçait la fonction de Ministre de la Justice de Lu. Après la mort de son maître, Yuan Xian partit dans l'<a href="/wiki/Wei_(Printemps_et_Automnes)" title="Wei (Printemps et Automnes)">État de Wei</a>, où il a vécu reclus et dans la pauvreté<sup class="reference" id="cite_ref-Huang1997204_6-0"><a href="#cite_note-Huang1997204-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.
</p><p>Cependant, après que ce duc eut préféré prendre du plaisir trois jours durant avec des danseuses au lieu d'assurer sa tâche de gouvernement, Confucius décida de quitter son poste de ministre et, en -497, partit pour treize années d’errance<sup class="reference" id="cite_ref-7"><a href="#cite_note-7"><span class="cite_crochet">[</span>7<span class="cite_crochet">]</span></a></sup>. Il rentra définitivement à Lu pour se consacrer jusqu’à sa mort, le 11 avril -479, à l’enseignement et à la compilation de textes anciens.
</p>

<h3><span class="mw-headline" id="Les_dangers_du_pays_de_Song">Les dangers du pays de Song</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=20" title="Modifier la section : Les dangers du pays de Song">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=20" title="Modifier la section : Les dangers du pays de Song">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p><span class="need_ref" style="cursor:help;" title="Ce passage nécessite une référence (demandé le 10 juillet 2020).">Confucius et ses disciples s'étaient rassemblés dans un bois pour se reposer et écouter l'enseignement du maître. Quand <a class="new" href="/w/index.php?title=Huan_Tui&amp;action=edit&amp;redlink=1" title="Huan Tui (page inexistante)">Huan Tui</a>, ministre de la guerre du <a href="/wiki/Song_(%C3%89tat)" title="Song (État)">pays de Song</a>, tenta de l'assassiner en faisant tomber un arbre sur lui. Confucius se contenta de l'éviter. Puis, quelque temps plus tard, accompagné de ses disciples, il quitta le pays de Song.</span><sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Aide:R%C3%A9f%C3%A9rence_n%C3%A9cessaire" title="Aide:Référence nécessaire">[réf. nécessaire]</a></sup>
</p>
<h3><span id="Pi.C3.A9g.C3.A9_entre_Chen_et_Cai"></span><span class="mw-headline" id="Piégé_entre_Chen_et_Cai">Piégé entre Chen et Cai</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=21" title="Modifier la section : Piégé entre Chen et Cai">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=21" title="Modifier la section : Piégé entre Chen et Cai">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Confucius et ses disciples étaient dans le <a href="/wiki/Chen_(%C3%89tat)" title="Chen (État)">pays de Chen</a> quand le roi de <a href="/wiki/Chu_(%C3%89tat)" title="Chu (État)">Chu</a> envoya des cadeaux à Confucius. Le roi Zhao de Chu voulait solliciter Confucius à exercer une charge dans son gouvernement. Cependant, Chen et Cai, qui étaient des ennemis de Chu, voulaient empêcher cela de se produire et cernèrent Confucius et ses disciples, les amenant dans un état de siège. Le roi de Chu l'ayant appris, qui tenait son camp à Chengfu, détacha un corps d'armée pour les dégager de leur position fâcheuse. L'opération fut un succès et Confucius et ses disciples purent s'échapper. Confucius était tout disposé à exercer une charge au sein du pays de Chu et voulut rencontrer son roi. Mais Zixi, grand conseiller de Chu, sentant sa position au sein du gouvernement de Chu menacée, ne ménagea pas ses efforts pour faire changer d'idée au roi de Chu qui, finalement, renonça.
</p>
<h3><span class="mw-headline" id="Les_dangers_du_bourg_de_Kuang">Les dangers du bourg de Kuang</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=22" title="Modifier la section : Les dangers du bourg de Kuang">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=22" title="Modifier la section : Les dangers du bourg de Kuang">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Étonnamment, Confucius ressemblait physiquement à un homme cruel nommé <a href="/wiki/Yang_Huo" title="Yang Huo">Yang Huo</a>. Or celui-ci avait commis des exactions contre la population du bourg de Kuang. Les gens de Kuang qui prenaient Confucius pour Yang Huo, le cernèrent et voulurent le pendre. Confucius réussit néanmoins à s'échapper.
</p>
<h2><span class="mw-headline" id="Le_retour_au_pays_de_Lu_et_mort">Le retour au <a href="/wiki/Lu_(%C3%89tat)" title="Lu (État)">pays de Lu</a> et mort</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=23" title="Modifier la section : Le retour au pays de Lu et mort">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=23" title="Modifier la section : Le retour au pays de Lu et mort">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Son disciple <a href="/wiki/Ran_Qiu" title="Ran Qiu">Ran Qiu</a>, avait été sollicité par <a class="new" href="/w/index.php?title=Ji_Kangzi&amp;action=edit&amp;redlink=1" title="Ji Kangzi (page inexistante)">Ji Kangzi</a> pour exercer une charge dans le gouvernement de Lu et il rentra à Lu. Après son arrivée, Ran Qiu demanda à Ji Kangzi, <a href="/wiki/Premier_ministre" title="Premier ministre">premier ministre</a> de l'<a href="/wiki/Lu_(%C3%89tat)" title="Lu (État)">État de Lu</a>, de faire revenir son maître. Ji Kangzi accepta. Confucius revint à Lu, mais n'allait pas exercer de charge dans le gouvernement. Il continua son enseignement jusqu'à sa mort.
</p>
<h2><span id="Le_pi.C3.A8ge_de_Yang_Huo"></span><span class="mw-headline" id="Le_piège_de_Yang_Huo">Le piège de <a href="/wiki/Yang_Huo" title="Yang Huo">Yang Huo</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=24" title="Modifier la section : Le piège de Yang Huo">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=24" title="Modifier la section : Le piège de Yang Huo">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Confuciustombqufu.jpg"><img alt="" class="thumbimage" data-file-height="1600" data-file-width="1200" decoding="async" height="293" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/07/Confuciustombqufu.jpg/220px-Confuciustombqufu.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/07/Confuciustombqufu.jpg/330px-Confuciustombqufu.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/07/Confuciustombqufu.jpg/440px-Confuciustombqufu.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Confuciustombqufu.jpg" title="Agrandir"></a></div>Tombe de Confucius</div></div></div>
<p><a href="/wiki/Yang_Huo" title="Yang Huo">Yang Huo</a> — tyran qui vivait en ce temps — était déterminé à rencontrer Confucius ; aussi décida-t-il de lui envoyer un cadeau au moment où Confucius n'était pas chez lui. D'après la tradition, un lettré qui n'est pas chez lui et qui reçoit un cadeau d'un seigneur doit aller chez ledit seigneur à pied le remercier de ses bonnes grâces. Or Confucius s'est résolu à ne pas le voir, estimant qu'il s'agissait d'un piège tendu par cet homme fourbe et cruel. Aussi décide-t-il d'aller le remercier au moment où il n'est pas chez lui, pour ne pas le voir. Cependant Yang Huo anticipe la manœuvre et prend les devants, tant et si bien que les deux se rencontrent sur le chemin. Quand il voit Yang Huo, il réalise qu'il est bel et bien piégé. Sa vivacité d'esprit le sort de cette mauvaise situation. Yang Huo voulait en fait solliciter Confucius à exercer des charges dans son pseudo-gouvernement, dans le but de semer le trouble dans le gouvernement légitime du prince Ting.
</p>
<h2><span id="Sa_pens.C3.A9e"></span><span class="mw-headline" id="Sa_pensée">Sa pensée</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=25" title="Modifier la section : Sa pensée">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=25" title="Modifier la section : Sa pensée">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>L'essentiel de la pensée de Confucius nous est parvenu à travers les <i><a href="/wiki/Entretiens_de_Confucius" title="Entretiens de Confucius">Analectes, ou Entretiens</a></i>, recueil de propos de Confucius et de ses disciples ainsi que de discussions entre eux, compilés par des disciples de deuxième génération, et à travers leur supplément, les <i><a href="/wiki/Entretiens_familiers_de_Confucius" title="Entretiens familiers de Confucius">Entretiens familiers de Confucius</a></i>.
</p><p>Bien qu’il n’ait jamais développé sa pensée de façon théorique, on peut dessiner à grands traits ce qu’étaient ses principales préoccupations et les solutions qu’il préconisait. Partant du constat qu’il n’est pas possible de <i>vivre avec les oiseaux et les bêtes sauvages</i>, et qu’il faut donc vivre en bonne société avec ses semblables, Confucius tisse un réseau de valeurs dont le but est l’harmonie des relations humaines. En son temps, la Chine était divisée en royaumes indépendants et belliqueux, les luttes pour l’<a href="/wiki/H%C3%A9g%C3%A9monie" title="Hégémonie">hégémonie</a> rendaient la situation instable et l’ancienne <a href="/wiki/Dynastie_Zhou" title="Dynastie Zhou">dynastie Zhou</a> avait perdu le rôle unificateur et pacificateur que lui conférait le <a href="/wiki/Mandat_du_Ciel" title="Mandat du Ciel">mandat du Ciel</a>. Confucius voulait donc restaurer ce mandat du Ciel qui conférait le pouvoir et l’efficacité à l’empereur vertueux. Cependant, bien qu’il affirme ne rien inventer et se contenter de transmettre la sagesse ancienne, Confucius a interprété les anciennes institutions selon ses aspirations, il a semé les graines de ce que certains auteurs appellent l'« humanisme chinois ».
</p><p>Mettant l’homme au centre de ses préoccupations et refusant de parler des esprits ou de la mort, Confucius n’a pas fondé de <a href="/wiki/Religion" title="Religion">religion</a> au sens occidental du terme, même si un culte lui a été dédié par la suite. Cherchant à fonder une <a href="/wiki/Confucianisme" title="Confucianisme">morale positive</a>, structurée par les « rites » et vivifiée par la « sincérité », mettant l’accent sur l’étude et la rectitude, Confucius représente pour les Chinois d’avant la Révolution l’éducateur par excellence, mais la lecture attentive des <i>Entretiens</i> montre qu’il n’a pas voulu s’ériger en maître à penser, et qu’au contraire il voulait développer chez ses disciples l’esprit critique et la réflexion personnelle : « Je lève un coin du voile, si l’étudiant ne peut découvrir les trois autres, tant pis pour lui. »
</p>
<div class="thumb tright"><div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/Fichier:Confucius_02.png"><img alt="" class="thumbimage" data-file-height="390" data-file-width="192" decoding="async" height="345" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2d/Confucius_02.png/170px-Confucius_02.png" srcset="//upload.wikimedia.org/wikipedia/commons/2/2d/Confucius_02.png 1.5x" width="170"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Confucius_02.png" title="Agrandir"></a></div>Confucius.</div></div></div>
<p>Un apport très important, et révolutionnaire en quelque sorte, de Confucius, est à chercher dans la notion de « <a href="/wiki/Junzi" title="Junzi">Junzi</a> » (« gentilhomme ») qui, avant lui, dénotait une noblesse de sang et dont il a modifié le sens pour le transformer en noblesse du cœur, un peu comme le mot anglais <i>gentleman</i>. Le concept central de la doctrine de Confucius est <i><a href="/wiki/Ren_(confucianisme)" title="Ren (confucianisme)">Ren</a></i>, la bienveillance, dont la pratique a pour norme <i><a href="/wiki/Li_(confucianisme)" title="Li (confucianisme)">Li</a></i>, la moralité<sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>8<span class="cite_crochet">]</span></a></sup>. Son enseignement, bien que principalement orienté vers la formation de futurs hommes de pouvoir, était ouvert à tous, pas seulement aux fils de princes. On peut faire remonter à cette impulsion de départ la longue tradition des <a href="/wiki/Examens_imp%C3%A9riaux" title="Examens impériaux">examens impériaux</a>, chargés de pourvoir l’État en hommes intègres et cultivés, que le plus humble paysan pouvait (en théorie) tenter. Bien que cette institution « <a href="/wiki/M%C3%A9ritocratie" title="Méritocratie">méritocratique</a> » ait subi différents avatars et distorsions, elle a certainement joué un rôle prépondérant dans la pérennité de la culture chinoise et dans la relative stabilité du <a href="/wiki/C%C3%A9leste_Empire" title="Céleste Empire">Céleste Empire</a> pendant deux millénaires.
</p><p>Selon Confucius, la soumission au père et au prince va de soi et garantit la cohésion des familles et du pays, mais elle s’accompagne d’un devoir de (respectueuses) remontrances si le père ou le prince vont dans la mauvaise direction. De très nombreux lettrés chinois, se réclamant à juste titre de l’enseignement de leur maître, ont péri ou été bannis, pour avoir osé critiquer l’empereur quand celui-ci, sous l’influence de courtisans ou de prêtres taoïstes, ne prenait plus soin de son peuple et laissait le pays sombrer dans la famine ou la guerre civile.
</p>
<h2><span id="Sa_post.C3.A9rit.C3.A9"></span><span class="mw-headline" id="Sa_postérité">Sa postérité</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=26" title="Modifier la section : Sa postérité">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=26" title="Modifier la section : Sa postérité">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>La postérité de Confucius, en Chine et en Extrême-Orient, ne saurait être sur-évaluée. Ses commentateurs et ses continuateurs proches comme <a href="/wiki/Mencius" title="Mencius">Mencius</a> et <a href="/wiki/Xun_Zi" title="Xun Zi">Xun Zi</a> ont formé un corps de doctrine, appelé <a href="/wiki/Confucianisme" title="Confucianisme">confucianisme</a>, choisi comme philosophie d’État en Chine pendant la <a href="/wiki/Dynastie_Han" title="Dynastie Han">dynastie Han</a>.
</p><p>Les <a href="/wiki/Mission_j%C3%A9suite_en_Chine" title="Mission jésuite en Chine">Jésuites en Chine</a> réalisent un transfert culturel de la pensée confucéenne aux élites européennes du <abbr class="abbr" title="17ᵉ siècle"><span class="romain">XVII</span><sup style="font-size:72%">e</sup></abbr> et <abbr class="abbr" title="18ᵉ siècle"><span class="romain">XVIII</span><sup style="font-size:72%">e</sup></abbr> siècles, favorisant la sinophilie, voire la sinomanie des intellectuels. Ils font de Confucius un saint, ce qui est un des éléments déclencheurs de la <a href="/wiki/Querelle_des_rites" title="Querelle des rites">Querelle des rites</a><sup class="reference" id="cite_ref-9"><a href="#cite_note-9"><span class="cite_crochet">[</span>9<span class="cite_crochet">]</span></a></sup>.
</p><p>Jusqu’à la fin de l’Empire, en <a href="/wiki/1911" title="1911">1911</a>, le système des examens, basé sur le corpus confucéen, est resté en vigueur. Certains analystes, chinois ou occidentaux, pensent que l’influence du confucianisme est toujours prépondérante à l’époque actuelle. La <a href="/wiki/Cor%C3%A9e_du_Sud" title="Corée du Sud">Corée du Sud</a> (cf. article <a href="/wiki/Yi_I" title="Yi I">Yi I</a>) et <a href="/wiki/Singapour" title="Singapour">Singapour</a> se réclament toujours de cette doctrine politique (2007). Le Japon se revendique également de cette doctrine pour les bases de sa société, depuis la transformation de la société par <a href="/wiki/Hayashi_Razan" title="Hayashi Razan">Hayashi Razan</a>, sous l'ère Edo, et aujourd'hui encore, on considère que les racines de la société nippone sont shinto-confucianistes. Une seconde mondialisation après celle des jésuites est véhiculée après la Seconde Guerre mondiale par le <a href="/wiki/Sinologie" title="Sinologie">sinologue</a> <a href="/wiki/James_Legge" title="James Legge">James Legge</a> ou le philosophe <a href="/wiki/Pragmatisme" title="Pragmatisme">pragmatiste</a> <a class="new" href="/w/index.php?title=Herbert_Fingarette&amp;action=edit&amp;redlink=1" title="Herbert Fingarette (page inexistante)">Herbert Fingarette</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Herbert_Fingarette" title="en:Herbert Fingarette"><span class="indicateur-langue" title="Article en anglais : « Herbert Fingarette »">(en)</span></a> (auteur de <i>Confucius, The Secular as Sacred</i>, « Confucius, le séculier en tant que sacré », paru en 1972)<sup class="reference" id="cite_ref-10"><a href="#cite_note-10"><span class="cite_crochet">[</span>10<span class="cite_crochet">]</span></a></sup>.
</p><p>Cette continuité apparente du confucianisme en Chine ne doit cependant pas cacher les constants renouvellements, suivis de retours aux sources ou d’éclipses temporaires, qui ont animé l’histoire de la pensée chinoise. Ainsi le renouveau du confucianisme, instauré par <a href="/wiki/Zhu_Xi" title="Zhu Xi">Zhu Xi</a> pendant la <a href="/wiki/Dynastie_Song" title="Dynastie Song">dynastie Song</a>, après une relative mise en retrait durant la dynastie des Tang, a intégré les apports anciens de la pensée <a href="/wiki/Tao%C3%AFsme" title="Taoïsme">taoïste</a> et les apports plus récents du <a href="/wiki/Bouddhisme" title="Bouddhisme">bouddhisme</a> en une orthodoxie restée relativement incontestée depuis lors. C'est depuis la fondation de la <a href="/wiki/R%C3%A9publique_de_Chine_(1912-1949)" title="République de Chine (1912-1949)">République de Chine</a> que l’enseignement des <a href="/wiki/Classique_chinois#Origine_des_Classiques" title="Classique chinois">Quatre Livres</a> et des <a href="/wiki/Classique_chinois#Cinq,_six,_sept,_neuf_et_treize_Classiques" title="Classique chinois">Cinq Classiques</a> confucéens n'est plus obligatoire :
</p><p>Les <i>Quatre Livres</i> (<span class="lang-zh-hans" lang="zh-hans">四书</span> / <span class="lang-zh-hant" lang="zh-hant">四書</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">sì shū</span></span>) sont :
</p>
<ul><li>la <i><a href="/wiki/Grande_%C3%89tude" title="Grande Étude">Grande Étude</a></i> (<span class="lang-zh-hans" lang="zh-hans">大学</span> / <span class="lang-zh-hant" lang="zh-hant">大學</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">dà xué</span></span>),</li>
<li>l’<i><a href="/wiki/Zhong_Yong" title="Zhong Yong">Invariable Milieu</a></i> (<span class="lang-zh" lang="zh">中庸</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">zhōng yóng</span></span>),</li>
<li>les <i><a href="/wiki/Entretiens_de_Confucius" title="Entretiens de Confucius">Entretiens de Confucius</a></i> (<span class="lang-zh-hans" lang="zh-hans">论语</span> / <span class="lang-zh-hant" lang="zh-hant">論語</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">lùn yǔ</span></span>),</li>
<li>le <i><a href="/wiki/Mencius" title="Mencius">Mencius</a> (Livre)</i> (<span class="lang-zh" lang="zh">孟子</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">mèng zǐ</span></span>).</li></ul>
<div class="thumb tright"><div class="thumbinner" style="width:152px;"><a class="image" href="/wiki/Fichier:Liji.jpg"><img alt="" class="thumbimage" data-file-height="1648" data-file-width="2391" decoding="async" height="103" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/68/Liji.jpg/150px-Liji.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/68/Liji.jpg/225px-Liji.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/68/Liji.jpg/300px-Liji.jpg 2x" width="150"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Liji.jpg" title="Agrandir"></a></div>Le <i>Livre des Rites</i>.</div></div></div>
<p>Les <i>Cinq Classiques</i> (<span class="lang-zh" lang="zh">五經 Wǔ jīng</span>) sont :
</p>
<ul><li>le <i><a href="/wiki/Classique_des_vers" title="Classique des vers">Canon des Poèmes</a></i> (<span class="lang-zh-hans" lang="zh-hans">诗经</span> / <span class="lang-zh-hant" lang="zh-hant">詩經</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">shī jīng</span></span>),</li>
<li>le <i><a href="/wiki/Classique_des_documents" title="Classique des documents">Canon de l'Histoire</a></i> (<span class="lang-zh-hans" lang="zh-hans">书经</span> / <span class="lang-zh-hant" lang="zh-hant">書經</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">shū jīng</span></span>),</li>
<li>le <i><a href="/wiki/Yi_Jing" title="Yi Jing">Livre des mutations</a></i> ou <i>Yi Jing</i> (<span class="lang-zh-hans" lang="zh-hans">易经</span> / <span class="lang-zh-hant" lang="zh-hant">易經</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">yì jīng</span></span>),</li>
<li>le <i><a href="/wiki/Classique_des_rites" title="Classique des rites">Livre des Rites</a></i> (<span class="lang-zh-hans" lang="zh-hans">礼记</span> / <span class="lang-zh-hant" lang="zh-hant">禮記</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">lǐ jì</span></span>),</li>
<li>les <i><a href="/wiki/Annales_des_Printemps_et_Automnes" title="Annales des Printemps et Automnes">Annales des Printemps et Automnes</a></i> (<span class="lang-zh" lang="zh">春秋</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">chūn qiū</span></span>, alias <span class="lang-zh" lang="zh">麟經</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">lín jīng</span></span>).</li>
<li>Un sixième classique a été perdu : le <i>Canon de la musique</i> (<span class="lang-zh-hans" lang="zh-hans">乐经</span> / <span class="lang-zh-hant" lang="zh-hant">樂經</span>, <span style="font-style:italic;"><span class="lang-zh-latn-pinyin" lang="zh-latn-pinyin">yuèjīng</span></span>).</li></ul>
<p>Deux mouvements inverses s'observent actuellement<sup class="reference" id="cite_ref-11"><a href="#cite_note-11"><span class="cite_crochet">[</span>11<span class="cite_crochet">]</span></a></sup> : développement en Chine continentale d’écoles confucéennes privées qui inculquent aux élèves l'apprentissage par cœur des classiques de Confucius dont la figure est reconstruite et réinventée ; à l'opposé, surtout chez les sinologues occidentaux, un mouvement de déconstruction de la figure de Confucius et du texte des <i><a href="/wiki/Entretiens_de_Confucius" title="Entretiens de Confucius">Entretiens</a></i><sup class="reference" id="cite_ref-12"><a href="#cite_note-12"><span class="cite_crochet">[</span>12<span class="cite_crochet">]</span></a></sup>.
</p>
<h2><span id="Ses_.C3.A9l.C3.A8ves"></span><span class="mw-headline" id="Ses_élèves">Ses élèves</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=27" title="Modifier la section : Ses élèves">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=27" title="Modifier la section : Ses élèves">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Confucius a eu plus de 3 000 élèves ; parmi <span class="nowrap">lesquels 72</span> (<span class="nowrap">77 selon</span> certaines sources) maîtrisent les 6 arts enseignés<sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Aide:R%C3%A9f%C3%A9rence_n%C3%A9cessaire" title="Aide:Référence nécessaire">[<abbr class="abbr" title="référence">réf.</abbr> souhaitée]</a></sup>.
</p><p>Et parmi les disciples, <span class="nowrap">10 sont</span> considérés comme des sages ou philosophes dans la tradition confucéenne. Ils sont appelés les <a href="/wiki/Dix_sages_de_l%27%C3%A9cole_confuc%C3%A9enne" title="Dix sages de l'école confucéenne">dix sages de l'école confucéenne</a>.
</p><p>D'après le <span class="nowrap">verset 3</span> (<span class="nowrap">ou 2</span> selon version) du <span class="nowrap">chapitre 11</span> du <a href="/wiki/Entretiens_de_Confucius" title="Entretiens de Confucius">Lun Yu</a> :
</p>
<ul><li><a href="/wiki/Yan_Hui_(philosophe)" title="Yan Hui (philosophe)">Yan Hui</a>, <a href="/wiki/Min_Sun" title="Min Sun">Min Ziqian</a>, <a href="/wiki/Ran_Geng" title="Ran Geng">Ran Boniu</a> et <a href="/wiki/Ran_Yong" title="Ran Yong">Ran Yong</a> étaient remarquables par leurs vertus ;</li></ul>
<ul><li><a href="/wiki/Zai_Yu" title="Zai Yu">Zai Yu</a> et <a href="/wiki/Duanmu_Ci" title="Duanmu Ci">Zigong</a>, par leur habileté à parler ;</li></ul>
<ul><li><a href="/wiki/Ran_Qiu" title="Ran Qiu">Ran Qiu</a> et <a href="/wiki/Zilu" title="Zilu">Zilu</a>, par leur habileté à gouverner ;</li></ul>
<ul><li><a href="/wiki/Yan_Yan_(philosophe)" title="Yan Yan (philosophe)">Ziyou</a> et <a href="/wiki/Bu_Shang" title="Bu Shang">Zixia</a>, par leur habileté dans l'étude.</li></ul>
<h2><span class="mw-headline" id="Bibliographie">Bibliographie</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=28" title="Modifier la section : Bibliographie">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=28" title="Modifier la section : Bibliographie">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Publications_anciennes">Publications anciennes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Confucius&amp;veaction=edit&amp;section=29" title="Modifier la section : Publications anciennes">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Confucius&amp;action=edit&amp;section=29" title="Modifier la section : Publications anciennes">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:1783_morale_de_Confucius_1005146.jpg"><img alt="" class="thumbimage" data-file-height="1419" data-file-width="1650" decoding="async" height="189" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/ca/1783_morale_de_Confucius_1005146.jpg/220px-1783_morale_de_Confucius_1005146.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/ca/1783_morale_de_Confucius_1005146.jpg/330px-1783_morale_de_Confucius_1005146.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/ca/1783_morale_de_Confucius_1005146.jpg/440px-1783_morale_de_Confucius_1005146.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:1783_morale_de_Confucius_1005146.jpg" title="Agrandir"></a></div>En 1783, imprimé par <a href="/wiki/Jacques-Fran%C3%A7ois_Valade" title="Jacques-François Valade">Jacques-François Valade</a> et <a href="/wiki/Hubert_Cazin" title="Hubert Cazin">Hubert Cazin</a>, gravure de <a href="/wiki/R%C3%A9mi_Delvaux" title="Rémi Delvaux">Rémi Delvaux</a> exemplaire de la <a href="/wiki/Biblioth%C3%A8que_Carnegie_(Reims)" title="Bibliothèque Carnegie (Reims)">bibliothèque Carnegie (Reims)</a>.</div></div></div>
<p>Tous ses livres moraux ont été traduits en latin et paraphrasés par <a href="/wiki/Prospero_Intorcetta" title="Prospero Intorcetta">Prospero Intorcetta</a>, <a class="new" href="/w/index.php?title=Christian_Herdtrich&amp;action=edit&amp;redlink=1" title="Christian Herdtrich (page inexistante)">Christian Herdtrich</a>, <a href="/wiki/Fran%C3%A7ois_de_Rougemont" title="François de Rougemont">François de Rougemont</a> et <a href="/wiki/Philippe_Couplet" title="Philippe Couplet">Philippe Couplet</a>, sous le titre de <i>Confucius, Sinarum philosophus</i>, <a href="/wiki/Paris" title="Paris">Paris</a>, <a href="/wiki/1687" title="1687">1687</a>, in-folio.
</p><p>Le <i>Shū Jīng</i> a été traduit en français, par le Père <a href="/wiki/Antoine_Gaubil" title="Antoine Gaubil">Antoine Gaubil</a>, <a href="/wiki/1770" title="1770">1770</a> ; le <i>Zhōng Yóng</i> a été publié en chinois, avec traduction latine et française, par <a href="/wiki/Jean-Pierre_Abel-R%C3%A9musat" title="Jean-Pierre Abel-Rémusat">Abel-Rémusat</a>, <a href="/wiki/1817" title="1817">1817</a>, in-4 ; le <i>Ta hio</i>, par <a href="/wiki/Guillaume_Pauthier" title="Guillaume Pauthier">Guillaume Pauthier</a> (chinois, latin et français), <a href="/wiki/1837" title="1837">1837</a>, in-8.
</p><p>On trouve aussi plusieurs des ouvrages de Confucius dans les <i>Sinensis imperii libri classici sex</i> du Père <a href="/wiki/Fran%C3%A7ois_No%C3%ABl" title="François Noël">François Noël</a>, <a href="/wiki/Prague" title="Prague">Prague</a>, <a href="/wiki/1711" title="1711">1711</a>, collection traduite en français par l'abbé <a href="/wiki/Fran%C3%A7ois-Andr%C3%A9-Adrien_Pluquet" title="François-André-Adrien Pluquet">François-André-Adrien Pluquet</a>, <a href="/wiki/1784" title="1784">1784</a>, 7 volumes in-18.
</p><p>La <i>Vie</i> de Confucius a été écrite par le Père <a href="/wiki/Joseph-Marie_Amiot" title="Joseph-Marie Amiot">Joseph-Marie Amiot</a> dans les <i>Mémoires sur les Chinois</i>. On a publié la <i>Morale de Confucius</i>, <a href="/wiki/Amsterdam" title="Amsterdam">Amsterdam</a>, <a href="/wiki/1688" title="1688">1688</a>, 1 volume in-8.
</p>
