<p><b>Fumer</b> est une pratique consistant à brûler ou chauffer une <a href="/wiki/Substance" title="Substance">substance</a> pour en inhaler la <a href="/wiki/Fum%C3%A9e" title="Fumée">fumée</a> par la <a href="/wiki/Bouche_humaine" title="Bouche humaine">bouche</a> ou le <a href="/wiki/Nez" title="Nez">nez</a>. 
Plusieurs types de produits peuvent être absorbés de cette façon : 
</p>
<ul><li>le <a href="/wiki/Tabac" title="Tabac">tabac</a> (via <a href="/wiki/Cigarette" title="Cigarette">Cigarette</a>, <a href="/wiki/Pipe" title="Pipe">pipe</a>, <a href="/wiki/Cigare" title="Cigare">cigare</a>, <a href="/wiki/Narguil%C3%A9" title="Narguilé">chicha</a>, <a class="mw-redirect" href="/wiki/Bid%C3%AE" title="Bidî">bidî</a>, <a href="/wiki/Tabac_chauff%C3%A9" title="Tabac chauffé">tabac chauffé</a>...) ;</li>
<li>des <a href="/wiki/Nicotine" title="Nicotine">composés nicotiniques</a> (via la <a href="/wiki/Cigarette_%C3%A9lectronique" title="Cigarette électronique">cigarette électronique</a>) ;</li>
<li>le <a href="/wiki/Cannabis" title="Cannabis">cannabis</a> ;</li>
<li>l'<a href="/wiki/Opium" title="Opium">opium</a> ;</li>
<li>la <a href="/wiki/Sauge_des_devins" title="Sauge des devins">Sauge des devins</a> ;</li>
<li>l'<a href="/wiki/H%C3%A9ro%C3%AFne" title="Héroïne">héroïne</a> ;</li>
<li>la <a href="/wiki/Coca%C3%AFne" title="Cocaïne">cocaïne</a>, le <a href="/wiki/Crack_(stup%C3%A9fiant)" title="Crack (stupéfiant)">crack</a> (un dérivé fumable de la <a href="/wiki/Coca%C3%AFne" title="Cocaïne">cocaïne</a>) ;</li>
<li>la <a href="/wiki/M%C3%A9thamph%C3%A9tamine" title="Méthamphétamine">méthamphétamine</a>, sous forme d'<i>ice</i>.</li></ul>
<p>Ce type de consommation se fait généralement à l'aide de <a href="/wiki/Pipe" title="Pipe">pipes</a> adaptées, et sous forme de <a href="/wiki/Cigarette" title="Cigarette">cigarettes</a> ou de <a href="/wiki/Cigare" title="Cigare">cigares</a>  (notamment dans le cas du tabac et du cannabis). L'<a href="/wiki/H%C3%A9ro%C3%AFne" title="Héroïne">héroïne</a> peut par exemple se fumer à l'aide de cigarettes roulées dans du papier <a href="/wiki/Aluminium" title="Aluminium">aluminium</a> sous laquelle une flamme est passée pour permettre l'inhalation des vapeurs.
</p>
<div class="thumb tleft"><div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/Fichier:Sigmund_Freud,_by_Max_Halberstadt_(cropped).jpg"><img alt="" class="thumbimage" data-file-height="2240" data-file-width="1647" decoding="async" height="231" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/Sigmund_Freud%2C_by_Max_Halberstadt_%28cropped%29.jpg/170px-Sigmund_Freud%2C_by_Max_Halberstadt_%28cropped%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/Sigmund_Freud%2C_by_Max_Halberstadt_%28cropped%29.jpg/255px-Sigmund_Freud%2C_by_Max_Halberstadt_%28cropped%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/36/Sigmund_Freud%2C_by_Max_Halberstadt_%28cropped%29.jpg/340px-Sigmund_Freud%2C_by_Max_Halberstadt_%28cropped%29.jpg 2x" width="170"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Sigmund_Freud,_by_Max_Halberstadt_(cropped).jpg" title="Agrandir"></a></div><a href="/wiki/Sigmund_Freud" title="Sigmund Freud">Sigmund Freud</a>, dont le médecin assista son suicide en raison d'un <a href="/wiki/Cancer_de_la_bouche" title="Cancer de la bouche">cancer de la bouche</a> dû à son <a href="/wiki/Tabagisme" title="Tabagisme">tabagisme</a><sup class="reference" id="cite_ref-Gay_1-0"><a href="#cite_note-Gay-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup></div></div></div>
<p>Ces produits sont consommés pour leurs propriétés <a href="/wiki/Stup%C3%A9fiant" title="Stupéfiant">stupéfiantes</a>. Ce mode de consommation convertit les principes actifs des produits sous une forme permettant leur passage des <a href="/wiki/Poumon" title="Poumon">poumons</a> vers le <a href="/wiki/Appareil_cardiovasculaire" title="Appareil cardiovasculaire">système sanguin</a> puis le <a href="/wiki/Cerveau" title="Cerveau">cerveau</a> où ils agissent.
</p><p>Hors ces produits stupéfiants, certaines plantes sont parfois consommées sous forme de cigarette, comme les feuilles d'<a href="/wiki/Eucalyptus" title="Eucalyptus">eucalyptus</a>. Quelques-unes de ces cigarettes <a href="/wiki/Inde" title="Inde">indiennes</a> ne contiennent pas de tabac, mais des herbes aromatiques roulées dans une feuille d'eucalyptus. En <a href="/wiki/Indon%C3%A9sie" title="Indonésie">Indonésie</a>, les <a href="/wiki/Kretek" title="Kretek">kreteks</a> contiennent du <a href="/wiki/Giroflier" title="Giroflier">clou de girofle</a> en plus du tabac, pour la saveur. Certaines cultures traditionnelles africaines ou américaines font brûler des plantes médicinales sur un foyer pour en inhaler les fumées, mais ces pratiques liées à différentes formes de <a href="/wiki/Chamanisme" title="Chamanisme">chamanisme</a> tiennent davantage de la <a href="/wiki/Fumigation" title="Fumigation">fumigation</a>. 
</p>
<div class="thumb tright"><div class="thumbinner" style="width:266px;"><a class="image" href="/wiki/Fichier:Smoker_milanosis.jpg"><img alt="" class="thumbimage" data-file-height="654" data-file-width="1160" decoding="async" height="149" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/67/Smoker_milanosis.jpg/264px-Smoker_milanosis.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/67/Smoker_milanosis.jpg/396px-Smoker_milanosis.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/67/Smoker_milanosis.jpg/528px-Smoker_milanosis.jpg 2x" width="264"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Smoker_milanosis.jpg" title="Agrandir"></a></div>La pigmentation gingivale chez un fumeur.</div></div></div>
<p>En dehors des propriétés toxiques des produits inhalés, ce mode de consommation entraine dans de nombreux cas des risques élevés de <a href="/wiki/Cancer_du_poumon" title="Cancer du poumon">cancer du poumon</a>, de la <a href="/wiki/Cancer_des_voies_a%C3%A9rodigestives_sup%C3%A9rieures" title="Cancer des voies aérodigestives supérieures">gorge</a> et de la <a href="/wiki/Cancer_de_la_bouche" title="Cancer de la bouche">langue</a>, en raison de l'absorption de produits cancérigènes accompagnant la combustion du produit consommé.
</p><p>On parle parfois de <a href="/wiki/Tabagisme_passif" title="Tabagisme passif">fumeur passif</a> pour désigner une personne qui respire les fumées d'autres fumeurs sans fumer elle-même.
</p>
