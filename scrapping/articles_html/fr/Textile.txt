<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:William-adolphe_bouguereau_the_spinner.jpg"><img alt="" class="thumbimage" data-file-height="1939" data-file-width="1060" decoding="async" height="402" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/William-adolphe_bouguereau_the_spinner.jpg/220px-William-adolphe_bouguereau_the_spinner.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/64/William-adolphe_bouguereau_the_spinner.jpg/330px-William-adolphe_bouguereau_the_spinner.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/64/William-adolphe_bouguereau_the_spinner.jpg/440px-William-adolphe_bouguereau_the_spinner.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:William-adolphe_bouguereau_the_spinner.jpg" title="Agrandir"></a></div><i>La Fileuse</i>, <a href="/wiki/William_Bouguereau" title="William Bouguereau">William Bouguereau</a>, <a href="/wiki/XIXe_si%C3%A8cle" title="XIXe siècle"><abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle</a>.</div></div></div>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Simple-textile-magnified.jpg"><img alt="" class="thumbimage" data-file-height="1333" data-file-width="2000" decoding="async" height="147" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2d/Simple-textile-magnified.jpg/220px-Simple-textile-magnified.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2d/Simple-textile-magnified.jpg/330px-Simple-textile-magnified.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2d/Simple-textile-magnified.jpg/440px-Simple-textile-magnified.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Simple-textile-magnified.jpg" title="Agrandir"></a></div>Détail d'un objet tissé.</div></div></div>
<p>Un <b>textile</b> est un <a href="/wiki/Mat%C3%A9riau" title="Matériau">matériau</a> susceptible d'être <a href="/wiki/Tissage" title="Tissage">tissé</a> ou <a href="/wiki/Tricot" title="Tricot">tricoté</a>. Initialement, il désigne donc un matériau qui peut se diviser en <a href="/wiki/Fibre" title="Fibre">fibres</a> ou en <a href="/wiki/Fil_textile" title="Fil textile">fils textiles</a>, tels le <a href="/wiki/Coton" title="Coton">coton</a>, le <a href="/wiki/Chanvre" title="Chanvre">chanvre</a>, le <a href="/wiki/Lin_(textile)" title="Lin (textile)">lin</a>, la <a href="/wiki/Laine" title="Laine">laine</a> (textiles <a href="/wiki/Compos%C3%A9_organique" title="Composé organique">organiques</a>) ou la pierre d'<a href="/wiki/Amiante" title="Amiante">amiante</a> (textile <a href="/wiki/Min%C3%A9ral" title="Minéral">minéral</a>), puis avec la découverte de nouvelles techniques, les <a href="/wiki/Fibre_synth%C3%A9tique" title="Fibre synthétique">fibres synthétiques</a>.
</p><p>L'action de séparer les fibres d'un textile s'appelle le <a href="/wiki/Filage_textile" title="Filage textile">filage</a>. Par extension, le mot textile peut également s'appliquer au résultat après transformation, un drap est un textile.
</p><p>S'il est tissé, le textile forme un tissu. Dans le cas contraire, il forme une étoffe servant à rembourrer et orner. À la fin du <a href="/wiki/XVIe_si%C3%A8cle" title="XVIe siècle"><abbr class="abbr" title="16ᵉ siècle"><span class="romain">XVI</span><sup style="font-size:72%">e</sup></abbr> siècle</a>, l'étoffe prend le sens plus spécifique de textile servant à l'habillement ou à l'ameublement. Aujourd'hui, on trouve des tissus formés par pressage ou agglomération de textile, une extension technique moderne aboutissant à l'expression contradictoire <a href="/wiki/Textile_non_tiss%C3%A9" title="Textile non tissé">tissu non tissé</a>.
</p><p>On distingue deux grandes classes de textiles auxquelles s'ajoutent plusieurs sous classes possibles :
</p>
<ul><li>Textiles traditionnels : textiles pour lesquels on porte l'attention sur l'apparence et le confort. Il s'agit surtout du domaine de la <a href="/wiki/Mode_(habillement)" title="Mode (habillement)">mode</a>, souvent du <a href="/wiki/V%C3%AAtement" title="Vêtement">vêtement</a>, mais aussi de l'<a class="mw-redirect" href="/wiki/Ameublement" title="Ameublement">ameublement</a> (<a href="/wiki/Drap_(literie)" title="Drap (literie)">draps</a>, tentures, <a href="/wiki/Rideau" title="Rideau">rideaux</a>, <a href="/wiki/Nappe_de_table" title="Nappe de table">nappes</a>, <a class="mw-disambig" href="/wiki/Serviette" title="Serviette">serviettes</a>, <a href="/wiki/Tapisserie" title="Tapisserie">tapisseries</a>).</li>
<li>Textiles techniques : sont classés dans cette catégorie tous textiles pour lesquels importent les caractéristiques <a href="/wiki/M%C3%A9canique_(science)" title="Mécanique (science)">mécaniques</a>, chimiques, physico-chimiques et ayant une application technique : <a href="/wiki/G%C3%A9otextile" title="Géotextile">géotextile</a>, textile médical, matériaux composites à renfort textile.</li></ul>
<p>Par exemple les <a href="/wiki/Filtre_(physique)" title="Filtre (physique)">filtres</a>, le <a href="/wiki/Feutre_(textile)" title="Feutre (textile)">feutre</a>, les <a href="/wiki/M%C3%A8che_(pansement)" title="Mèche (pansement)">mèches</a>, le <a href="/wiki/Fil_textile" title="Fil textile">fil</a>, les <a href="/wiki/Tricot_(textile)" title="Tricot (textile)">tricots</a>, le <a href="/wiki/Papier" title="Papier">papier</a>... sont des textiles.
</p><p>On parle de textiles intelligents ou actifs dès lors que le textile a la capacité de sentir une information dans son environnement et d'y répondre avec un comportement spécifique.
</p>

</p>
</div></div>
<ul><li>650 000 ans (date établie par l'analyse <a href="/wiki/Pal%C3%A9og%C3%A9n%C3%A9tique" title="Paléogénétique">paléogénétique</a> du <a href="/wiki/Pediculus_humanus" title="Pediculus humanus">pou du vêtement</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>) Les premiers <a href="/wiki/V%C3%AAtement" title="Vêtement">vêtements</a> portés étaient probablement en peaux et fourrures d'animaux rêches et grossières, protégeant le <a href="/wiki/Chasseur-cueilleur" title="Chasseur-cueilleur">chasseur-cueilleur</a> préhistorique des glaciations du <a href="/wiki/Pl%C3%A9istoc%C3%A8ne" title="Pléistocène">Pléistocène</a> : en utilisant des <a href="/wiki/Grattoir" title="Grattoir">grattoirs</a> pour racler la viande d'animaux, ils se sont servis de leur peau comme <a href="/wiki/Costume" title="Costume">costume</a> drapé ou enfilé, ont utilisé de fines lanières de cuir pour attacher les fourrures<sup class="reference" id="cite_ref-2"><a href="#cite_note-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.</li>
<li>40 000 ans, l’<a href="/wiki/Homme_de_Cro-Magnon" title="Homme de Cro-Magnon">homme de Cro-Magnon</a> a développé des outils pointus plus fins comme des <a href="/wiki/Poin%C3%A7on" title="Poinçon">poinçons</a> ou des <a href="/wiki/Aiguille_%C3%A0_coudre" title="Aiguille à coudre">aiguilles à coudre</a> en os d'animaux, pouvant percer de petits trous dans les peaux, et ainsi lacer ou coudre des <a href="/wiki/Tunique" title="Tunique">tuniques</a>.</li>
<li>C'est le <a href="/wiki/Mouton" title="Mouton">mouton</a> qui fut d'abord domestiqué en <a href="/wiki/M%C3%A9sopotamie" title="Mésopotamie">Mésopotamie</a> en raison de la qualité de sa <a href="/wiki/Laine" title="Laine">laine</a>, <a href="/wiki/Hammurabi" title="Hammurabi">Hammurabi</a> appelant la <a href="/wiki/Babylone" title="Babylone">Babylone</a> le « pays de la laine ». Facile à travailler, elle était filée et tissée avec des techniques encore utilisées en <a href="/wiki/Vannerie" title="Vannerie">vannerie</a>, la laine tissée étant plus chaude que les <a href="/wiki/Fourrure" title="Fourrure">fourrures</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>. Le premier outil de filage consistait en un petit bout de bois doté d'un crochet qui permettait d'attraper le fil. Il était possible de rouler la branche sur la cuisse afin de rendre la torsion plus rapide. Le fil était quant à lui enroulé autour de la branche afin de pouvoir être stocké et maintenu en place. Il est possible de filer avec la branche. Toutefois, si ce procédé est particulièrement adapté à l'apprentissage, il en demeure relativement lent. Une alternative fut donc nécessaire. La maîtrise de la fabrication d'objets et de vêtements créés avec des fibres textiles durant les <a href="/wiki/Pr%C3%A9histoire" title="Préhistoire">temps préhistoriques</a> est une étape essentielle pour les chances de survie des populations préhistoriques<sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>.</li>
<li>34 000 ans, la découverte de fibres <a href="/wiki/Teinture" title="Teinture">teintes</a> de <a href="/wiki/Lin_(textile)" title="Lin (textile)">lin</a> naturel et de <a href="/wiki/Laine" title="Laine">laine</a> de chèvre portant des marques de torsion dans des couches d'argile de la <i>grotte de Dzudzuana</i> en <a href="/wiki/G%C3%A9orgie_(pays)" title="Géorgie (pays)">Géorgie</a> suggère l'utilisation de matériaux textiles. Bien qu'elles aient pu être utilisées comme <a href="/wiki/Cordage" title="Cordage">cordage</a> pour l'emmanchement des outils en pierre ou pour le <a class="mw-redirect" href="/wiki/Tressage" title="Tressage">tressage</a> de nattes et paniers, ces fibres ont probablement servi au <a href="/wiki/Tissage" title="Tissage">tissage</a> de vêtements à coudre, l'équipe de chercheurs ayant trouvé associés à ces fibres des mites, des larves de coléoptères et des spores de <a class="new" href="/w/index.php?title=Chaetomium&amp;action=edit&amp;redlink=1" title="Chaetomium (page inexistante)">Chaetomium</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Chaetomium" title="en:Chaetomium"><span class="indicateur-langue" title="Article en anglais : « Chaetomium »">(en)</span></a> typiques de la dégradation des textiles<sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>.</li>
<li>Le tissage rend l'étoffe plus résistante. Cette technique <a href="/wiki/N%C3%A9olithique" title="Néolithique">néolithique</a> nécessite le <a href="/wiki/Filage_textile" title="Filage textile">filage</a> de la laine de mouton ou de chèvre, de la fibre de coton, laine, lin, ou soie, ces fibres pouvant subir une torsion à la main pour former un fil solide. L'art du <a href="/wiki/Filage_textile" title="Filage textile">filage</a> est attesté dès la <a href="/wiki/S%C3%A9dentarisation" title="Sédentarisation">sédentarisation</a> des hommes qui découvrent qu'il est possible de fabriquer un fil solide en parallélisant les poils ou les fibres végétales (laine, lin) puis en donnant manuellement une torsion aux faisceaux de fibres<sup class="reference" id="cite_ref-invention_6-0"><a href="#cite_note-invention-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.</li>
<li>vers <a class="mw-redirect" href="/wiki/-8000" title="-8000">-8000</a> L'homme préhistorique apprend progressivement à macérer les fibres végétales pour les rendre flexibles (technique du <a href="/wiki/Rouissage" title="Rouissage">rouissage</a>) ainsi qu'à détacher les poils des <a href="/wiki/Cuir" title="Cuir">cuirs</a> grâce à des <a class="mw-redirect" href="/wiki/Silex_taill%C3%A9s" title="Silex taillés">silex taillés</a>, fabriquant d'abord des <a href="/wiki/Feutre_(textile)" title="Feutre (textile)">feutres</a>. Le premier feutre est évoqué sur des motifs de peinture murales du site néolithique de <a href="/wiki/%C3%87atal_H%C3%B6y%C3%BCk" title="Çatal Höyük">Çatal Höyük</a>, de lin, laine, poils, fourrure, voire en écorce d'arbre, mais le feutre reste une étoffe moins résistante que le tissu<sup class="reference" id="cite_ref-7"><a href="#cite_note-7"><span class="cite_crochet">[</span>7<span class="cite_crochet">]</span></a></sup>.</li>
<li>dès <a class="mw-redirect" href="/wiki/-6000" title="-6000">-6000</a> en <a href="/wiki/Jud%C3%A9e" title="Judée">Judée</a> une forme de <a href="/wiki/Tricot" title="Tricot">tricot</a>, le <a href="/wiki/Nalebinding" title="Nalebinding">nalbinding</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>8<span class="cite_crochet">]</span></a></sup>.</li>
<li><a href="/wiki/VIe_mill%C3%A9naire_av._J.-C." title="VIe millénaire av. J.-C.">VIe millénaire av. J.-C.</a> le filage au <a href="/wiki/Fuseau_(bobine_de_fil)" title="Fuseau (bobine de fil)">fuseau</a> et à la <a href="/wiki/Quenouille" title="Quenouille">quenouille</a>, constitués de différents matériaux, pour le <a href="/wiki/Lin_cultiv%C3%A9" title="Lin cultivé">lin</a> et la laine est attesté (découverte dans le village néolithique de <a class="mw-redirect" href="/wiki/Sesklo" title="Sesklo">Sesklo</a> de <a href="/wiki/Fusa%C3%AFole" title="Fusaïole">fusaïole</a>)<sup class="reference" id="cite_ref-9"><a href="#cite_note-9"><span class="cite_crochet">[</span>9<span class="cite_crochet">]</span></a></sup> jusqu'à l'apparition du <a href="/wiki/Rouet_(outil)" title="Rouet (outil)">rouet</a> au début du <a href="/wiki/XIVe_si%C3%A8cle" title="XIVe siècle"><abbr class="abbr" title="14ᵉ siècle"><span class="romain">XIV</span><sup style="font-size:72%">e</sup></abbr> siècle</a> au <a href="/wiki/Moyen-Orient" title="Moyen-Orient">Moyen-Orient</a>. C'est au <a href="/wiki/XVIIe_si%C3%A8cle" title="XVIIe siècle"><abbr class="abbr" title="17ᵉ siècle"><span class="romain">XVII</span><sup style="font-size:72%">e</sup></abbr> siècle</a> qu'on ajoute une <a href="/wiki/P%C3%A9dale_(m%C3%A9canique)" title="Pédale (mécanique)">pédale</a> au rouet pour libérer la main droite du fileur et améliorer la technique. Mais malgré ce progrès, le <a href="/wiki/Tissage" title="Tissage">tissage</a> et le filage restent des opérations lentes, artisanales et relativement onéreuses<sup class="reference" id="cite_ref-invention_6-1"><a href="#cite_note-invention-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.</li>
<li>En <a href="/wiki/1746" title="1746">1746</a>, la première manufacture d'<a class="mw-redirect" href="/wiki/Indienne_(tissus)" title="Indienne (tissus)">indiennes</a> <a href="/wiki/Mulhouse" title="Mulhouse">mulhousienne</a> est créée dans ce qui est encore la <a href="/wiki/R%C3%A9publique_de_Mulhouse" title="République de Mulhouse">République de Mulhouse</a> (<i>Stadtrepublik Mülhausen</i>) par Koechlin, Schmaltzer et <a href="/wiki/Jean-Henri_Dollfus_p%C3%A8re" title="Jean-Henri Dollfus père">Jean-Henri Dollfus</a>. Dans les <a href="/wiki/Ann%C3%A9es_1760" title="Années 1760">années 1760</a> apparaît, au <a href="/wiki/Royaume-Uni" title="Royaume-Uni">Royaume-Uni</a>, le premier <a class="new" href="/w/index.php?title=M%C3%A9tier_%C3%A0_filer&amp;action=edit&amp;redlink=1" title="Métier à filer (page inexistante)">métier à filer</a> mécanique (<i>Spinning jenny</i>).</li>
<li>En <a href="/wiki/1771" title="1771">1771</a>, <a href="/wiki/Richard_Arkwright" title="Richard Arkwright">Richard Arkwright</a> crée la première <b>filature</b> industrielle.</li>
<li><a href="/wiki/Samuel_Crompton" title="Samuel Crompton">Crompton</a> invente quant à lui la <i>spinning mule</i> permettant à un seul ouvrier de commander jusqu'à 1 000 fuseaux.</li>
<li>En <a href="/wiki/1812" title="1812">1812</a>, tous les métiers à filer du <a href="/wiki/Royaume-Uni" title="Royaume-Uni">Royaume-Uni</a> produisent autant que quatre millions de rouets<sup class="reference" id="cite_ref-invention_6-2"><a href="#cite_note-invention-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.</li></ul>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Jeton_de_la_corporation_des_commerces_de_draperie.jpg"><img alt="" class="thumbimage" data-file-height="610" data-file-width="1200" decoding="async" height="112" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Jeton_de_la_corporation_des_commerces_de_draperie.jpg/220px-Jeton_de_la_corporation_des_commerces_de_draperie.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Jeton_de_la_corporation_des_commerces_de_draperie.jpg/330px-Jeton_de_la_corporation_des_commerces_de_draperie.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Jeton_de_la_corporation_des_commerces_de_draperie.jpg/440px-Jeton_de_la_corporation_des_commerces_de_draperie.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Jeton_de_la_corporation_des_commerces_de_draperie.jpg" title="Agrandir"></a></div>Jeton de la corporation des commerces de draperie</div></div></div>
<p>Le filage industriel se développe avec deux inventions : d'une part, la machine à égrener le <a href="/wiki/Coton" title="Coton">coton</a> pour fournir la fibre ; d'autre part, celle du <a href="/wiki/M%C3%A9tier_%C3%A0_tisser" title="Métier à tisser">métier à tisser</a> pour utiliser le fil. L'expansion des filatures crée un exode rural qui engendre une mécanisation agricole visant à maintenir les niveaux de production et oblige les artisans fileurs à se reconvertir. Le travail en filature ne demandant ni force, ni aptitude spéciale, la main d'œuvre bon marché que sont les femmes et les enfants est préférée, avant que l'évolution de la législation ne finisse par interdire le travail des enfants<sup class="reference" id="cite_ref-invention_6-3"><a href="#cite_note-invention-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.
</p>
<h2><span id="Mati.C3.A8res_premi.C3.A8res"></span><span class="mw-headline" id="Matières_premières">Matières premières</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=2" title="Modifier la section : Matières premières">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=2" title="Modifier la section : Matières premières">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les matières textiles sont généralement classées en trois grandes catégories en fonction de leur origine. On distingue ainsi les matières naturelles (végétales ou animales), artificielles et les <a href="/wiki/Fibre_synth%C3%A9tique" title="Fibre synthétique">matières synthétiques</a>.
</p>
<ul><li>Les matières naturelles : ces matières premières proviennent d’origine végétale ou animale. Chaque matière inclut divers avantages et inconvénients, et est généralement déclinée en différentes variétés.
<ul><li>La <a href="/wiki/Laine" title="Laine">laine</a> : d’origine animale, elle est obtenue à partir de la toison du <a href="/wiki/Mouton" title="Mouton">mouton</a> ou d’autres animaux. Cette matière se démarque par sa douceur et sa légèreté, mais aussi par son confort et sa capacité à tenir chaud. Cependant, la laine est une matière délicate, résiste mal aux frottements et est sensible à la lumière. On distingue différentes variétés de laine parmi lesquelles le <a href="/wiki/M%C3%A9rinos" title="Mérinos">mérinos</a>, le <a href="/wiki/Cachemire_(tissu)" title="Cachemire (tissu)">cachemire</a>, <a href="/wiki/Angora" title="Angora">l’angora</a> et <a href="/wiki/Alpaga" title="Alpaga">l’alpaga</a>.</li>
<li>La <a href="/wiki/Soie" title="Soie">soie</a> : également d’origine animale, la soie est issue du cocon produit par la <a href="/wiki/Bombyx" title="Bombyx">chenille du bombyx</a> du mûrier communément appelée <a class="mw-redirect" href="/wiki/Ver_%C3%A0_soie" title="Ver à soie">ver à soie</a>. Cette matière aux filaments longs et brillants est résistante tout en gardant une certaine douceur au toucher. Le confort qu’elle procure et sa bonne isolation en fait un produit <a href="/wiki/Haut_de_gamme" title="Haut de gamme">haut de gamme</a> au prix souvent élevé.</li>
<li>Le <a href="/wiki/Coton" title="Coton">coton</a> : d’origine végétale, le coton est une fibre qui recouvre les graines du <a href="/wiki/Cotonnier" title="Cotonnier">cotonnier</a>. Très doux au toucher, il est généralement agréable à porter et bénéficie d’une bonne robustesse et souplesse. Il a tendance à se froisser facilement et n’est pas bon isolant.</li>
<li>Le <a href="/wiki/Lin_(textile)" title="Lin (textile)">lin</a> : le lin est une matière textile extraite des fibres de la plante du même nom. Cette matière très résistante d’origine végétale est dotée d’un fort pouvoir absorbant et d’une certaine fraîcheur au porter, ce qui la rend idéale pour l’été. Le lin a cependant tendance à se froisser facilement et à jaunir lorsqu’il est exposé à la lumière.</li></ul></li>
<li> Les matières artificielles : ces matières sont constituées de fibres fabriquées à partir de la <a href="/wiki/Cellulose" title="Cellulose">cellulose</a> extraite des végétaux. La matière première artificielle la plus utilisée est sans aucun doute la <a href="/wiki/Viscose" title="Viscose">viscose</a> qui est fabriquée à partir de cellulose extraite de bois notamment. La viscose est dotée d’un bon pouvoir absorbant et se démarque par sa brillance similaire à la <a href="/wiki/Soie" title="Soie">soie</a>. Elle est en revanche critiquée pour sa facilité à se froisser et sa faible résistance à l’humidité. À noter que ce n’est pas la seule matière artificielle, on dénombre également le <a href="/wiki/Modal_(textile)" title="Modal (textile)">modal</a>, le <a href="/wiki/Cupro" title="Cupro">cupro</a> ou encore l’<a href="/wiki/Ac%C3%A9tate_de_cellulose" title="Acétate de cellulose">acétate de cellulose</a>.</li>
<li>Les matières synthétiques : les matières présentées ci-dessous sont produites par synthèse de composés chimiques provenant majoritairement d’<a class="mw-redirect" href="/wiki/Hydrocarbures" title="Hydrocarbures">hydrocarbures</a> ou d’<a href="/wiki/Amidon" title="Amidon">amidon</a>. Elles sont généralement mélangées à des <a class="mw-redirect" href="/wiki/Fibres_naturelles" title="Fibres naturelles">fibres naturelles</a> afin de faire baisser le <a class="mw-redirect" href="/wiki/Prix_de_revient" title="Prix de revient">prix de revient</a> tout en bénéficiant des avantages apportés par ces matières de synthèse.
<ul><li>Le <a href="/wiki/Polyester" title="Polyester">polyester</a> : très bon <a href="/wiki/Isolant_thermique" title="Isolant thermique">isolant thermique</a>, le polyester est résistant à l’usure, la luminosité et aux <a class="mw-redirect" href="/wiki/Mites" title="Mites">mites</a>. Ce textile supposé infroissable est cependant doté d’un faible pouvoir absorbant et ne laisse pas respirer la peau.</li>
<li>Le <a href="/wiki/Polyamide" title="Polyamide">polyamide</a> : le polyamide matière textile résiste parfaitement à l’usure et aux <a class="mw-redirect" href="/wiki/Mites" title="Mites">mites</a> tout en étant un très bon isolant thermique et bénéficiant d’une excellente résistance à la traction. Le polyamide demeure malgré ses qualités un textile au pouvoir absorbant faible, au toucher froid et relativement sensible à la lumière.</li>
<li>L'<a href="/wiki/Fibre_acrylique" title="Fibre acrylique">acrylique</a> : l’acrylique est une matière synthétique résultant de la réaction de l’<a href="/wiki/Ammoniac" title="Ammoniac">ammoniac</a> sur un dérivé du pétrole. C’est un textile très résistant aux tractions, agréable au toucher, insensible aux <a class="mw-redirect" href="/wiki/Mites" title="Mites">mites</a> mais également un très bon isolant thermique. Comme tous les textiles de synthèse, son pouvoir absorbant est relativement faible. Il a également tendance à retenir l’<a class="mw-redirect" href="/wiki/%C3%89lectricit%C3%A9_statique" title="Électricité statique">électricité statique</a> et à jaunir sous l’effet de la chaleur.</li>
<li>L'<a href="/wiki/%C3%89lasthanne" title="Élasthanne">élasthanne</a> : cette matière synthétique est reconnue pour son excellente élasticité. L’élasthanne est d’ailleurs régulièrement mélangée à d’autres fibres textiles dans le but de les rendre plus extensibles. Outre cette qualité, cette matière très légère est agréable au toucher et suffisamment résistante. Elle a cependant tendance à jaunir de par sa sensibilité à la chaleur et à la lumière.</li></ul></li></ul>

<h3><span class="mw-headline" id="Fibres">Fibres</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=4" title="Modifier la section : Fibres">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=4" title="Modifier la section : Fibres">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>

<p>Les fibres textiles sont classées en trois grandes catégories :
</p>
<ul><li>Les fibres naturelles (existant à l'état naturel comme la <a href="/wiki/Fibre_de_bambou" title="Fibre de bambou">Fibre de bambou</a>) ; elles furent les premières à être utilisées pour la confection de <a href="/wiki/V%C3%AAtement" title="Vêtement">vêtements</a>.</li>
<li>Les fibres chimiques :
<ul><li>Les fibres artificielles (fabriquées à partir de matières premières naturelles).</li>
<li>Les fibres synthétiques (obtenues par réactions chimiques).</li></ul></li>
<li>Les fibres minérales (<a href="/wiki/Silicate" title="Silicate">silicate</a> mixte de <a href="/wiki/Carbone" title="Carbone">carbone</a> et de <a href="/wiki/Magn%C3%A9sium" title="Magnésium">magnésium</a>) permettent la confection de tissus ignifugés<sup class="reference" id="cite_ref-10"><a href="#cite_note-10"><span class="cite_crochet">[</span>10<span class="cite_crochet">]</span></a></sup> utilisés dès l'<a href="/wiki/Antiquit%C3%A9" title="Antiquité">Antiquité</a>.</li></ul>
<h3><span class="mw-headline" id="Filature">Filature</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=5" title="Modifier la section : Filature">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=5" title="Modifier la section : Filature">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Textile-Spinning_room.jpg"><img alt="" class="thumbimage" data-file-height="1704" data-file-width="2272" decoding="async" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b8/Textile-Spinning_room.jpg/220px-Textile-Spinning_room.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b8/Textile-Spinning_room.jpg/330px-Textile-Spinning_room.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b8/Textile-Spinning_room.jpg/440px-Textile-Spinning_room.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Textile-Spinning_room.jpg" title="Agrandir"></a></div>Filature industrielle.</div></div></div>
<p>La fabrication d'un fil nécessite le décorticage et le nettoyage de la matière première (égrenage), le desserrement et la parallélisation des fibres (<a href="/wiki/Cardage" title="Cardage">cardage</a>, peignage) puis enfin la filature.
</p><p>La réalisation d'un fil est une succession d'étapes dépendant de la qualité du fil souhaité et du type de fibres à travailler, mais qui comporte toujours au moins trois phases :
</p>
<ul><li>plusieurs filaments sont tirés de la filasse et rassemblés en mèche ;</li>
<li>la mèche est roulée en fil par torsion ;</li>
<li>le fil est mis en <a href="/wiki/Bobine_(textile)" title="Bobine (textile)">bobine</a> pour être tissé.</li></ul>
<p>Il existe deux grands processus de filature :
</p>
<ul><li>la filature pour fibres longues (filature type <a href="/wiki/Laine" title="Laine">laine</a>) ;</li>
<li>la filature pour fibres courtes (filature type <a href="/wiki/Coton" title="Coton">coton</a>).</li></ul>
<p>Pour ces deux processus, on part de bourres de fibres nettoyées, si nécessaire, qu'on transforme en <i>ruban</i> puis en <i>mèche</i> puis en fil.
</p>
<ul><li>Filature de fibres continues ou filage :
<ul><li>Pour les <a href="/wiki/Fibre_synth%C3%A9tique" title="Fibre synthétique">fibres synthétiques</a> : par filage, on obtient un filament. Les filaments sont convertis (coupés) ou craqués pour obtenir des fibres pouvant être mélangées ;</li>
<li>Pour les <a href="/wiki/Liste_de_fibres_naturelles" title="Liste de fibres naturelles">fibres naturelles</a> : Un <a class="mw-redirect" href="/wiki/Ver_%C3%A0_soie" title="Ver à soie">ver à soie</a> est capable de sécréter un filament pouvant mesurer jusqu'à 1 500 <abbr class="abbr" title="mètre">m</abbr>. La <a href="/wiki/Soie" title="Soie">soie</a> n'entre généralement pas dans les processus de filature décrits au-dessus. On assemble les filaments des soies, puis on fait un retordage de ces assemblages, qui peuvent ensuite être coupés pour être mélangés à d'autres fibres.</li></ul></li></ul>
<h3><span class="mw-headline" id="Tissage">Tissage</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=6" title="Modifier la section : Tissage">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=6" title="Modifier la section : Tissage">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Le tissu est obtenu par le <a href="/wiki/Tissage" title="Tissage">tissage</a> qui est le résultat de l'entrecroisement, dans un même <a href="/wiki/Plan_(math%C3%A9matiques)" title="Plan (mathématiques)">plan</a>, de fils disposés dans le sens de la <a href="/wiki/Fil_de_cha%C3%AEne" title="Fil de chaîne">chaîne</a> et de fils disposés, perpendiculairement aux fils de chaîne, dans le sens de la <a href="/wiki/Fil_de_trame" title="Fil de trame">trame</a>. Le liage obtenu entre ces fils de chaîne et trame se définit par une <a href="/wiki/Armure_(tissage)" title="Armure (tissage)">armure</a>.
</p><p>On distingue trois grandes classes fondamentales d'<a href="/wiki/Armure_(tissage)" title="Armure (tissage)">armures</a> : <a href="/wiki/Armure_(tissage)#Armure_toile" title="Armure (tissage)">toile</a>, <a href="/wiki/Serge_(tissu)" title="Serge (tissu)">sergé</a> et <a href="/wiki/Satin" title="Satin">satin</a>.
</p><p>Il existe des armures dérivées des trois précédentes : le reps, le cannelé, le croisé, le satin à répétition, etc. Un tissu peut être composé de plusieurs armures différentes et dans ce cas on parle de <i>tissu <a href="/wiki/Fa%C3%A7onn%C3%A9" title="Façonné">façonné</a></i> (e.g. Le <a href="/wiki/Velours_de_G%C3%AAnes" title="Velours de Gênes">velours de Gênes</a>).
</p><p>Le <a href="/wiki/M%C3%A9tier_Jacquard" title="Métier Jacquard">métier Jacquard</a> permet la sélection de fils de chaîne de façon indépendante tandis que les métiers à cadres font une sélection de cadres et donc de groupes de fils.
</p><p>Selon l'utilisation qui est faite des fils, on parle de <i>duites</i> (fils de trame) et de <i>fils</i> (fils de chaîne). On peut aussi trouver des fils <i>fantaisie</i> quand une grande importance est donnée à l'esthétisme du fil au lieu de sa régularité. Il existe aussi les fils dit <i>techniques</i> utilisés dans des applications techniques.
</p><p>Le tissage s'accompagne d'étapes précises dont les plus importantes sont :
</p>
<ul><li>le bobinage : les fils sont disposés sur les bobines ;</li>
<li>l'<a href="/wiki/Ourdissage" title="Ourdissage">ourdissage</a> : préparation de la chaîne sur le <a href="/wiki/M%C3%A9tier_%C3%A0_tisser" title="Métier à tisser">métier à tisser</a> ;</li>
<li>le rentrage : les fils de chaîne sont enfilés dans des tiges métalliques (les lisses) puis dans les dents du peigne ;</li>
<li>une fois la pièce textile descendue du métier industriel et jusqu'à l'informatisation, les tissus de luxe (<a class="new" href="/w/index.php?title=Drap_d%27Elbeuf&amp;action=edit&amp;redlink=1" title="Drap d'Elbeuf (page inexistante)">drap d'Elbeuf</a>) destiné à l'habillement connaissent le <a class="new" href="/w/index.php?title=Rentrayage&amp;action=edit&amp;redlink=1" title="Rentrayage (page inexistante)">rentrayage</a> qui consiste à réparer à l'aiguille les <a href="/wiki/Grappe" title="Grappe">grappes</a>, c'est-à-dire les erreurs commises par le métier à l'occasion de la rupture d'un fil de chaîne ou de trame ou tout autre incident.</li></ul>
<h3><span class="mw-headline" id="Tricot">Tricot</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=7" title="Modifier la section : Tricot">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=7" title="Modifier la section : Tricot">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Articles détaillés : <a href="/wiki/Tricot" title="Tricot">Tricot</a> et <a href="/wiki/Tricot_(textile)" title="Tricot (textile)">Tricot (textile)</a>.</div></div>
<p>Plusieurs méthodes de <a href="/wiki/Tricot" title="Tricot">tricotage</a> ont été développées.
</p><p>Le <a class="new" href="/w/index.php?title=Tricotage_%C3%A0_mailles_cueillies&amp;action=edit&amp;redlink=1" title="Tricotage à mailles cueillies (page inexistante)">tricotage à mailles cueillies</a>, appelé aussi tricotage <i>trame</i> est le plus connu. Il permet l'obtention de tricot <a href="/wiki/Jersey_(tissu)" title="Jersey (tissu)">jersey</a>, <a href="/wiki/Interlock" title="Interlock">interlock</a>, <a class="new" href="/w/index.php?title=C%C3%B4te_1x1&amp;action=edit&amp;redlink=1" title="Côte 1x1 (page inexistante)">côte 1x1</a>, <a class="new" href="/w/index.php?title=C%C3%B4te_anglaise&amp;action=edit&amp;redlink=1" title="Côte anglaise (page inexistante)">côte anglaise</a>, etc. Ces mailles sont fréquemment utilisées dans les <a href="/wiki/Sous-v%C3%AAtement" title="Sous-vêtement">sous-vêtements</a>, les <a href="/wiki/Tee-shirt" title="Tee-shirt">tee-shirts</a>, les <a href="/wiki/Pull-over" title="Pull-over">pull-overs</a>, les <a href="/wiki/Chaussette" title="Chaussette">chaussettes</a>, etc.
</p><p>Le <a class="new" href="/w/index.php?title=Tricotage_%C3%A0_mailles_jet%C3%A9es&amp;action=edit&amp;redlink=1" title="Tricotage à mailles jetées (page inexistante)">tricotage à mailles jetées</a> ou <i>chaîne</i> permet la réalisation d'articles indémaillables. Les <a href="/wiki/Armure_(tissage)" title="Armure (tissage)">armures</a> les plus fréquentes sont la <a class="new" href="/w/index.php?title=Charmeuse&amp;action=edit&amp;redlink=1" title="Charmeuse (page inexistante)">charmeuse</a>, l'atlas, le <a href="/wiki/Satin" title="Satin">satin</a>. Ces mailles sont utilisées dans la confection de <a href="/wiki/Maillot_de_bain" title="Maillot de bain">maillots de bain</a>, de <a href="/wiki/Lingerie" title="Lingerie">lingerie</a>, de voilages.
</p>
<h3><span id="Non-tiss.C3.A9"></span><span class="mw-headline" id="Non-tissé">Non-tissé</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=8" title="Modifier la section : Non-tissé">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=8" title="Modifier la section : Non-tissé">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a class="mw-redirect" href="/wiki/Textile_non-tiss%C3%A9" title="Textile non-tissé">Textile non-tissé</a>.</div></div>
<p>Les non-tissées sont des textiles dont les fibres sont maintenues de façon aléatoires, ils sont souvent classés selon leur domaine d'application ou leurs caractéristiques techniques.
</p>
<h4><span class="mw-headline" id="Voie_de_fabrication">Voie de fabrication</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=9" title="Modifier la section : Voie de fabrication">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=9" title="Modifier la section : Voie de fabrication">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul><li>voie humide (dite <i>papetière</i>)</li>

<h3><span class="mw-headline" id="Ennoblissement">Ennoblissement</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=12" title="Modifier la section : Ennoblissement">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=12" title="Modifier la section : Ennoblissement">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a href="/wiki/Ennoblissement" title="Ennoblissement">Ennoblissement</a>.</div></div>
<p>Les techniques d'ennoblissement ont pour but de modifier les propriétés du textile « brut ».
</p><p>Une fois les textiles préparés (flambage, désencollage, etc.), ils peuvent recevoir une opération de teinture ou d'impression.
</p><p>Pour leur donner « de la main » (du toucher), des apprêts mécaniques (émerisage, grattage, etc.) ou des apprêts chimiques sont utilisés. Enfin, des fonctions (bactériostatisme, déperlance, hydrophilie, protection UV, etc.) peuvent être <i>greffées</i> sur ces textiles par apprêts chimiques.
</p>

<h3><span id="Textiles_.C3.A0_usage_technique"></span><span class="mw-headline" id="Textiles_à_usage_technique">Textiles à usage technique</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=15" title="Modifier la section : Textiles à usage technique">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=15" title="Modifier la section : Textiles à usage technique">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Les TUT sont de plus en plus dénommés <a class="mw-redirect" href="/wiki/Textiles_techniques" title="Textiles techniques">textiles techniques</a> et fonctionnels. Ils contribuent à la diversification du secteur textile traditionnel, en réponse aux délocalisations notamment.
</p><p>Ils regroupent des tissages de matériaux dont les performances et propriétés fonctionnelles qui diffèrent de celles des fibres textiles traditionnelles. On les retrouvera notamment surtout dans des applications techniques et parfois '<b>extrêmes'</b> : ailes d'avions, voiles de bateaux, pansements, vestes de pompier, prothèses médicales, stabilisateur de route, para-grêle, dirigeables, etc.
</p><p>La production de TUT croît régulièrement depuis les années 1990 (Marché estimé à 65 milliards d'euros en 1995, puis à 85 milliards d'euros en 2005, et qui pourrait atteindre 100 milliards d'euros vers 2010.
</p><p>Le <abbr class="abbr" title="Premier">1<sup>er</sup></abbr> producteur et le <abbr class="abbr" title="Premier">1<sup>er</sup></abbr> consommateur de <a href="/wiki/Textile_technique" title="Textile technique">textile technique</a> en Europe serait l'<a href="/wiki/Allemagne" title="Allemagne">Allemagne</a> où la recherche est pilotée par un réseau de compétences supra régional dit <i>Conseil en Recherches Textiles</i>, avec 16 unités de recherche (en 2007) et de nombreux partenaires institutionnels et industriels.
</p>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a href="/wiki/G%C3%A9otextile" title="Géotextile">Géotextile</a>.</div></div>
<p>Un nouveau type de textile fait son apparition : les <a class="mw-redirect" href="/wiki/Textiles_intelligents" title="Textiles intelligents">Smart Textiles</a> incorporant de l'électronique pour plus d'interaction avec l'utilisateur ou l'environnement. Ces nouveaux produits promettent des applications dans les domaines du médical (vêtement avec capteur cardio-vasculaire, respiratoire, thermomètre, etc.), du loisir (veste avec lecteur MP3 intégré, mode : vêtement lumineux), de la sécurité (dossard clignotant et communicant, etc.). On évoque aussi des tissus susceptibles de produire de l'électricité, c'est-à-dire jouant le rôle de panneaux solaires, susceptible de recharger des batteries de téléphone, ordinateur, etc. voire d'alimenter des dirigeables.
</p>

<h2><span id=".C3.89conomie"></span><span class="mw-headline" id="Économie">Économie</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=22" title="Modifier la section : Économie">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=22" title="Modifier la section : Économie">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="Industrie_textile">Industrie textile</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Textile&amp;veaction=edit&amp;section=23" title="Modifier la section : Industrie textile">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Textile&amp;action=edit&amp;section=23" title="Modifier la section : Industrie textile">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a href="/wiki/Industrie_textile" title="Industrie textile">Industrie textile</a>.</div></div>
<div class="thumb tright"><div class="thumbinner" style="width:172px;"><a class="image" href="/wiki/Fichier:Konische_Garnspulen_ungef%C3%A4rbt.jpeg"><img alt="" class="thumbimage" data-file-height="2700" data-file-width="1762" decoding="async" height="260" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f7/Konische_Garnspulen_ungef%C3%A4rbt.jpeg/170px-Konische_Garnspulen_ungef%C3%A4rbt.jpeg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f7/Konische_Garnspulen_ungef%C3%A4rbt.jpeg/255px-Konische_Garnspulen_ungef%C3%A4rbt.jpeg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f7/Konische_Garnspulen_ungef%C3%A4rbt.jpeg/340px-Konische_Garnspulen_ungef%C3%A4rbt.jpeg 2x" width="170"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Konische_Garnspulen_ungef%C3%A4rbt.jpeg" title="Agrandir"></a></div>Bobines <a href="/wiki/C%C3%B4ne_(g%C3%A9om%C3%A9trie)" title="Cône (géométrie)">coniques</a> de fil textile</div></div></div>
<p>L'industrie textile débute par une chaîne de transformation partant de matières premières fibreuses jusqu'à des produits semi-ouvrés ou entièrement manufacturés. Elle regroupe de nombreux métiers et commence par transformer des matières premières (fibres naturelles, artificielles ou synthétiques, pigments, additifs) en fils. Les métiers associés sont notamment la <a href="/wiki/Filature_textile" title="Filature textile">filature</a>, le <a href="/wiki/Guipage" title="Guipage">guipage</a>, le <a href="/wiki/Moulinage" title="Moulinage">moulinage</a> ou encore la <a class="new" href="/w/index.php?title=Texturation&amp;action=edit&amp;redlink=1" title="Texturation (page inexistante)">texturation</a>. 
À partir de fils unidimensionnels, le tissage ou tricotage (ou des techniques de tissus non-tissés) produisent des surfaces textiles bidimensionnelles (voire tridimensionnelles<sup class="reference" id="cite_ref-13"><a href="#cite_note-13"><span class="cite_crochet">[</span>13<span class="cite_crochet">]</span></a></sup>) qui peuvent être teintes et/ou imprimées et/ou brodées. Des propriétés particulières leur sont éventuellement apportées par des apprêts chimiques, mécaniques, des enduction, contre-collage, etc.). Les surfaces textiles sont alors transformées en habits, meubles, rideaux ou utilisées pour de nombreux autres buts (ex : géotextiles stabilisant des routes, des voies ferrées..), pour drainer des terrains (agrotextiles), pour faire voler des hélicoptères, suppléer une articulation déficiente ou encore protéger un pompier du feu (textiles techniques fonctionnels).
</p><p>En déclin en occident, cette industrie demeure cependant dynamique dans les secteurs du textile technique et du textile de luxe. La majorité des  1 280 entreprises textile actives en France se situe dans les régions : <a href="/wiki/Alsace" title="Alsace">Alsace</a>, <a href="/wiki/Champagne-Ardenne" title="Champagne-Ardenne">Champagne</a>, <a href="/wiki/Lorraine" title="Lorraine">Lorraine</a>, <a href="/wiki/Midi-Pyr%C3%A9n%C3%A9es" title="Midi-Pyrénées">Midi-Pyrénées</a>, <a href="/wiki/Nord" title="Nord">Nord</a>, <a href="/wiki/Normandie" title="Normandie">Normandie</a>, <a href="/wiki/Picardie_(ancienne_r%C3%A9gion_administrative)" title="Picardie (ancienne région administrative)">Picardie</a>, <a href="/wiki/Rh%C3%B4ne-Alpes" title="Rhône-Alpes">Rhône-Alpes</a>, avec par exemple des entreprises telles que Jules Tournier &amp; Fils et la société Bel Maille, spécialistes des tissus techniques.
</p><p>L'industrie textile est florissante en <a href="/wiki/Asie" title="Asie">Asie</a> (<a href="/wiki/Inde" title="Inde">Inde</a>, <a href="/wiki/Bangladesh" title="Bangladesh">Bangladesh</a>, <a href="/wiki/Chine" title="Chine">Chine</a>, etc.) où sont implantées des usines officielles ou des sous-traitants (y compris travaillant pour de <a href="/wiki/Marque_(marketing)" title="Marque (marketing)">grandes marques</a>). Il est reproché à ces usines de rarement respecter le règlement européen <a class="mw-redirect" href="/wiki/REACH" title="REACH">REACH</a>, et de mal protéger la santé ou l'intégrité physique de leurs personnels (et souvent, des enfants y travaillent). Les rares usines traitant leurs eaux ne filtrent pas les produits chimiques (<a href="/wiki/Nonylph%C3%A9nol" title="Nonylphénol">Nonylphénol</a> par exemple) qui polluent l'eau et les sols locaux et contribuent à la pollution globale, aussi subie par les pays acheteurs de ces vêtements. Certains polluants et des <a href="/wiki/Microplastique" title="Microplastique">microplastiques</a>, voire des <a href="/wiki/Nanoplastique" title="Nanoplastique">nanoplastiques</a>  (dégradation des fibres synthétiques) quittent les vêtements au fur et à mesure des lavages, polluants des eaux mal filtrés par les usines d'épuration, ou les <a href="/wiki/Boues_d%27%C3%A9puration" title="Boues d'épuration">boues d'épuration</a>. 80 % des <a href="/wiki/L%C3%A9gume" title="Légume">légumes</a> français contiendraient notamment du <a href="/wiki/Nonylph%C3%A9nol" title="Nonylphénol">Nonylphénol</a>.
<br/>Les grandes marques (françaises ou autres) arborent des <a class="mw-redirect" href="/wiki/%C3%89colabel" title="Écolabel">écolabels</a>  ou des <a class="mw-redirect" href="/wiki/Assertion_environnementale" title="Assertion environnementale">assertions environnementales</a> ou socio-environnementales sur leur site mais ne peuvent ou ne veulent pas contrôler leur <a class="mw-redirect" href="/wiki/Cha%C3%AEne_d%27approvisionnement" title="Chaîne d'approvisionnement">chaîne d'approvisionnement</a>. L'<a class="mw-disambig" href="/wiki/%C3%89tiquetage" title="Étiquetage">étiquetages</a> des vêtements et autres textiles produits hors Europe ne permet pas de savoir si le règlement <a class="mw-redirect" href="/wiki/REACH" title="REACH">REACH</a> ou d'autres exigences environnementales sont respectés, ni la législation internationale du travail, y compris pour des vêtements haut-de-gamme si le textile a été importé de l'étranger sans <a href="/wiki/Tra%C3%A7abilit%C3%A9" title="Traçabilité">traçabilité</a> <sup class="reference" id="cite_ref-14"><a href="#cite_note-14"><span class="cite_crochet">[</span>14<span class="cite_crochet">]</span></a></sup>.
</p><p>Les ouvriers des usines textiles asiatiques sont en outre mal payés ; selon la sociologue et <a class="mw-redirect" href="/wiki/Autrice" title="Autrice">autrice</a> d'une enquête sur la « fast fashion » et ses impacts socio-environnementaux, résumée dans un ouvrage intitulé <i>Une mode éthique est-elle possible ?</i> (2018)<sup class="reference" id="cite_ref-15"><a href="#cite_note-15"><span class="cite_crochet">[</span>15<span class="cite_crochet">]</span></a></sup>,  <a class="new" href="/w/index.php?title=Majdouline_Sba%C3%AF&amp;action=edit&amp;redlink=1" title="Majdouline Sbaï (page inexistante)">majdouline Sbaï</a>, à la fin des années 2010, 50 % des textiles sont en outre <span class="citation">« vendus en soldes. Quand on regarde le découpage du prix d’un vêtement, il apparaît que les phases de production et de transport représentent le plus souvent moins de 10 % du prix final. Le salaire de l’ouvrier ne représente même pas 1 % de ce prix. 90 % de la valeur va à la conception, au <a href="/wiki/Marketing" title="Marketing">marketing</a>, à la <a class="mw-redirect" href="/wiki/Commercialisation" title="Commercialisation">distribution</a> et à la <a href="/wiki/Marge_commerciale" title="Marge commerciale">marge</a> (...) les <a href="/wiki/Soldes" title="Soldes">soldes</a> sont devenues le <a href="/wiki/Mod%C3%A8le_d%27entreprise" title="Modèle d'entreprise">business model</a> de la fast fashion (...) C’est une logique de vente événementielle quasi-permanente pour que les gens aient envie de revenir dans les <a class="mw-redirect" href="/wiki/Magasin_(commerce)" title="Magasin (commerce)">magasins</a>. Les <a href="/wiki/Marque_(marketing)" title="Marque (marketing)">marques</a> font du <a href="/wiki/R%C3%A9assortiment" title="Réassortiment">réassort</a> permanent pour que les gens aient soif de nouveauté. Le prix initial auquel était vendu un produit ne veut plus rien dire. Des marques produisent même spécialement pour les soldes. Certaines marques produisent en effet à ces occasions des vêtements avec le même design mais avec un tissu de moins bonne qualité pour le vendre moins cher »</span>, avec comme point d'orgue le <a class="mw-redirect" href="/wiki/Black_Friday_(shopping)" title="Black Friday (shopping)">Black Friday</a>. 
<br/>En réaction à cette tendance des mouvements comme le collectif <i><a class="new" href="/w/index.php?title=De_l%27%C3%A9thique_sur_l%27%C3%A9tiquette&amp;action=edit&amp;redlink=1" title="De l'éthique sur l'étiquette (page inexistante)">De l'éthique sur l'étiquette</a></i> (créé par quarante-sept syndicats et <a href="/wiki/Organisation_de_consommateurs" title="Organisation de consommateurs">organisations de consommateurs</a> et de <a href="/wiki/Solidarit%C3%A9_internationale_(concept)" title="Solidarité internationale (concept)">solidarité internationale</a>)<sup class="reference" id="cite_ref-16"><a href="#cite_note-16"><span class="cite_crochet">[</span>16<span class="cite_crochet">]</span></a></sup> plaident pour un <a href="/wiki/Commerce_%C3%A9quitable" title="Commerce équitable">commerce équitable</a>, le retour de la qualité et de la transparence sur les prix et les filières et pour une éthique sociale et <a class="mw-redirect" href="/wiki/%C3%89thique_environnementale" title="Éthique environnementale">environnementale</a><sup class="reference" id="cite_ref-17"><a href="#cite_note-17"><span class="cite_crochet">[</span>17<span class="cite_crochet">]</span></a></sup>, y compris dans le secteur de l'habillement<sup class="reference" id="cite_ref-18"><a href="#cite_note-18"><span class="cite_crochet">[</span>18<span class="cite_crochet">]</span></a></sup>
</p>
