<p>Une <b>lentille optique</b> est un composant fait d'un matériau généralement <span class="page_h"><a href="/wiki/Isotropie#Optique" title="Isotropie">isotrope</a></span> et transparent pour la lumière dans le domaine spectral d'intérêt. C'est le plus souvent un type de <a href="/wiki/Verre_optique" title="Verre optique">verre optique</a>, ou des verres plus classiques, des plastiques, des matériaux organiques, voire des <a href="/wiki/M%C3%A9tallo%C3%AFde" title="Métalloïde">métalloïdes</a> tels que le <a href="/wiki/Germanium" title="Germanium">germanium</a>. Les lentilles sont destinées à faire converger ou diverger la <a href="/wiki/Lumi%C3%A8re" title="Lumière">lumière</a>.
</p><p>Leur utilisation implique que leur <a href="/wiki/Indice_de_r%C3%A9fraction" title="Indice de réfraction">indice de réfraction</a> soit différent de celui du milieu dans lequel elles sont plongées (<a href="/wiki/Air" title="Air">air</a>, <a href="/wiki/Huile" title="Huile">huile</a>, <a href="/wiki/Eau" title="Eau">eau</a>…). Les lentilles possèdent la plupart du temps un axe de <a href="/wiki/Sym%C3%A9trie_(transformation_g%C3%A9om%C3%A9trique)" title="Symétrie (transformation géométrique)">symétrie</a> confondu avec l’<a href="/wiki/Axe_optique" title="Axe optique">axe optique</a> mais les techniques récentes et les besoins de l'industrie et de la recherche font qu'une part non négligeable des lentilles n'ont pas d'axe de symétrie.
</p><p>Il existe aussi des lentilles qui agissent sur d’autres types de <a href="/wiki/Rayonnement_%C3%A9lectromagn%C3%A9tique" title="Rayonnement électromagnétique">rayonnement électromagnétique</a>, au moyen d’un <a href="/wiki/Champ_%C3%A9lectrique" title="Champ électrique">champ électrique</a> et/ou d’un <a href="/wiki/Champ_magn%C3%A9tique" title="Champ magnétique">champ magnétique</a>.
</p>

<h2><span class="mw-headline" id="Histoire">Histoire</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=1" title="Modifier la section : Histoire">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=1" title="Modifier la section : Histoire">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>C'est à <a href="/wiki/Ninive" title="Ninive">Ninive</a>, ancienne capitale de l'<a class="mw-redirect" href="/wiki/Empire_assyrien" title="Empire assyrien">empire assyrien</a>, qu'au <a href="/wiki/XIXe_si%C3%A8cle" title="XIXe siècle"><abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle</a>, Sir <a href="/wiki/Austen_Henry_Layard" title="Austen Henry Layard">Austen Henry Layard</a> découvre les premiers verres lentiformes, dans des <a href="/wiki/Strate_(g%C3%A9ologie)" title="Strate (géologie)">couches</a> datant de 4 000 ans avant notre ère. On ignore à quoi ils servaient exactement, mais cette découverte soulève de nombreuses questions, parce que cette ville fut longtemps le grand centre commercial et culturel du <a href="/wiki/Moyen-Orient" title="Moyen-Orient">Moyen-Orient</a> et que la vulgarisation des connaissances acquises est une pratique rare dans l'<a href="/wiki/Antiquit%C3%A9" title="Antiquité">Antiquité</a>.
</p><p>Les premières mentions sans équivoque de l'utilisation d’une lentille proviennent de la <a href="/wiki/Gr%C3%A8ce_antique" title="Grèce antique">Grèce antique</a>. <a href="/wiki/Aristophane" title="Aristophane">Aristophane</a> évoque notamment dans sa pièce <i>Les Nuées</i>, écrite en <a class="mw-redirect" href="/wiki/-423" title="-423">423 av. J.-C.</a>, un <span class="citation">« <i>verre à feu</i> »</span> (une lentille convexe utilisée pour produire du <a href="/wiki/Feu" title="Feu">feu</a> en focalisant le <a href="/wiki/Rayonnement_solaire" title="Rayonnement solaire">rayonnement solaire</a>). Les écrits de <a href="/wiki/Pline_l%27Ancien" title="Pline l'Ancien">Pline l'Ancien</a> (<a href="/wiki/23" title="23">23</a>-<a href="/wiki/79" title="79">79</a>) montrent également qu’un tel dispositif était connu dans l’<a href="/wiki/Empire_romain" title="Empire romain">Empire romain</a>. Ils mentionnent ce qui peut être interprété comme la première utilisation d’une lentille pour corriger la <a href="/wiki/Vue" title="Vue">vue</a> en décrivant l’utilisation que fait <a href="/wiki/N%C3%A9ron" title="Néron">Néron</a> d’une <a href="/wiki/%C3%89meraude" title="Émeraude">émeraude</a> de forme convexe lors des spectacles de <a href="/wiki/Gladiateur" title="Gladiateur">gladiateurs</a> (probablement pour corriger un défaut de vision). <a href="/wiki/S%C3%A9n%C3%A8que" title="Sénèque">Sénèque</a> (<a class="mw-redirect" href="/wiki/-3" title="-3">3 av. J.-C.</a> - <a href="/wiki/65" title="65">65</a>) décrit l’effet grossissant d’un globe en verre rempli d’<a href="/wiki/Eau" title="Eau">eau</a>.
</p><p>Le <a href="/wiki/Math%C3%A9maticien" title="Mathématicien">mathématicien</a> <a href="/wiki/Arabe" title="Arabe">arabe</a> <a href="/wiki/Alhazen" title="Alhazen">Alhazen</a> (<a href="/wiki/965" title="965">965</a>-<a href="/wiki/1038" title="1038">1038</a>), a écrit le premier <a class="mw-redirect" href="/wiki/Trait%C3%A9_d%27optique_(Alhacen)" title="Traité d'optique (Alhacen)">traité d'optique</a> qui décrit comment le <a href="/wiki/Cristallin" title="Cristallin">cristallin</a> forme une image sur la <a href="/wiki/R%C3%A9tine" title="Rétine">rétine</a>.
</p><p>Les lentilles n’ont cependant pas été utilisées par le grand public avant la généralisation des <a class="mw-redirect" href="/wiki/Lunettes_de_vue" title="Lunettes de vue">lunettes de vue</a>, probablement inventées en <a href="/wiki/Italie" title="Italie">Italie</a> dans les années <a href="/wiki/1280" title="1280">1280</a>.
</p>
<h2><span class="mw-headline" id="Principe">Principe</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=2" title="Modifier la section : Principe">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=2" title="Modifier la section : Principe">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Articles détaillés : <a href="/wiki/Indice_de_r%C3%A9fraction" title="Indice de réfraction">Indice de réfraction</a>, <a href="/wiki/Verre_optique" title="Verre optique">Verre optique</a> et <a href="/wiki/R%C3%A9fraction" title="Réfraction">Réfraction</a>.</div></div>
<p>Lorsque l’onde lumineuse rencontre un <a href="/wiki/Dioptre" title="Dioptre">dioptre</a> (surface séparant deux milieux homogènes d’indices différents), la lumière est <a href="/wiki/R%C3%A9fraction" title="Réfraction">réfractée</a> en suivant les lois de <a class="mw-redirect" href="/wiki/Loi_de_Snell-Descartes" title="Loi de Snell-Descartes">Snell-Descartes</a>. Une lentille est l'association de deux dioptres et par conséquent va appliquer deux réfractions à la lumière incidente.
</p><p>Dans le cadre de l'optique géométrique, on considère ainsi que les rayons lumineux issus d'un objet sont réfractés par la lentille, sauf ceux passant par le <a href="/wiki/Centre_optique" title="Centre optique">centre optique</a> de la lentille.
</p>

<p>Dans le cadre plus général de l'optique physique où l'approximation des rayons lumineux n'est pas prise en compte mais où l'on considère que la lumière est une onde électromagnétique, l'effet des lentilles se considère différemment. Les surfaces d'onde arrivant sur la lentille vont voir leur propagation altérée par le nouveau milieu transparent et la lentille va ainsi modifier la surface d'onde. Ainsi on considère souvent qu'une onde plane est transformée en onde quasi sphérique par une lentille optique parfaite.
</p>
<h2><span id="Caract.C3.A9ristiques_d.E2.80.99une_lentille"></span><span class="mw-headline" id="Caractéristiques_d’une_lentille">Caractéristiques d’une lentille</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=3" title="Modifier la section : Caractéristiques d’une lentille">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=3" title="Modifier la section : Caractéristiques d’une lentille">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les lentilles sphériques possèdent un <a href="/wiki/Axe_optique" title="Axe optique">axe optique</a> qui est confondu avec leur axe de révolution. D'autres types de lentilles dont les dioptres ne sont pas des portions de sphère à symétrie de révolution, ne possèdent pas à proprement parler un axe optique. Par exemple les lentilles asphériques, <i>freeform</i> ou cylindriques n'ont pas d'axe optique en soi. On appelle <a href="/wiki/Centre_optique" title="Centre optique">centre optique</a> le point de l'axe optique par lequel les rayons peuvent passer sans être déviés.

<h3><span class="mw-headline" id="Foyers_objet_et_image">Foyers objet et image</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=4" title="Modifier la section : Foyers objet et image">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=4" title="Modifier la section : Foyers objet et image">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li>on appelle <b><a class="mw-redirect" href="/wiki/Foyer_image" title="Foyer image">foyer image</a></b> <i>F′</i> l’image d’un objet situé à l’infini sur l'axe optique principal : c’est donc le point où se focalisent des rayons qui se propagent parallèlement à l’axe optique<sup class="reference" id="cite_ref-Physique_MPSI-PSI-PTSI_page_61_1-0"><a href="#cite_note-Physique_MPSI-PSI-PTSI_page_61-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>.</li>
<li>on appelle <b><a class="mw-redirect" href="/wiki/Foyer_objet" title="Foyer objet">foyer objet</a></b> <i>F</i> le point dont l’image est située à l’infini sur l'axe optique principal : les rayons issus de ce point se propagent — après traversée de la lentille — parallèlement à l’axe optique<sup class="reference" id="cite_ref-Physique_MPSI-PSI-PTSI_page_61_1-1"><a href="#cite_note-Physique_MPSI-PSI-PTSI_page_61-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>.</li></ul>
<p>Le plan perpendiculaire à l’axe optique et passant par <i>F′</i> est appelé <b>plan focal image</b>, et ses points sont des <b>foyers image secondaires</b>.
De même, le plan perpendiculaire à l'axe optique et passant par le foyer objet <i>F</i> est appelé <b>plan focal objet</b>.
</p>

<h3><span class="mw-headline" id="Espaces_objet_et_image">Espaces objet et image</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=5" title="Modifier la section : Espaces objet et image">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=5" title="Modifier la section : Espaces objet et image">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>

<h3><span id="Stigmatisme_et_aplan.C3.A9tisme"></span><span class="mw-headline" id="Stigmatisme_et_aplanétisme">Stigmatisme et aplanétisme</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=6" title="Modifier la section : Stigmatisme et aplanétisme">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=6" title="Modifier la section : Stigmatisme et aplanétisme">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Articles détaillés : <a href="/wiki/Stigmatisme" title="Stigmatisme">Stigmatisme</a> et <a href="/wiki/Aplan%C3%A9tisme" title="Aplanétisme">Aplanétisme</a>.</div></div>
<p>Le <a href="/wiki/Stigmatisme" title="Stigmatisme">stigmatisme</a> d'une optique désigne la propriété d'une lentille à former pour un point objet un unique point image<sup class="reference" id="cite_ref-2"><a href="#cite_note-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>. Cette propriété n'est pas vérifiée en général pour des lentilles sphériques sauf dans 2 cas particuliers :
</p>
<ul><li>pour les points de Young-Weierstrass ;</li>
<li>pour le centre du rayon de courbure d'un <a href="/wiki/Dioptre" title="Dioptre">dioptre</a>.</li></ul>
<p>Une optique non stigmatique forme une <a href="/wiki/Tache" title="Tache">tache</a> image au lieu d'un point. En fonction de la configuration du <a href="/wiki/Syst%C3%A8me_optique" title="Système optique">système optique</a> il peut donc être préférable d'opter pour des <a href="/wiki/Lentille_asph%C3%A9rique" title="Lentille asphérique">lentilles asphériques</a>. Par exemple, dans le cas d'un objet situé à l'infini, une lentille à face parabolique est stigmatique<sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>.
</p><p>Un <a href="/wiki/Syst%C3%A8me_optique" title="Système optique">système optique</a> est dit <a href="/wiki/Aplan%C3%A9tisme" title="Aplanétisme">aplanétique</a> si l'image d'un objet <a class="mw-redirect" href="/wiki/Perpendiculaire" title="Perpendiculaire">perpendiculaire</a> à l'axe de <a href="/wiki/Sym%C3%A9trie" title="Symétrie">symétrie</a> du système est elle même perpendiculaire à ce dernier.
</p>
<h3><span class="mw-headline" id="Aberrations">Aberrations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=7" title="Modifier la section : Aberrations">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=7" title="Modifier la section : Aberrations">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Lens_chromatic_aberration.png"><img alt="" class="thumbimage" data-file-height="346" data-file-width="332" decoding="async" height="229" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f9/Lens_chromatic_aberration.png/220px-Lens_chromatic_aberration.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f9/Lens_chromatic_aberration.png/330px-Lens_chromatic_aberration.png 1.5x, //upload.wikimedia.org/wikipedia/commons/f/f9/Lens_chromatic_aberration.png 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Lens_chromatic_aberration.png" title="Agrandir"></a></div>L’aberration chromatique obtenue (en 1) peut être corrigée par l’ajout d’une deuxième lentille (en 2).</div></div></div>
<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Articles détaillés : <a href="/wiki/Aberration_g%C3%A9om%C3%A9trique" title="Aberration géométrique">Aberration géométrique</a> et <a href="/wiki/Aberration_chromatique" title="Aberration chromatique">Aberration chromatique</a>.</div></div>
<p>Les lentilles optiques ne forment pas des images parfaites : l’image d’un point n’est généralement pas un point, mais une tache du fait de la diffraction et des aberrations. Ces aberrations affectent la qualité des images, mais peuvent cependant être minimisées par l’association de deux lentilles de verres optiques différents qui agissent en correction.
</p><p>On classe les aberrations en deux grandes familles :
</p>
<ul><li><a href="/wiki/Aberration_chromatique" title="Aberration chromatique">aberrations chromatiques</a> : l’image se forme différemment selon la couleur de la lumière ;</li>
<li><a href="/wiki/Aberration_g%C3%A9om%C3%A9trique" title="Aberration géométrique">aberrations géométriques</a>, qui caractérisent les écarts au stigmatisme rigoureux, écarts d’autant plus grands que l’on s’éloigne des <a class="mw-redirect" href="/wiki/Conditions_de_Gauss" title="Conditions de Gauss">conditions de Gauss</a>.</li></ul>
<h2><span class="mw-headline" id="Types_de_lentilles">Types de lentilles</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=8" title="Modifier la section : Types de lentilles">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=8" title="Modifier la section : Types de lentilles">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Une <a href="/wiki/Lentille_mince" title="Lentille mince">lentille mince</a> est une lentille qui remplit les trois conditions suivantes :
</p>
<ul><li>l’épaisseur de la lentille est petite devant le rayon de la première face ;</li>
<li>l’épaisseur de la lentille est petite devant le rayon de la seconde face ;</li>
<li>l’épaisseur de la lentille est plus petite que la valeur absolue de la différence entre le rayon de la première face et le rayon de la deuxième face, les rayons étant comptés algébriquement.</li></ul>
<p>Les <a href="/wiki/Lentille_%C3%A9paisse" title="Lentille épaisse">lentilles épaisses</a> n’ont pas les mêmes propriétés que les <a href="/wiki/Lentille_mince" title="Lentille mince">lentilles minces</a>.
</p>
<h3><span id="Lentilles_sph.C3.A9riques"></span><span class="mw-headline" id="Lentilles_sphériques">Lentilles sphériques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=9" title="Modifier la section : Lentilles sphériques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=9" title="Modifier la section : Lentilles sphériques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Lens_shapes.svg"><img alt="" class="thumbimage" data-file-height="406" data-file-width="553" decoding="async" height="162" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/01/Lens_shapes.svg/220px-Lens_shapes.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/01/Lens_shapes.svg/330px-Lens_shapes.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/01/Lens_shapes.svg/440px-Lens_shapes.svg.png 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Lens_shapes.svg" title="Agrandir"></a></div>Différentes formes de lentilles sphériques avec les plans principaux associés. <p>Les lentilles sphériques se divisent en deux grandes familles<sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup> : 
</p>
<ul><li>lentilles convergentes (à bord mince) : rapprochent les rayons de l’axe optique en transformant un faisceau parallèle en un faisceau convergent ;</li>
<li>lentilles divergentes (à bord épais) : éloignent les rayons de l’axe optique en transformant un faisceau parallèle en un faisceau divergent.</li></ul>

<h3><span class="mw-headline" id="Lentilles_cylindriques">Lentilles cylindriques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=10" title="Modifier la section : Lentilles cylindriques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=10" title="Modifier la section : Lentilles cylindriques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:CylindricalLenses_2.png"><img alt="" class="thumbimage" data-file-height="589" data-file-width="750" decoding="async" height="173" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b0/CylindricalLenses_2.png/220px-CylindricalLenses_2.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b0/CylindricalLenses_2.png/330px-CylindricalLenses_2.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b0/CylindricalLenses_2.png/440px-CylindricalLenses_2.png 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:CylindricalLenses_2.png" title="Agrandir"></a></div>Deux lentilles cylindriques, plan-convexe à gauche et plan-concave à droite.</div></div></div>
<p>Les lentilles cylindriques sont des <a href="/wiki/Cylindre" title="Cylindre">cylindres</a> au sens large (forme obtenue en faisant translater une courbe selon une génératrice). Les dioptres sont eux-mêmes des cylindres. Elles sont parfois désignées sous l’appellation "lentilles anamorphiques" (en particulier dans le monde du <a href="/wiki/Cin%C3%A9ma" title="Cinéma">cinéma</a>) car elles peuvent être utilisées pour réaliser des <a href="/wiki/Anamorphose" title="Anamorphose">anamorphoses</a>.
</p><p>Pour les rayons se propageant dans un plan contenant une génératrice, elles se comportent comme une <a href="/wiki/Vitre" title="Vitre">vitre</a>, le rayon n’est pas dévié. Pour les rayons se propageant dans un plan perpendiculaire aux génératrices, elles se comportent comme une lentille classique à symétrie axiale.
</p><p>Elles sont principalement utilisées pour :
</p>
<ul><li>à but pédagogique : vues dans l’axe des génératrices (« de profil »), elles sont similaires à une lentille classique vue en coupe ; cette forme permet de les placer de manière stable sur une table, ou de les placer sur un tableau avec des aimants, et de visualiser le trajet des rayons ;</li>
<li>pour les <a href="/wiki/Anamorphose" title="Anamorphose">anamorphoses</a> : les rayons ne sont pas déviés de manière identique selon le plan de propagation, ce qui déforme l’image ; cette propriété est notamment utilisée en prise de vues anamorphique (<a href="/wiki/CinemaScope" title="CinemaScope">CinemaScope</a>), selon le principe de l’<a href="/wiki/Hypergonar" title="Hypergonar">Hypergonar</a> ;</li>
<li>pour la correction de l'<a href="/wiki/Astigmatisme_(optique)" title="Astigmatisme (optique)">astigmatisme</a>, par exemple en sortie de <a href="/wiki/Diode_laser" title="Diode laser">diode laser</a>.</li></ul>

<h2><span class="mw-headline" id="Lentille_diffractive">Lentille diffractive</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Lentille_optique&amp;veaction=edit&amp;section=13" title="Modifier la section : Lentille diffractive">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Lentille_optique&amp;action=edit&amp;section=13" title="Modifier la section : Lentille diffractive">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les lentilles dites diffractives peuvent être utilisées pour réduire le nombre d'éléments dans les systèmes de lentilles conventionnels et éliminer le besoin de matériaux pour corriger les <a class="mw-disambig" href="/wiki/Aberration" title="Aberration">aberrations</a> <a class="mw-disambig" href="/wiki/Chromatique" title="Chromatique">chromatiques</a>. Elles sont des éléments très minces avec une hauteur totale égale à λ / (n - 1), où λ est la longueur d'onde de fonctionnement et n est l'<a href="/wiki/Indice_de_r%C3%A9fraction" title="Indice de réfraction">indice de réfraction</a>.Composée d'une série de zones qui deviennent plus fines vers le bord de la lentille.
</p><p>Les éléments diffractifs fonctionnent le mieux sur une seule longueur d'onde, sur toute autre <a href="/wiki/Longueur_d%27onde" title="Longueur d'onde">longueur d'onde</a> l'efficacité et le contraste de l'<a href="/wiki/Image" title="Image">image</a> sont réduits.
</p>

<div class="bandeau-container bandeau-section metadata bandeau-niveau-information"><div class="bandeau-cell bandeau-icone-css loupe">Article détaillé : <a class="mw-redirect" href="/wiki/Lentille_adaptative" title="Lentille adaptative">lentille adaptative</a>.</div></div>
<p>Les gouttes d’eau comme la rosée peuvent être une forme de lentille naturelle. De petites tailles, elles ont la forme d'une calotte sphérique. Plus grosses, elles se déforment sous leur propre poids. La valeur limite est de quelques millimètres pour l'eau. Elles sont ainsi employées dans certaines optiques des appareils photos des <a class="mw-redirect" href="/wiki/T%C3%A9l%C3%A9phones_mobiles" title="Téléphones mobiles">téléphones mobiles</a>, permettant d'avoir une focale variable en déformant électriquement la lentille<sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>.
</p><p>Le <a href="/wiki/Cristallin" title="Cristallin">cristallin</a> de l'œil peut être modélisé en <a href="/wiki/Optique_g%C3%A9om%C3%A9trique" title="Optique géométrique">optique géométrique</a> comme une lentille convergente de <a href="/wiki/Vergence" title="Vergence">vergence</a> variable (proche de 60 <a href="/wiki/Dioptrie" title="Dioptrie">dioptries</a> pour un œil <a href="/wiki/Emm%C3%A9tropie" title="Emmétropie">emmétrope</a> au repos)<sup class="reference" id="cite_ref-6"><a href="#cite_note-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>. Grâce à l'action des muscles ciliaires, le cristallin (constitué de cellules souples) peut sensiblement se déformer pour s'adapter à une vision de près ou de loin. Cette capacité d'adaptation est l'<a href="/wiki/Accommodation" title="Accommodation">accommodation</a>, elle diminue progressivement avec l'âge pouvant aboutir à une <a href="/wiki/Presbytie" title="Presbytie">presbytie</a>.
</p>
