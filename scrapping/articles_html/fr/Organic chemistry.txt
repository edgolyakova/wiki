<p>La <b>chimie organique</b> est le domaine de la <a href="/wiki/Chimie" title="Chimie">chimie</a> qui étudie les <a href="/wiki/Compos%C3%A9_organique" title="Composé organique">composés organiques</a>, c'est-à-dire les <a href="/wiki/Compos%C3%A9_chimique" title="Composé chimique">composés</a> du <a href="/wiki/Carbone" title="Carbone">carbone</a> (à l'exception de quelques composés simples qui par tradition relèvent de la <a href="/wiki/Chimie_min%C3%A9rale" title="Chimie minérale">chimie minérale</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1"><span class="cite_crochet">[</span>a<span class="cite_crochet">]</span></a></sup>)<sup class="reference" id="cite_ref-2"><a href="#cite_note-2"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>. Ces composés peuvent être naturels ou <a href="/wiki/Synth%C3%A8se_organique" title="Synthèse organique">synthétiques</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>Une caractéristique du carbone consiste en l’aptitude qu’ont ses atomes à s’enchaîner les uns aux autres, par des <a href="/wiki/Liaison_covalente" title="Liaison covalente">liaisons covalentes</a>, d'une façon presque indéfinie, pour former des <a href="/wiki/Cha%C3%AEne_carbon%C3%A9e" title="Chaîne carbonée">chaînes carbonées</a> d’une grande diversité<sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>. Les composés organiques sont ainsi constitués de molécules caractérisées par des enchaînements carbonés propres aux molécules dites « organiques ».
</p><p>L'aptitude caractéristique du carbone implique qu'<span class="citation">« il suffit alors de quelques autres <a href="/wiki/%C3%89l%C3%A9ment_chimique" title="Élément chimique">éléments</a> […] pour former avec lui des millions de molécules différentes, dont la <a href="/wiki/Masse_mol%C3%A9culaire" title="Masse moléculaire">masse moléculaire</a> peut atteindre 100 000 ou même 1 000 000 ; on parle alors de <i><a href="/wiki/Macromol%C3%A9cule" title="Macromolécule">macromolécules</a></i> »</span><sup class="reference" id="cite_ref-arnaud_orga_5-0"><a href="#cite_note-arnaud_orga-5"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>.
</p><p>Les molécules organiques contiennent fréquemment des atomes d’<a href="/wiki/Hydrog%C3%A8ne" title="Hydrogène">hydrogène</a> et souvent des atomes d'<a href="/wiki/Oxyg%C3%A8ne" title="Oxygène">oxygène</a> ou d'<a href="/wiki/Azote" title="Azote">azote</a><sup class="reference" id="cite_ref-arnaud_orga_5-1"><a href="#cite_note-arnaud_orga-5"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup> et les molécules synthétiques proviennent souvent du pétrole.
</p><p>La chimie organique étudie en particulier leur <a href="/wiki/Structure_chimique" title="Structure chimique">structure chimique</a>, leurs propriétés, leurs caractéristiques, leur <a href="/wiki/Composition_chimique" title="Composition chimique">composition chimique</a>, leurs <a href="/wiki/R%C3%A9action_chimique" title="Réaction chimique">réactions chimiques</a> et leur préparation (par <a href="/wiki/Synth%C3%A8se_chimique" title="Synthèse chimique">synthèse</a> ou autres moyens). Ces composés peuvent comprendre d'autres éléments chimiques, comme les <a href="/wiki/Halog%C3%A8ne" title="Halogène">halogènes</a> (<a href="/wiki/Fluor" title="Fluor">fluor</a>, <a href="/wiki/Chlore" title="Chlore">chlore</a>, <a href="/wiki/Brome" title="Brome">brome</a>, <a href="/wiki/Iode" title="Iode">iode</a>) ainsi que le <a href="/wiki/Bore" title="Bore">bore</a>, le <a href="/wiki/Silicium" title="Silicium">silicium</a>, le <a href="/wiki/Phosphore" title="Phosphore">phosphore</a>, le <a href="/wiki/Soufre" title="Soufre">soufre</a> ; plus rarement, le <a href="/wiki/Lithium" title="Lithium">lithium</a>, le <a href="/wiki/Sodium" title="Sodium">sodium</a>, le <a href="/wiki/Magn%C3%A9sium" title="Magnésium">magnésium</a>, le <a href="/wiki/Cuivre" title="Cuivre">cuivre</a>, le <a href="/wiki/Titane" title="Titane">titane</a>, le <a href="/wiki/Potassium" title="Potassium">potassium</a>, le <a href="/wiki/Fer" title="Fer">fer</a>, le <a href="/wiki/Cobalt" title="Cobalt">cobalt</a>, le <a href="/wiki/Zinc" title="Zinc">zinc</a> et le <a href="/wiki/Plomb" title="Plomb">plomb</a>. Cette dernière est appelée <a href="/wiki/Chimie_organom%C3%A9tallique" title="Chimie organométallique">chimie organométallique</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>.
</p><p>La première définition de la chimie « organique » par <a href="/wiki/Nicolas_L%C3%A9mery" title="Nicolas Lémery">Nicolas Lémery</a> dans son <i>Cours de chimie</i> publié en 1690 était due à la conception erronée selon laquelle les <a href="/wiki/Compos%C3%A9_organique" title="Composé organique">composés organiques</a> seraient les seuls entrant en jeu dans les processus du <a href="/wiki/Vie" title="Vie">vivant</a>. Cependant, les molécules organiques peuvent être produites par des processus sans rapport avec le vivant et le vivant dépend aussi de la <a class="mw-redirect" href="/wiki/Chimie_inorganique" title="Chimie inorganique">chimie inorganique</a>. Par exemple, de nombreuses <a href="/wiki/Enzyme" title="Enzyme">enzymes</a> ont besoin de <a href="/wiki/M%C3%A9tal_de_transition" title="Métal de transition">métaux de transition</a> comme le <a href="/wiki/Fer" title="Fer">fer</a> ou le <a href="/wiki/Cuivre" title="Cuivre">cuivre</a> pour être actifs ; et des matériaux comme les <a href="/wiki/Coquille_(mollusque)" title="Coquille (mollusque)">coquillages</a>, les <a href="/wiki/Dent" title="Dent">dents</a> ou les <a href="/wiki/Os" title="Os">os</a> sont constitués en partie de composés organiques et en partie de <a href="/wiki/Compos%C3%A9_inorganique" title="Composé inorganique">matière inorganique</a> (minérale).
</p><p>Bien qu'il y ait un recouvrement avec la <a href="/wiki/Biochimie" title="Biochimie">biochimie</a>, cette dernière s'intéresse spécifiquement aux molécules fabriquées par les <a class="mw-redirect" href="/wiki/Organisme_vivant" title="Organisme vivant">organismes vivants</a> qui appartiennent aux grands groupes classiques (<a href="/wiki/Lipide" title="Lipide">lipides</a>, <a href="/wiki/Glucide" title="Glucide">glucides</a>, <a href="/wiki/Prot%C3%A9ine" title="Protéine">protides</a>, <a href="/wiki/Acide_nucl%C3%A9ique" title="Acide nucléique">acides nucléiques</a>) ainsi qu'aux petites molécules produites par le <a href="/wiki/M%C3%A9tabolisme_cellulaire" title="Métabolisme cellulaire">métabolisme</a>. Les <a href="/wiki/Compos%C3%A9_organique" title="Composé organique">composés organiques</a> sont donc au cœur de ces disciplines. On les désignera sous le terme général de « substances » organiques qui inclut des macromolécules comme les <a href="/wiki/Prot%C3%A9ine" title="Protéine">protéines</a> (polymères <a href="/wiki/Polypeptide" title="Polypeptide">polypeptidiques</a>).
</p>
<meta property="mw:PageProp/toc"/>
<h2><span class="mw-headline" id="Historique">Historique</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chimie_organique&amp;veaction=edit&amp;section=1" title="Modifier la section : Historique">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chimie_organique&amp;action=edit&amp;section=1" title="Modifier la section : Historique">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>La chimie organique s'oppose par ailleurs à la <a class="mw-redirect" href="/wiki/Chimie_inorganique" title="Chimie inorganique">chimie inorganique</a> (<i>minérale</i> ou « générale »), laquelle s'occupe de l'étude des substances issues du monde minéral (la <a href="/wiki/Terre" title="Terre">Terre</a>, l'<a href="/wiki/Eau" title="Eau">eau</a> et l'<a href="/wiki/Atmosph%C3%A8re_terrestre" title="Atmosphère terrestre">atmosphère</a>). Cette séparation tient au fait que jusqu'au début du <a href="/wiki/XIXe_si%C3%A8cle" title="XIXe siècle"><abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle</a>, les chimistes pensaient généralement que les composés des organismes vivants étaient trop complexes de par leur structure et que l'homme ne pouvait les synthétiser car leur formation avait nécessité l'intervention d'une « <a class="mw-redirect" href="/wiki/Force_vitale" title="Force vitale">force vitale</a> » (voir <a href="/wiki/Vitalisme" title="Vitalisme">vitalisme</a>). Ces composés étaient également particuliers du fait qu'ils pouvaient se reproduire. Ils appelèrent ces composés « organiques » et continuèrent à les ignorer<sup class="reference" id="cite_ref-7"><a href="#cite_note-7"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.
</p><p>L'essor de la chimie organique commença lorsque les chimistes découvrirent que ces composés pouvaient être abordés de façon similaire aux <a class="mw-redirect" href="/wiki/Chimie_inorganique" title="Chimie inorganique">composés inorganiques</a> et pouvaient être recrées en laboratoire sans avoir recours à la « force vitale ». Aux alentours de 1816, <a href="/wiki/Michel-Eug%C3%A8ne_Chevreul" title="Michel-Eugène Chevreul">Eugène Chevreul</a> commença une étude des <a href="/wiki/Savon" title="Savon">savons</a> à partir de différents <a href="/wiki/Corps_gras" title="Corps gras">corps gras</a> et <a href="/wiki/Alcali" title="Alcali">alcalis</a>. Il sépara les différents acides qui, en combinaison avec les alcali, produisaient le savon. Ainsi, il démontra qu'il était possible de changer chimiquement les graisses afin de produire de nouveaux composés sans l'aide d'une « force vitale ». En 1828, <a href="/wiki/Friedrich_W%C3%B6hler" title="Friedrich Wöhler">Friedrich Wöhler</a> fut le premier à produire l'<a href="/wiki/Ur%C3%A9e" title="Urée">urée</a>, un constituant de l'<a href="/wiki/Urine" title="Urine">urine</a> qui est une molécule organique, à partir du <a href="/wiki/Cyanate_d%27ammonium" title="Cyanate d'ammonium">cyanate d'ammonium</a> NH<sub>4</sub>OCN qui est un réactif inorganique. Cette réaction fut ensuite appelée la <a href="/wiki/Synth%C3%A8se_de_W%C3%B6hler" title="Synthèse de Wöhler">synthèse de Wöhler</a>. Il fut très prudent et ne déclara pas, ni à ce moment ni plus tard, la fin de la « <a href="/wiki/Tao_(culture)" title="Tao (culture)">théorie de la force vitale</a> », mais ceci est maintenant considéré comme le tournant historique.
</p><p>De 1850 à 1865, le chimiste français <a href="/wiki/Marcellin_Berthelot" title="Marcellin Berthelot">Marcellin Berthelot</a> (1827-1907), professeur au <a href="/wiki/Coll%C3%A8ge_de_France" title="Collège de France">Collège de France</a>, se consacre à la <a href="/wiki/Synth%C3%A8se_chimique" title="Synthèse chimique">synthèse organique</a> et reconstitue le <a href="/wiki/M%C3%A9thane" title="Méthane">méthane</a>, le <a href="/wiki/M%C3%A9thanol" title="Méthanol">méthanol</a>, l'<a href="/wiki/Ac%C3%A9tyl%C3%A8ne" title="Acétylène">éthyne</a> et le <a href="/wiki/Benz%C3%A8ne" title="Benzène">benzène</a> à partir de leurs <a href="/wiki/%C3%89l%C3%A9ment_chimique" title="Élément chimique">éléments</a>, et expose ses théories dans son livre <i>La Chimie Organique Fondée sur la Synthèse</i>.
</p><p>Un autre grand pas fut franchi en 1856 lorsque <a href="/wiki/William_Henry_Perkin" title="William Henry Perkin">William Henry Perkin</a>, alors qu'il cherchait à produire de la <a href="/wiki/Quinine" title="Quinine">quinine</a>, synthétisa de manière accidentelle la <a href="/wiki/Teinture" title="Teinture">teinture</a> organique maintenant appelée <a href="/wiki/Mauv%C3%A9ine" title="Mauvéine">mauvéine</a>. Cette découverte rapporta beaucoup d'argent et augmenta l'intérêt pour la chimie organique. Une autre étape fut la préparation en laboratoire du <a href="/wiki/Dichlorodiph%C3%A9nyltrichloro%C3%A9thane" title="Dichlorodiphényltrichloroéthane">DDT</a> par Othmer Zeidler en 1874, mais les propriétés <a href="/wiki/Insecticide" title="Insecticide">insecticides</a> de la molécule ne furent découvertes que beaucoup plus tard.
</p><p>Une avancée cruciale pour la chimie organique fut le développement du concept de structure chimique, de manière indépendante et simultanée par <a href="/wiki/Friedrich_Kekul%C3%A9_von_Stradonitz" title="Friedrich Kekulé von Stradonitz">Friedrich August Kekule</a> et <a href="/wiki/Archibald_Scott_Couper" title="Archibald Scott Couper">Archibald Scott Couper</a> en 1858. Les deux hommes suggérèrent que les atomes de carbone <a href="/wiki/T%C3%A9travalence" title="Tétravalence">tétravalents</a> pouvaient se lier les uns aux autres afin de former un squelette carboné et que les détails des liaisons entre les atomes pouvaient être découverts par une interprétation de certaines réactions chimiques.
</p><p>Le développement de la chimie organique continua avec la découverte des <a href="/wiki/Hydrocarbure" title="Hydrocarbure">hydrocarbures</a> et de leur séparation par <a href="/wiki/Distillation_fractionn%C3%A9e" title="Distillation fractionnée">distillation fractionnée</a> en <a href="/wiki/Compos%C3%A9_chimique" title="Composé chimique">composés chimiques</a> de <a href="/wiki/Point_d%27%C3%A9bullition" title="Point d'ébullition">points d'ébullition</a> différents. La transformation des différents composants du pétrole grâce à des procédés chimiques de plus en plus nombreux engendra l'<a href="/wiki/P%C3%A9trochimie" title="Pétrochimie">industrie pétrochimique</a> dont dérive la synthèse du <a href="/wiki/Caoutchouc_(mat%C3%A9riau)" title="Caoutchouc (matériau)">caoutchouc</a>, de plusieurs <a href="/wiki/Adh%C3%A9sif" title="Adhésif">adhésifs</a> organiques et des <a href="/wiki/Mati%C3%A8re_plastique" title="Matière plastique">plastiques</a>.
</p><p>L'<a href="/wiki/Industrie_pharmaceutique" title="Industrie pharmaceutique">industrie pharmaceutique</a> débuta pendant la dernière décennie du <abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle lorsque la production d'acide acétylsalicylique, plus connu sous le nom d'<a href="/wiki/Acide_ac%C3%A9tylsalicylique" title="Acide acétylsalicylique">aspirine</a>, commença en <a href="/wiki/Allemagne" title="Allemagne">Allemagne</a> par <a href="/wiki/Bayer_(entreprise)" title="Bayer (entreprise)">Bayer</a>.
</p><p>La première tentative d'amélioration systématique d'un médicament eut lieu avec le développement de l'<a href="/wiki/Arsph%C3%A9namine" title="Arsphénamine">arsphénamine</a> (Salvarsan). De nombreux <a href="/wiki/D%C3%A9riv%C3%A9_(chimie)" title="Dérivé (chimie)">dérivés</a> d'une molécule active mais très toxique <a href="/wiki/Arsanilate_de_sodium" title="Arsanilate de sodium">(atoxyl</a>) qu'on qualifierait maintenant de <i><a class="new" href="/w/index.php?title=Lead_compound&amp;action=edit&amp;redlink=1" title="Lead compound (page inexistante)">lead compound</a> <a class="extiw" href="https://en.wikipedia.org/wiki/Lead_compound" title="en:Lead compound"><span class="indicateur-langue" title="Article en anglais : « Lead compound »">(en)</span></a></i>, furent synthétisés et testés par <a href="/wiki/Paul_Ehrlich" title="Paul Ehrlich">Paul Ehrlich</a> et son équipe. À l'issue de cette optimisation (un <i><a href="/wiki/Conception_de_m%C3%A9dicament" title="Conception de médicament">drug design</a></i> embryonnaire), le composé présentant le meilleur rapport efficacité/toxicité fut sélectionné pour la production.
</p><p>Les premières réactions organiques furent souvent le résultat de découvertes <a href="/wiki/S%C3%A9rendipit%C3%A9" title="Sérendipité">fortuites</a> mais, à partir de la seconde moitié du <abbr class="abbr" title="19ᵉ siècle"><span class="romain">XIX</span><sup style="font-size:72%">e</sup></abbr> siècle, l'étude systématique des composés organiques se développa. Au début du <abbr class="abbr" title="20ᵉ siècle"><span class="romain">XX</span><sup style="font-size:72%">e</sup></abbr> siècle, les progrès en chimie organique permirent la synthèse de molécules complexes en suivant un protocole par étapes. Au même moment, on découvrit que les polymères et les enzymes étaient des molécules organiques de grande taille et que le pétrole était d'origine biologique.
</p><p>Le processus permettant de synthétiser une molécule précise à partir de précurseurs simples - et le plus souvent commerciaux - est appelé <a href="/wiki/Synth%C3%A8se_totale" title="Synthèse totale">synthèse totale</a>. La <a href="/wiki/Synth%C3%A8se_totale" title="Synthèse totale">synthèse totale</a> de composés naturels complexes, initiée avec la préparation de l'<a href="/wiki/Ur%C3%A9e" title="Urée">urée</a>, gagna en complexité avec le <a href="/wiki/Glucose" title="Glucose">glucose</a> et le <a href="/wiki/Terpin%C3%A9ol" title="Terpinéol">terpineol</a> et, en 1907, la synthèse totale fut utilisée dans un but commercial pour la première fois par <a class="new" href="/w/index.php?title=Gustaf_Komppa&amp;action=edit&amp;redlink=1" title="Gustaf Komppa (page inexistante)">Gustaf Komppa</a> avec le <a href="/wiki/Camphre" title="Camphre">camphre</a>. Les avancées dans le domaine pharmaceutique ont été conséquentes : il devint possible de synthétiser des hormones humaines complexes (<a href="/wiki/St%C3%A9ro%C3%AFde" title="Stéroïde">stéroïdes</a>, <a href="/wiki/Insuline" title="Insuline">insuline</a>) et d'en obtenir des dérivés. Depuis le début du <abbr class="abbr" title="20ᵉ siècle"><span class="romain">XX</span><sup style="font-size:72%">e</sup></abbr> siècle, la puissance de la synthèse totale n'a cessé d'augmenter, rendant possible la préparation de molécules aussi complexes que la <a href="/wiki/Vitamine_B12" title="Vitamine B12">vitamine B12</a>. De nos jours, les composés synthétisés peuvent comporter des dizaines de <a href="/wiki/Centre_st%C3%A9r%C3%A9og%C3%A8ne" title="Centre stéréogène">centres stéréogènes</a> dont la stéréochimie peut être contrôlée grâce à la <a href="/wiki/Synth%C3%A8se_asym%C3%A9trique" title="Synthèse asymétrique">synthèse asymétrique</a>.
</p><p>Actuellement, plus de quarante-cinq millions de composés sont disponibles<sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>7<span class="cite_crochet">]</span></a></sup>, souvent obtenus par voie synthétique et parmi lesquels rares sont les produits que l'on trouve dans la nature.
</p><p>La chimie organique se définit maintenant simplement par l'étude des composés à base de <a href="/wiki/Carbone" title="Carbone">carbone</a> autres que les <a href="/wiki/Oxyde_de_carbone" title="Oxyde de carbone">oxydes de carbone</a>, les <a href="/wiki/Cyanure" title="Cyanure">cyanures</a>, les <a href="/wiki/Carbonate" title="Carbonate">carbonates</a> et les <a href="/wiki/Carbure" title="Carbure">carbures</a> autres que les <a href="/wiki/Hydrocarbure" title="Hydrocarbure">hydrocarbures</a>. On l'appelle également la <i>chimie du carbone</i> (voir aussi <a href="/wiki/Compos%C3%A9_organique" title="Composé organique">Composé organique</a>).
</p>
<h2><span id="Caract.C3.A9ristiques"></span><span class="mw-headline" id="Caractéristiques">Caractéristiques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chimie_organique&amp;veaction=edit&amp;section=2" title="Modifier la section : Caractéristiques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chimie_organique&amp;action=edit&amp;section=2" title="Modifier la section : Caractéristiques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>La raison pour laquelle il existe autant de composés carbonés est la capacité du carbone de former des <a href="/wiki/Liaison_covalente" title="Liaison covalente">liaisons covalentes</a> avec lui-même et donc de former de nombreuses <a href="/wiki/Cha%C3%AEne_carbon%C3%A9e" title="Chaîne carbonée">chaînes</a> de différentes longueurs, ainsi que des <a href="/wiki/Cycloalcane" title="Cycloalcane">cycles</a> de différentes tailles. La plupart des composés organiques sont fort sensibles à la <a href="/wiki/Temp%C3%A9rature" title="Température">température</a> et se décomposent généralement au-dessus de <span title="392 °F ou 473,2 K">200 </span><abbr class="abbr" title="degré Celsius">°C</abbr>. Ils ont tendance à être peu <a href="/wiki/Solubilit%C3%A9" title="Solubilité">solubles</a> dans l'<a href="/wiki/Eau" title="Eau">eau</a>, en tout cas moins solubles que les sels inorganiques. En revanche, et à l'inverse de tels sels, ils ont tendance à être solubles dans les <a href="/wiki/Solvant" title="Solvant">solvants</a> organiques tels que l'<a href="/wiki/%C3%89ther_di%C3%A9thylique" title="Éther diéthylique">éther diéthylique</a> ou l'<a href="/wiki/%C3%89thanol" title="Éthanol">éthanol</a>. D'une manière générale, on peut retenir que les semblables (molécules plus ou moins <a href="/wiki/Polarit%C3%A9_(chimie)" title="Polarité (chimie)">polaires</a>, <a href="/wiki/Protique" title="Protique">protiques</a>…) dissolvent les semblables.
</p>
<h2><span id="Repr.C3.A9sentation"></span><span class="mw-headline" id="Représentation">Représentation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chimie_organique&amp;veaction=edit&amp;section=3" title="Modifier la section : Représentation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chimie_organique&amp;action=edit&amp;section=3" title="Modifier la section : Représentation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les composés organiques sont constitués d'<a href="/wiki/Atome" title="Atome">atomes</a> de <a href="/wiki/Carbone" title="Carbone">carbone</a> et d'<a href="/wiki/Hydrog%C3%A8ne" title="Hydrogène">hydrogène</a> ; leur structure peut posséder d'autres atomes. Dans un souci de simplification, les chimistes ont pris l'habitude de représenter les molécules qu'ils manipulent sans faire figurer les atomes de carbone et d'hydrogène. Cette représentation est appelée <a href="/wiki/Formule_topologique" title="Formule topologique">formule topologique</a>.
</p>

<h3><span id="Compos.C3.A9s_aromatiques"></span><span class="mw-headline" id="Composés_aromatiques">Composés aromatiques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chimie_organique&amp;veaction=edit&amp;section=6" title="Modifier la section : Composés aromatiques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chimie_organique&amp;action=edit&amp;section=6" title="Modifier la section : Composés aromatiques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Un <a href="/wiki/Compos%C3%A9_aromatique" title="Composé aromatique">composé aromatique</a> doit répondre à trois critères :
</p>
<ul><li>être une molécule <a href="/wiki/Compos%C3%A9_cyclique" title="Composé cyclique">cyclique</a> contenant des atomes de carbone et pouvant contenir des <a href="/wiki/H%C3%A9t%C3%A9roatome" title="Hétéroatome">hétéroatomes</a> ;</li>
<li>avoir un <a href="/wiki/Syst%C3%A8me_conjugu%C3%A9" title="Système conjugué">système conjugué</a> de types π-σ-π ou p-σ-π (avec σ = <a href="/wiki/Liaison_covalente" title="Liaison covalente">liaison simple</a>, π = <a href="/wiki/Liaison_double" title="Liaison double">double liaison</a> et p = <a href="/wiki/Doublet_non_liant" title="Doublet non liant">doublet non liant</a>) ;</li>
<li>respecter la <a href="/wiki/R%C3%A8gle_de_H%C3%BCckel" title="Règle de Hückel">règle de Hückel</a>.</li></ul>
<p>Si le cycle contient au moins un atome autre que du carbone (en général <a href="/wiki/Azote" title="Azote">N</a> et/ou <a href="/wiki/Oxyg%C3%A8ne" title="Oxygène">O</a>, parfois <a href="/wiki/Soufre" title="Soufre">S</a>, mais d'autres possibilités existent : le sélénium par exemple), on parle d'<i>hétérocycle</i>.
</p>
<ul><li>Composés <a href="/wiki/Benz%C3%A8ne" title="Benzène">benzéniques</a> :
<ul><li>benzène et ses dérivés (<a href="/wiki/Ph%C3%A9nol_(groupe)" title="Phénol (groupe)">phénols</a>, <a href="/wiki/Amine_aromatique" title="Amine aromatique">amines aromatiques</a>, etc.)</li>
<li><a href="/wiki/Hydrocarbure_aromatique_polycyclique" title="Hydrocarbure aromatique polycyclique">benzoïdes</a> (benzènes <a href="/wiki/Compos%C3%A9_polycyclique" title="Composé polycyclique">polycycliques</a>)</li>
<li><a href="/wiki/Fuller%C3%A8ne" title="Fullerène">fullerènes</a></li></ul></li></ul>
<ul><li><a href="/wiki/H%C3%A9t%C3%A9rocycle" title="Hétérocycle">Hétérocycles</a> :
<ul><li><a href="/wiki/Pyridine" title="Pyridine">pyridine</a></li>
<li><a href="/wiki/Furane" title="Furane">furane</a></li>
<li><a href="/wiki/Thioph%C3%A8ne" title="Thiophène">thiophène</a></li>
<li><a href="/wiki/Pyrrole" title="Pyrrole">pyrrole</a></li>
