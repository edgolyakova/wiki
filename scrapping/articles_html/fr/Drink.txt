<p>Une <b>boisson</b>, ou un <b>breuvage</b>, est un <a href="/wiki/Liquide" title="Liquide">liquide</a> destiné à la consommation. On trouve des boissons froides ou chaudes, gazeuses, alcoolisées ou non alcoolisées.
</p><p>Le terme est aussi utilisé dans le sens plus restaurant de <a href="/wiki/Boisson_alcoolis%C3%A9e" title="Boisson alcoolisée">boisson alcoolisée</a>, comme dans les expressions « pris de boisson » ou « sous l'effet de la boisson ».
</p><p>La boisson la plus naturelle est l'<a href="/wiki/Eau" title="Eau">eau</a>, la seule que boivent les animaux (avec le lait maternel chez les <a href="/wiki/Mammif%C3%A8re" title="Mammifère">mammifères</a>) et la seule qui réhydrate efficacement. L'eau est aussi le composant essentiel de la plupart des autres boissons.
</p><p>La première boisson que boivent les <a href="/wiki/Homo_sapiens" title="Homo sapiens">êtres humains</a> (ainsi que tous les mammifères) est le <a href="/wiki/Lait" title="Lait">lait</a> maternel (parfois remplacé par du lait de vache), lors de l'<a class="mw-redirect" href="/wiki/Allaitement" title="Allaitement">allaitement</a> et jusqu'au <a href="/wiki/Sevrage_(alimentation)" title="Sevrage (alimentation)">sevrage</a>.
</p><p>"La boisson" désigne aussi un alcool de Normandie composé de marc de cidre mélangé avec de l'eau qui remplaçait l'eau jusqu'aux années 1960 et qu'enfants comme adultes buvaient lors des repas et dans la journée.
</p>

<h2><span class="mw-headline" id="Origine">Origine</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=1" title="Modifier la section : Origine">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=1" title="Modifier la section : Origine">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>L'eau est une boisson d'origine <a href="/wiki/Min%C3%A9ral" title="Minéral">minérale</a>.</li></ul>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a class="image" href="/wiki/Fichier:Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg"><img alt="" class="thumbimage" data-file-height="888" data-file-width="1297" decoding="async" height="151" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/24/Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg/220px-Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/24/Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg/330px-Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/24/Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg/440px-Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg 2x" width="220"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Deutsches_Reitpony_-_Stute_mit_s%C3%A4ugendem_Fohlen.jpg" title="Agrandir"></a></div><a href="/wiki/Jument" title="Jument">Jument</a> allaitant son poulain.</div></div></div>
<ul><li>Le lait et les <a href="/wiki/Boisson_lact%C3%A9e" title="Boisson lactée">boissons lactées</a>, sont d'origine <a href="/wiki/Animal" title="Animal">animale</a>, ainsi que l'<a href="/wiki/Hydromel" title="Hydromel">hydromel</a>, fabriqué à partir du <a href="/wiki/Miel" title="Miel">miel</a>. Dans certaines cultures, le <a href="/wiki/Sang" title="Sang">sang</a> d'animaux (pur, mêlé à du lait ou à d'autres boissons) est aussi utilisé comme boisson.</li>
<li>Toutes les autres boissons sont d'origine <a href="/wiki/V%C3%A9g%C3%A9tal" title="Végétal">végétale</a>. Elles sont fabriquées à partir des <a href="/wiki/Fruit_(alimentation_humaine)" title="Fruit (alimentation humaine)">fruits</a>, des <a href="/wiki/Graine" title="Graine">graines</a>, des <a href="/wiki/Feuille" title="Feuille">feuilles</a>, de l'<a href="/wiki/%C3%89corce" title="Écorce">écorce</a>, de la <a href="/wiki/S%C3%A8ve" title="Sève">sève</a>, de diverses <a href="/wiki/Esp%C3%A8ce" title="Espèce">espèces</a> de <a href="/wiki/Plante" title="Plante">plantes</a>. Elles sont produites par divers procédés : <a href="/wiki/Pression" title="Pression">pression</a>, <a href="/wiki/Centrifugation" title="Centrifugation">centrifugation</a>, <a href="/wiki/Percolation" title="Percolation">percolation</a>, <a href="/wiki/Lixiviation" title="Lixiviation">lixiviation</a>, <a href="/wiki/Infusion" title="Infusion">infusion</a>, <a href="/wiki/D%C3%A9coction" title="Décoction">décoction</a>, <a href="/wiki/Mac%C3%A9ration_(cuisine,_parfumerie_et_pharmacop%C3%A9e)" title="Macération (cuisine, parfumerie et pharmacopée)">macération</a>, <a href="/wiki/Fermentation" title="Fermentation">fermentation</a>, <a href="/wiki/Distillation" title="Distillation">distillation</a>, <a href="/wiki/M%C3%A9lange" title="Mélange">mélange</a>...</li></ul>
<h2><span class="mw-headline" id="Production">Production</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=2" title="Modifier la section : Production">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=2" title="Modifier la section : Production">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Une boisson est une forme de <a href="/wiki/Liquide" title="Liquide">liquide</a> qui a été préparée pour la consommation humaine. La préparation peut comprendre un certain nombre <a href="/wiki/%C3%89tapes_(revue)" title="Étapes (revue)">d'étapes</a> différentes, certaines avant le <a href="/wiki/Transport" title="Transport">transport</a>, d'autres immédiatement avant la consommation
</p>
<h3><span id="Purification_de_l.27eau"></span><span class="mw-headline" id="Purification_de_l'eau">Purification de l'eau</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=3" title="Modifier la section : Purification de l'eau">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=3" title="Modifier la section : Purification de l'eau">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Avant toutes utilisations, l'eau est purifiée. Les méthodes de purification comprennent la filtration et l'ajout de produits chimiques, tels que la chloration. L'importance de l'eau purifiée est soulignée par l'Organisation mondiale de la santé et UNICEF, qui annonçaient qu'en 2017, une moyenne de 297 000 enfants âgés de 5 ans sur 3 milliards de personnes n'ont pas accès à une eau purifiée ; subissant ainsi des maladies mortelles comme la diarrhée, le choléra ou encore la typhoïde<sup class="reference" id="cite_ref-1"><a href="#cite_note-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>.
</p>
<h3><span id="St.C3.A9rilisation"></span><span class="mw-headline" id="Stérilisation">Stérilisation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=4" title="Modifier la section : Stérilisation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=4" title="Modifier la section : Stérilisation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<h3><span id="Pasteurisation_.26_st.C3.A9rilisation"></span><span class="mw-headline" id="Pasteurisation_&amp;_stérilisation">Pasteurisation &amp; stérilisation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=5" title="Modifier la section : Pasteurisation &amp; stérilisation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=5" title="Modifier la section : Pasteurisation &amp; stérilisation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Étudiée par Louis Pasteur entre 1860 et 1864, la <a href="/wiki/Pasteurisation" title="Pasteurisation">pasteurisation</a> est une méthode pour détruire par la chaleur les <a href="/wiki/Micro-organisme" title="Micro-organisme">micro-organismes</a> à l'intérieur d'un liquide, afin de le rendre acceptable à la consommation. Le liquide est d'abord chauffé à une <a href="/wiki/Temp%C3%A9rature" title="Température">température</a> comprise entre 60 et <span title="212°F ou 373,2 K">100</span> <abbr class="abbr" title="degré Celsius">°C</abbr>, pendant une période précise, puis est brutalement refroidi. Contrairement à la stérilisation, la pasteurisation peut détériorer les vertus nutritives de l'aliment, mais pas son apparence<sup class="reference" id="cite_ref-:1_2-0"><a href="#cite_note-:1-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>Ainsi, le liquide se conserve plus longtemps et communique moins de <a href="/wiki/Maladie" title="Maladie">maladies</a>. Cette technique est principalement utilisée pour le <a href="/wiki/Lait" title="Lait">lait</a> et la bière, mais aussi pour certains jus de fruits<sup class="reference" id="cite_ref-:1_2-1"><a href="#cite_note-:1-2"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>.
</p><p>La stérilisation est un procédé qui se rapproche de la pasteurisation, mais se différencie dans ses objectifs. Ceux-ci ne sont pas de limiter, mais bien de purifier un liquide, c'est-à-dire détruire tous les micro-organismes présents. Cela consiste à exposer le produit à des températures très élevées (au-delà de <span title="212°F ou 373,2 K">100</span> <abbr class="abbr" title="degré Celsius">°C</abbr>)<sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>. À l'inverse, cela permet une conservation plus longue qu'avec la pasteurisation et garantit ne pas propager de maladies.
</p>
<h3><span class="mw-headline" id="Extraction_de_jus">Extraction de jus</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=6" title="Modifier la section : Extraction de jus">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=6" title="Modifier la section : Extraction de jus">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Le processus d'extraction du jus de fruits et légumes peut prendre plusieurs formes. Un simple écrasement de la plupart des fruits fournira une quantité importante de liquide, bien qu'une pression plus intense puisse être appliquée pour obtenir le maximum de jus du fruit. Le broyage et le pressage sont des procédés utilisés dans la production de vin.
</p>
<h3><span class="mw-headline" id="Infusion">Infusion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=7" title="Modifier la section : Infusion">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=7" title="Modifier la section : Infusion">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>L'infusion est le processus d'extraction d'arômes de la matière végétale en permettant à la matière de rester en suspension dans l'eau. Ce procédé est utilisé dans la production de thés, tisanes et peut être utilisé pour préparer du café (lors de l'utilisation d'une presse à café ).
</p>
<h3><span class="mw-headline" id="Percolation">Percolation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=8" title="Modifier la section : Percolation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=8" title="Modifier la section : Percolation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Le nom est dérivé du mot « percoler » qui signifie faire passer (un solvant) à travers une substance perméable, en particulier pour extraire un constituant soluble. Dans le cas de la préparation du café, le solvant est l'eau, la substance perméable est le marc de café et les constituants solubles sont les composés chimiques qui donnent au café sa couleur, son goût, son arôme et ses propriétés stimulantes.
</p>
<h3><span class="mw-headline" id="Fermentation">Fermentation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=9" title="Modifier la section : Fermentation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=9" title="Modifier la section : Fermentation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La <a href="/wiki/Fermentation" title="Fermentation">fermentation</a> est un processus métabolique qui convertit le sucre en éthanol. La fermentation a été utilisée par les humains pour la production de boissons depuis le néolithique. En vinification, le jus de raisin est associé à de la levure dans un environnement anaérobie pour permettre la fermentation. La quantité de sucre dans le <a href="/wiki/Vin" title="Vin">vin</a> et la durée donnée pour la fermentation déterminent le niveau d'<a href="/wiki/Boisson_alcoolis%C3%A9e" title="Boisson alcoolisée">alcool</a> et la douceur du vin.
</p><p>Lors du <a href="/wiki/Brassage" title="Brassage">brassage</a> de la bière, il y a quatre ingrédients principaux: l'eau, les céréales, la levure et le houblon. Le grain est encouragé à germer par trempage et séchage à la chaleur, un processus connu sous le nom de maltage. Il est ensuite broyé avant d'être à nouveau trempé pour créer les sucres nécessaires à la fermentation. Ce processus est connu sous le nom de brassage. Le houblon est ajouté pour aromatiser, puis la levure est ajoutée au mélange (maintenant appelé moût ) pour démarrer le processus de fermentation.
</p>
<h3><span class="mw-headline" id="Distillation">Distillation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=10" title="Modifier la section : Distillation">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=10" title="Modifier la section : Distillation">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La distillation est une méthode de séparation des mélanges basée sur les différences de volatilité des composants dans un mélange liquide bouillant. C'est l'une des méthodes utilisées pour la purification de <a href="/wiki/Eau" title="Eau">l'eau</a>. C'est aussi une méthode de production de spiritueux à partir de boissons alcoolisées plus douces.
</p>
<h3><span id="M.C3.A9lange"></span><span class="mw-headline" id="Mélange">Mélange</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=11" title="Modifier la section : Mélange">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=11" title="Modifier la section : Mélange">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Une <a href="/wiki/Boisson_alcoolis%C3%A9e" title="Boisson alcoolisée">boisson alcoolisée</a> mélangée contenant deux <a class="mw-redirect" href="/wiki/Ingr%C3%A9dients" title="Ingrédients">ingrédients</a> ou plus est appelée <a href="/wiki/Cocktail" title="Cocktail">cocktail</a>. Les cocktails étaient à l'origine un mélange d'alcools, de <a href="/wiki/Sucre" title="Sucre">sucre</a>, d'eau et d' amers. Le terme est maintenant souvent utilisé pour presque toutes les boissons mélangées qui contiennent de l'alcool, y compris les mélangeurs, les cocktails, etc. Un cocktail aujourd'hui contient généralement un ou plusieurs types d'alcool et un ou plusieurs mélangeurs, tels que la soude ou jus de fruits. Les ingrédients supplémentaires peuvent être du sucre, du <a href="/wiki/Miel" title="Miel">miel</a>, du <a href="/wiki/Lait" title="Lait">lait</a>, de la <a href="/wiki/Cr%C3%A8me_(produit_laitier)" title="Crème (produit laitier)">crème</a> et diverses herbes.
</p>
<h2><span class="mw-headline" id="Types_de_boissons">Types de boissons</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=12" title="Modifier la section : Types de boissons">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=12" title="Modifier la section : Types de boissons">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="Boissons_non_alcoolis.C3.A9es"></span><span class="mw-headline" id="Boissons_non_alcoolisées">Boissons non alcoolisées</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=13" title="Modifier la section : Boissons non alcoolisées">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=13" title="Modifier la section : Boissons non alcoolisées">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Une boisson non alcoolisée est une boisson qui ne contient pas d' <a href="/wiki/Boisson_alcoolis%C3%A9e" title="Boisson alcoolisée">alcool</a>. Les boissons telles que les <a class="mw-redirect" href="/wiki/Boissons_gazeuses" title="Boissons gazeuses">boissons gazeuses</a>, l'eau gazeuse, les <a href="/wiki/Soda" title="Soda">sodas</a>, le thé glacé, la <a href="/wiki/Limonade" title="Limonade">limonade</a>, la bière de racine, le <a href="/wiki/Lait" title="Lait">lait</a>, le <a href="/wiki/Chocolat" title="Chocolat">chocolat</a> chaud, le thé, le café, les <a href="/wiki/Milk-shake" title="Milk-shake">milk-shakes</a>, <a href="/wiki/Eau" title="Eau">l'eau</a> plate et les <a href="/wiki/Boisson_%C3%A9nergisante" title="Boisson énergisante">boissons énergisantes</a> ne contiennent pas d'alcool.
</p>
<h4><span class="mw-headline" id="Eau">Eau</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=14" title="Modifier la section : Eau">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=14" title="Modifier la section : Eau">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>L'<a href="/wiki/Eau" title="Eau">eau</a> est essentielle à l'existence des êtres vivants, animaux, plantes ou microorganismes. Elle est aussi le principal constituant de toutes les boissons et l'ingrédient principal de la plupart. Ainsi, elle est la boisson la plus consommée au monde<sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>. Cependant, 97% de l'eau sur Terre est de l'eau salée non potable<sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>5<span class="cite_crochet">]</span></a></sup>. L'eau douce se trouve dans les <a href="/wiki/Rivi%C3%A8re" title="Rivière">rivières</a>, les lacs, les zones humides, les eaux souterraines et les glaciers gelés. Moins de 1% des approvisionnements en eau douce de la planète sont accessibles par les eaux de surface et les sources souterraines qui sont rentables à récupérer<sup class="reference" id="cite_ref-6"><a href="#cite_note-6"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.
</p><p>Dans les cultures occidentales, l'eau est principalement bue froide. Dans la culture chinoise, elle est généralement bue chaude.
</p>
<h4><span class="mw-headline" id="Lait">Lait</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=15" title="Modifier la section : Lait">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=15" title="Modifier la section : Lait">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Considéré comme l'une des premières boissons existants dans la nature, le lait est la principale source de nutrition des bébés<sup class="reference" id="cite_ref-7"><a href="#cite_note-7"><span class="cite_crochet">[</span>7<span class="cite_crochet">]</span></a></sup>. Elle est la dixième boisson la plus consommée au monde<sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>8<span class="cite_crochet">]</span></a></sup>.
</p><p>Dans de nombreuses cultures, en particulier dans le monde occidental, les humains continuent de consommer du lait laitier au-delà de la petite enfance, en utilisant le lait d'autres animaux (en particulier les bovins, les chèvres et les moutons). Le <a href="/wiki/Lait" title="Lait">lait</a> végétal, terme général désignant tout <a href="/wiki/Produit_laitier" title="Produit laitier">produit laitier</a> dérivé d'une source végétale, est également consommé depuis longtemps dans divers pays et cultures. Les variétés populaires sont la plupart au niveau international du lait de soja, lait d'amande, lait de riz et de lait de coco .
</p>
<h4><span id="Jus_et_boissons_.C3.A0_base_de_jus"></span><span class="mw-headline" id="Jus_et_boissons_à_base_de_jus">Jus et boissons à base de jus</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=16" title="Modifier la section : Jus et boissons à base de jus">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=16" title="Modifier la section : Jus et boissons à base de jus">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Le jus de <a href="/wiki/Fruit_(alimentation_humaine)" title="Fruit (alimentation humaine)">fruit</a> est un produit naturel qui contient peu ou pas d'additifs, ils sont généralement formés à partir de fruit et d'eau. Les jus de fruits sont très bons pour la santé mais seulement s'ils sont produits naturellement, c'est-à-dire de fruits et d'eau. Mais aujourd'hui les jus de fruits sont plutôt faits à partir de maints ingrédients y compris de produits chimiques (arôme, additifs...).
</p><p>Les produits d'agrumes tels que le jus d'orange et de mandarine sont des boissons familières pour le <a class="mw-redirect" href="/wiki/Petit-d%C3%A9jeuner" title="Petit-déjeuner">petit-déjeuner</a>, tandis que le <a href="/wiki/Jus_de_pamplemousse" title="Jus de pamplemousse">jus de pamplemousse</a> , l'<a href="/wiki/Ananas" title="Ananas">ananas</a>, la <a href="/wiki/Pomme" title="Pomme">pomme</a>, le <a href="/wiki/Raisin" title="Raisin">raisin</a>, le <a class="mw-redirect" href="/wiki/Citron_vert" title="Citron vert">citron vert</a> et le <a href="/wiki/Jus_de_citron" title="Jus de citron">jus de citron</a> sont également courants. L'eau de coco est un jus hautement nutritif et rafraîchissant. De nombreux types de baies sont écrasés ; leurs jus sont mélangés à de l'eau et parfois sucrés. La <a href="/wiki/Framboise" title="Framboise">framboise</a>, la mûre et les <a href="/wiki/Raisins" title="Raisins">raisins</a> de Corinthe sont des boissons à base de jus populaires, mais le pourcentage d'eau détermine également leur valeur nutritive. Le jus de raisin laissé fermenter produit du <a href="/wiki/Vin" title="Vin">vin</a> .
</p><p>Les <a href="/wiki/Fruit_(alimentation_humaine)" title="Fruit (alimentation humaine)">fruits</a> sont très périssables, donc la capacité d'extraire les jus et de les stocker était d'une grande valeur. Certains fruits sont très acides et les mélanger avec de l'eau et des sucres ou du miel était souvent nécessaire pour les rendre savoureux. Le stockage précoce des jus de fruits exigeait beaucoup de travail, exigeant le broyage des fruits et le mélange des jus purs résultants avec des sucres avant la mise en <a href="/wiki/Bouteille" title="Bouteille">bouteille</a>.
</p><p>Les jus de légumes sont généralement servis chauds ou froids. Différents types de légumes peuvent être utilisés pour faire du jus de légumes tels que les carottes , les tomates, les concombres , le céleri et bien d'autres. Certains jus de légumes sont mélangés avec du jus de fruits pour améliorer le goût du jus de légumes. De nombreux jus de légumes populaires, en particulier ceux à forte teneur en tomates, sont riches en sodium et leur consommation pour la santé doit donc être soigneusement considérée. Certains jus de légumes offrent les mêmes avantages pour la santé que les légumes entiers en termes de réduction des risques de maladies cardiovasculaires et de <a href="/wiki/Cancer" title="Cancer">cancer</a>.
</p><p>Néanmoins, la surconsommation de boissons sucrées, même d'origines naturelles, augmente les risques de cancers<sup class="reference" id="cite_ref-:0_9-0"><a href="#cite_note-:0-9"><span class="cite_crochet">[</span>9<span class="cite_crochet">]</span></a></sup>.
</p>
<h4><span class="mw-headline" id="Boissons_de_sommeil">Boissons de sommeil</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=17" title="Modifier la section : Boissons de sommeil">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=17" title="Modifier la section : Boissons de sommeil">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p><span class="need_ref" style="cursor:help;" title="Ce passage nécessite une référence.">Souvent, un dernier verre est une boisson prise peu de temps avant le coucher pour induire le <a href="/wiki/Sommeil" title="Sommeil">sommeil</a>. Par exemple, une petite boisson alcoolisée pour les ivrognes ou une tasse de lait chaud pour les gens sensés peut soi-disant favoriser une bonne nuit de sommeil. Aujourd'hui, la plupart des bonnets de nuit et des boissons relaxantes sont généralement des boissons non alcoolisées contenant des ingrédients apaisants. Ils sont considérés comme des boissons qui servent à détendre une personne. Contrairement à d'autres boissons apaisantes, telles que le thé, le lait chaud ou mais certaines ont prétendu contenir lait avec du miel ; les boissons de relaxation contiennent presque universellement plus d'un ingrédient actif. Il est connu que les boissons relaxantes contiennent d'autres ingrédients naturels et sont généralement exemptes de caféine et alcool, marijuana.</span><sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Aide:R%C3%A9f%C3%A9rence_n%C3%A9cessaire" title="Aide:Référence nécessaire">[réf. nécessaire]</a></sup>
</p>
<h4><span class="mw-headline" id="Sodas">Sodas</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=18" title="Modifier la section : Sodas">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=18" title="Modifier la section : Sodas">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Les sodas sont des boissons gazéifiées, non-alcoolisées, aromatisées et la plupart du temps sucrées. Souvent associés à des éléments synthétiques (<a href="/wiki/Ar%C3%B4me" title="Arôme">arômes</a>, <a href="/wiki/Additif" title="Additif">additifs</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10"><span class="cite_crochet">[</span>10<span class="cite_crochet">]</span></a></sup>, <a href="/wiki/%C3%89dulcorant" title="Édulcorant">édulcorants</a>, <a href="/wiki/Aspartame" title="Aspartame">aspartame</a>), le taux élevé de sucres ajoutés est fréquemment sources de mise en garde par les professionnels de la santé. Ce taux, lors d'une consommation régulière, est la cause de détérioration de la santé dentaire<sup class="reference" id="cite_ref-11"><a href="#cite_note-11"><span class="cite_crochet">[</span>11<span class="cite_crochet">]</span></a></sup>, de diabète, d'obésité. Des études démontrent que l'excès de sodas augmenteraient aussi les risques de cancers<sup class="reference" id="cite_ref-12"><a href="#cite_note-12"><span class="cite_crochet">[</span>12<span class="cite_crochet">]</span></a></sup><sup class="reference cite_virgule">,</sup><sup class="reference" id="cite_ref-:0_9-1"><a href="#cite_note-:0-9"><span class="cite_crochet">[</span>9<span class="cite_crochet">]</span></a></sup>.
</p><p>En 2016, le Royaume-Uni essaie de contrôler la surconsommation de boissons sucrées de type soda au travers de taxes<sup class="reference" id="cite_ref-13"><a href="#cite_note-13"><span class="cite_crochet">[</span>13<span class="cite_crochet">]</span></a></sup>. Dans le même élan, la France instaura une taxe sur les sodas en 2017, ainsi qu'une interdiction des fontaines en libre-service de sodas dans les lieux de restaurations<sup class="reference" id="cite_ref-14"><a href="#cite_note-14"><span class="cite_crochet">[</span>14<span class="cite_crochet">]</span></a></sup>.
</p><p>Il est recommandé de boire moins d'un verre de petite taille de soda par jour. Cela permet de limiter sa consommation de sucres et d'éléments chimiques, donc de rester en bonne santé. De plus, cela limite aussi la production de déchets (emballages, canettes, bouteilles en plastiques) qui finissent très souvent dans la nature<sup class="reference" id="cite_ref-15"><a href="#cite_note-15"><span class="cite_crochet">[</span>15<span class="cite_crochet">]</span></a></sup>.
</p>
<h4><span class="mw-headline" id="Autres_boissons_sans_alcool">Autres boissons sans alcool</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=19" title="Modifier la section : Autres boissons sans alcool">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=19" title="Modifier la section : Autres boissons sans alcool">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>D'autres boissons ne contiennents pas d'alcool: 
</p>
<dl><dt><a href="/wiki/Mocktail" title="Mocktail">mocktail</a></dt>
<dd>contraction des mots anglais mock ("simulé" en français) et cocktail,le mocktail, élaboré avec fruits, jus ou boissons gazeuses, mélange des boissons sans alcool.</dd></dl>
<p>L'autre nom ets <i>spirit free</i> ou <i>virgin</i><sup class="reference" id="cite_ref-16"><a href="#cite_note-16"><span class="cite_crochet">[</span>16<span class="cite_crochet">]</span></a></sup>.
</p><p>De même, la <a href="/wiki/Bi%C3%A8re_sans_alcool" title="Bière sans alcool">bière sans alcool</a> et le <a href="/wiki/Vin_sans_alcool" title="Vin sans alcool">vin sans alcool</a> ne contiennent pas d'alcool.
</p>
<h3><span id="Boissons_alcoolis.C3.A9es"></span><span class="mw-headline" id="Boissons_alcoolisées">Boissons alcoolisées</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=20" title="Modifier la section : Boissons alcoolisées">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=20" title="Modifier la section : Boissons alcoolisées">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Une boisson est considérée comme « alcoolisée » si elle contient de l'éthanol, communément appelé alcool<sup class="reference" id="cite_ref-17"><a href="#cite_note-17"><span class="cite_crochet">[</span>17<span class="cite_crochet">]</span></a></sup> (bien qu'en chimie la définition d'« alcool » englobe de nombreux autres composés). Dans de nombreux <a href="/wiki/Pays" title="Pays">pays</a>, boire des <a class="mw-redirect" href="/wiki/Boissons_alcoolis%C3%A9es" title="Boissons alcoolisées">boissons alcoolisées</a> dans un <a href="/wiki/Bar_(%C3%A9tablissement)" title="Bar (établissement)">bar</a> ou individuellement est une tradition culturelle<sup class="reference" id="cite_ref-18"><a href="#cite_note-18"><span class="cite_crochet">[</span>18<span class="cite_crochet">]</span></a></sup> mais dans d'autres pays cela est interdit<sup class="reference" id="cite_ref-19"><a href="#cite_note-19"><span class="cite_crochet">[</span>19<span class="cite_crochet">]</span></a></sup>. Du point de vue des <a href="/wiki/Religion" title="Religion">religions</a>, chez le christianisme boire de <a href="/wiki/Boisson_alcoolis%C3%A9e" title="Boisson alcoolisée">l'alcool</a> est un <span class="need_ref" style="cursor:help;" title="Une source est souhaitée pour ce passage.">sujet très controversé</span><sup class="need_ref_tag" style="padding-left:2px;"><a href="/wiki/Aide:R%C3%A9f%C3%A9rence_n%C3%A9cessaire" title="Aide:Référence nécessaire">[<abbr class="abbr" title="référence">réf.</abbr> souhaitée]</a></sup> puis ce que des groupes l'interdisent<sup class="reference" id="cite_ref-20"><a href="#cite_note-20"><span class="cite_crochet">[</span>20<span class="cite_crochet">]</span></a></sup> alors que d'autres l'autorise<sup class="reference" id="cite_ref-21"><a href="#cite_note-21"><span class="cite_crochet">[</span>21<span class="cite_crochet">]</span></a></sup>. En islam cela est considéré comme un péché capital<sup class="reference" id="cite_ref-22"><a href="#cite_note-22"><span class="cite_crochet">[</span>22<span class="cite_crochet">]</span></a></sup>, le bouddhisme l'interdit également<sup class="reference" id="cite_ref-23"><a href="#cite_note-23"><span class="cite_crochet">[</span>23<span class="cite_crochet">]</span></a></sup>, le judaïsme l'autorise<sup class="reference" id="cite_ref-24"><a href="#cite_note-24"><span class="cite_crochet">[</span>24<span class="cite_crochet">]</span></a></sup>.
</p>
<h4><span id="Bi.C3.A8re"></span><span class="mw-headline" id="Bière">Bière</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=21" title="Modifier la section : Bière">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=21" title="Modifier la section : Bière">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>La bière est une boisson alcoolisée produite par la saccharification de l'amidon et la fermentation du sucre résultant. L'amidon et les enzymes de saccharification sont souvent dérivés de céréales, le plus souvent d'orge maltée et de blé malté. La plupart des bières sont également aromatisées avec du houblon, qui ajoute de l'amertume et agit comme un conservateur naturel, bien que d'autres arômes tels que des herbes ou des fruits puissent parfois être inclus. La préparation de la bière s'appelle le brassage. La bière est la boisson alcoolisée la plus consommée au monde, et est la troisième boisson la plus populaire, après l'eau et le thé. Il aurait été découvert par la déesse Ninkasi vers 5300 avant notre ère, lorsqu'elle a accidentellement découvert de la levure après avoir laissé des céréales dans des bocaux qui ont ensuite été plu et laissés pendant plusieurs jours. Les femmes ont été les principales créatrices de la bière à travers l'histoire en raison de son association avec la domesticité et, tout au long de l'histoire, brassée à la maison pour la consommation familiale. Ce n'est que dans l'histoire récente que les hommes ont commencé à s'essayer au terrain. Certains pensent que c'est la boisson fermentée la plus ancienne.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25"><span class="cite_crochet">[</span>25<span class="cite_crochet">]</span></a></sup>
</p>
<h4><span class="mw-headline" id="Cidre">Cidre</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=22" title="Modifier la section : Cidre">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=22" title="Modifier la section : Cidre">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Le cidre est une boisson alcoolisée fermentée à base de jus de fruits, le plus souvent et traditionnellement de jus de pomme, mais aussi de jus de pêches, de poires (cidre « Perry ») ou d'autres fruits. Le cidre peut être fabriqué à partir de n'importe quelle variété de pomme, mais certains cultivars cultivés uniquement pour le cidre sont connus sous le nom de pommes à cidre<sup class="reference" id="cite_ref-26"><a href="#cite_note-26"><span class="cite_crochet">[</span>26<span class="cite_crochet">]</span></a></sup>. Le Royaume-Uni a la consommation de cidre par habitant la plus élevée, ainsi que les plus grandes entreprises productrices de cidre au monde<sup class="reference" id="cite_ref-27"><a href="#cite_note-27"><span class="cite_crochet">[</span>27<span class="cite_crochet">]</span></a></sup>, Depuis 2006, le Royaume-Uni produit 600 millions de litres de cidre chaque année (130 millions de gallons impériaux).
</p>
<h4><span class="mw-headline" id="Vin">Vin</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=23" title="Modifier la section : Vin">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=23" title="Modifier la section : Vin">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Le vin est une boisson alcoolisée à base de raisins fermentés ou d'autres fruits. L'équilibre chimique naturel des raisins leur permet de fermenter sans ajout de sucres, d' acides, d'enzymes, d'eau ou d'autres nutriments. La levure consomme les sucres des raisins et les convertit en alcool et en dioxyde de carbone. Différentes variétés de raisins et souches de levures produisent différents styles de vin. Les variations résultent des interactions très complexes entre le développement biochimique du fruit, les réactions impliquées dans la fermentation et terroir, ainsi que l'intervention humaine dans le processus global. Le produit final peut contenir des dizaines de milliers de composés chimiques en quantités variant de quelques pour cent à quelques parties par milliard.
</p><p>Vins fabriqués à partir de produits en plus de raisins sont habituellement nommés d'après le produit à partir duquel elles sont produites (par exemple, le vin de riz , grenade vin, vin de pomme et vin de sureau) et sont appelés génériquement vin de fruits . Le terme « vin » peut également faire référence à des boissons fermentées à l'amidon ou enrichies ayant une teneur en alcool plus élevée, telles que le vin d'orge, le huangjiu ou le saké.
</p><p>Le vin a une riche histoire remontant à des milliers d'années, la production la plus ancienne découverte à ce jour ayant eu lieu autour de 6000  avant JC en Géorgie. Il avait atteint les Balkans autour de 4500 avant JC et a été consommé et célébré dans la Grèce antique et à Rome.
</p>
<h4><span class="mw-headline" id="Esprits">Esprits</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=24" title="Modifier la section : Esprits">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=24" title="Modifier la section : Esprits">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Les spiritueux sont des boissons distillées sans sucre ajouté et contenant au moins 20% d' alcool par volume (ABV). Les spiritueux populaires incluent le borovička, le brandy, le gin, le rhum, le slivovitz, la tequila, la vodka et le whisky . Le brandy est un spiritueux créé en distillant du vin, tandis que la vodka peut être distillée à partir de toute matière végétale riche en amidon ou en sucre ; la plupart de la vodka aujourd'hui est produite à partir de céréales telles que le sorgho ou le maïs.
</p>
<h3><span class="mw-headline" id="Boissons_chaudes">Boissons chaudes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=25" title="Modifier la section : Boissons chaudes">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=25" title="Modifier la section : Boissons chaudes">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span id="Caf.C3.A9"></span><span class="mw-headline" id="Café">Café</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=26" title="Modifier la section : Café">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=26" title="Modifier la section : Café">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Le café est une boisson brassée préparée à partir des graines torréfiées de plusieurs espèces d'un arbuste à feuilles persistantes du genre Coffea . Les deux sources les plus courantes de grains de café sont le très apprécié Coffea arabica et la forme « robusta » du Coffea canephora, plus résistant. Les plants de café sont cultivés dans plus de 70 pays . Une fois mûres, les « baies » de café sont cueillies, transformées et séchées pour donner les graines à l'intérieur. Les graines sont ensuite torréfiées à des degrés divers, selon la saveur désirée, avant d'être moulues et brassées pour créer du café.
</p><p>Le café est légèrement acide (pH 5,0–5,1 ) et peut avoir un effet stimulant sur les humains en raison de sa teneur en caféine . C'est l'une des boissons les plus populaires au monde. Il peut être préparé et présenté de diverses manières. L'effet du café sur la santé humaine a fait l'objet de nombreuses études; cependant, les résultats ont varié en termes de bénéfice relatif du café.
</p><p>La culture du café a d'abord eu lieu dans le sud de l' Arabie ;  la première preuve crédible de consommation de café apparaît au milieu du <abbr class="abbr" title="15ᵉ siècle"><span class="romain">XV</span><sup style="font-size:72%">e</sup></abbr> siècle dans les sanctuaires soufis du Yémen .
</p>
<h4><span class="mw-headline" id="Chocolat_chaud">Chocolat chaud</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=27" title="Modifier la section : Chocolat chaud">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=27" title="Modifier la section : Chocolat chaud">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Le chocolat chaud, également connu sous le nom de chocolat à boire ou de cacao, est une boisson chauffée composée de chocolat rasé, de chocolat fondu ou de poudre de cacao , de lait ou d'eau chauffé et généralement un édulcorant. Le chocolat chaud peut être garni de crème fouettée. Le chocolat chaud à base de chocolat fondu est parfois appelé chocolat à boire, caractérisé par moins de douceur et une consistance plus épaisse.
</p>
<h2><span class="mw-headline" id="Composition_et_consommation_des_boissons">Composition et consommation des boissons</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Boisson&amp;veaction=edit&amp;section=28" title="Modifier la section : Composition et consommation des boissons">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Boisson&amp;action=edit&amp;section=28" title="Modifier la section : Composition et consommation des boissons">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>L'eau, la première boisson, est un liquide qui n'apporte aucun <a href="/wiki/Nutriment" title="Nutriment">élément nutritif</a>, et n'a aucun contenu <a href="/wiki/%C3%89nergie_chimique" title="Énergie chimique">énergétique</a>. Elle peut toutefois contenir des <a href="/wiki/Sel_(chimie)" title="Sel (chimie)">sels minéraux</a>. Les autres boissons peuvent apporter en outre des <a href="/wiki/Calorie" title="Calorie">calories</a> sous forme de <a href="/wiki/Sucre" title="Sucre">sucre</a>, d'<a href="/wiki/%C3%89thanol" title="Éthanol">alcool</a>..., des <a href="/wiki/Prot%C3%A9ine" title="Protéine">protéines</a>, des <a href="/wiki/Lipide" title="Lipide">matières grasses</a>.
</p><p>À noter : tous les liquides alimentaires ne sont pas considérés comme des boissons, certains sont des <a class="mw-redirect" href="/wiki/Aliment" title="Aliment">aliments</a>, tels que les <a href="/wiki/Huile_alimentaire" title="Huile alimentaire">huiles</a> et le <a href="/wiki/Vinaigre" title="Vinaigre">vinaigre</a>, qui sont utilisées en <a href="/wiki/Cuisine" title="Cuisine">cuisine</a>, les <a href="/wiki/Bouillon_(cuisine)" title="Bouillon (cuisine)">bouillons</a>, les <a href="/wiki/Soupe" title="Soupe">soupes</a> et <a href="/wiki/Potage" title="Potage">potages</a>... Certaines boissons, très riches comme le <a href="/wiki/Chocolat" title="Chocolat">chocolat</a>, sont à la frontière entre boisson et aliment. D'autres sont à la limite du <a href="/wiki/M%C3%A9dicament" title="Médicament">médicament</a>, notamment parmi les infusions. Certaines <a href="/wiki/Plante" title="Plante">plantes</a> qui servent à les préparer sont classées parmi les <a href="/wiki/Plante_m%C3%A9dicinale" title="Plante médicinale">plantes médicinales</a>.
</p><p>Les boissons se boivent généralement dans des <a href="/wiki/Verre" title="Verre">verres</a>, parfois dans des <a href="/wiki/Bol_(vaisselle)" title="Bol (vaisselle)">bols</a>, ou dans toutes sortes de <a href="/wiki/R%C3%A9cipient" title="Récipient">récipients</a> servant aussi à leur conditionnement : <a href="/wiki/Bouteille" title="Bouteille">bouteilles</a>, <a class="mw-redirect" href="/wiki/Canette_%C3%A0_boisson" title="Canette à boisson">canettes</a>, <a href="/wiki/Gourde_(r%C3%A9cipient)" title="Gourde (récipient)">gourdes</a>, <a href="/wiki/Flasque" title="Flasque">flasques</a>...
</p><p>Elle se prennent soit à <a href="/wiki/Table" title="Table">table</a> au cours des <a href="/wiki/Repas" title="Repas">repas</a>, dont elles peuvent constituer l'élément essentiel (comme le café ou <a href="/wiki/Caf%C3%A9_au_lait" title="Café au lait">café au lait</a> dans le <a href="/wiki/D%C3%A9jeuner" title="Déjeuner">déjeuner</a>), soit avant (<a href="/wiki/Ap%C3%A9ritif" title="Apéritif">apéritifs</a>) ou après (<a href="/wiki/Digestif" title="Digestif">digestifs</a>) les repas, soit entre les <a href="/wiki/Repas" title="Repas">repas</a> (boissons rafraîchissantes, café, thé, remontants, justifiant des moments de pause particuliers, comme la pause café, le goûter, le <i>tea break</i> des <a href="/wiki/Royaume-Uni" title="Royaume-Uni">Britanniques</a>.
</p><p>Certaines boissons se consomment chaudes, d'autres froides, voire fraîches ou « frappées ». La <a href="/wiki/Temp%C3%A9rature" title="Température">température</a> de consommation est un élément important en particulier pour les <a href="/wiki/Vin" title="Vin">vins</a>.
</p>
