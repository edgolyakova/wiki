<p>En métallurgie, un <b>alliage</b> est un mélange de plusieurs <a href="/wiki/%C3%89l%C3%A9ment_chimique" title="Élément chimique">éléments chimiques</a>, dont le principal constituant est un <a href="/wiki/M%C3%A9tal" title="Métal">métal</a>, et dont les caractéristiques sont celles d'un <a href="/wiki/M%C3%A9tal#Liaison_métallique_et_structures_cristallines_des_métaux" title="Métal">matériau métallique</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1"><span class="cite_crochet">[</span>note 1<span class="cite_crochet">]</span></a></sup> .
</p><p>Les caractéristiques mécaniques des métaux purs sont la plupart du temps relativement faibles. Le fait d'ajouter d'autres éléments permet de « durcir » le métal en augmentant ses caractéristiques mécaniques<sup class="reference" id="cite_ref-2"><a href="#cite_note-2"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>. Outre les renforcements mécaniques engendrés par déformation, tel que l'<a href="/wiki/%C3%89crouissage" title="Écrouissage">écrouissage</a>, il existe des durcissements chimiques par addition d'éléments en <a href="/wiki/Solution_solide" title="Solution solide">solution solide</a> ou par précipitation de phases secondaires durcissantes telles que les <a href="/wiki/Carbure" title="Carbure">carbures</a>. Ces ajouts permettent également de modifier les caractéristiques chimiques, telle que la résistance à la <a href="/wiki/Corrosion" title="Corrosion">corrosion</a>, ou d'améliorer d'autres caractéristiques, par exemple la <a href="/wiki/Coulabilit%C3%A9" title="Coulabilité">coulabilité</a>.
</p><p>Dans un alliage, l'élément métallique majoritaire, c'est-à-dire constituant la plus importante partie du mélange, est appelé « métal de base » ou « base ». Les éléments ajoutés volontairement sont appelés « éléments d'alliage » ou « éléments d'addition » et les éléments non désirés sont appelés « impuretés ».
</p><p>Les éléments d'alliage sont le plus souvent des métaux, mais peuvent également être d'autres éléments chimiques tels que le <a href="/wiki/Carbone" title="Carbone">carbone</a> (dans l'<a href="/wiki/Acier" title="Acier">acier</a> ou la <a href="/wiki/Fonte_(m%C3%A9tallurgie)" title="Fonte (métallurgie)">fonte</a>) ou le <a href="/wiki/Silicium" title="Silicium">silicium</a> (dans l'<a href="/wiki/Aluminium" title="Aluminium">aluminium</a>).
</p><p>Quand l'élément d'alliage n'est pas un métal, sa teneur reste généralement faible (quelques % massique au maximum). Ainsi, dans un acier la concentration en carbone est inférieure à 2 % massique (inférieure à 7 % massique dans le cas de la fonte), alors qu'il est possible de faire un alliage <a href="/wiki/Cuivre" title="Cuivre">cuivre</a>-<a href="/wiki/Zinc" title="Zinc">zinc</a> (communément appelé <a href="/wiki/Laiton" title="Laiton">laiton</a>) avec 50 % de chacun des éléments.
</p>

<p>Il existe également des alliages dits naturels, par exemple l'<a href="/wiki/%C3%89lectrum" title="Électrum">électrum</a>, alliage d'<a href="/wiki/Or" title="Or">or</a> et d'<a href="/wiki/Argent" title="Argent">argent</a> natifs utilisé dans la <a href="/wiki/Pr%C3%A9histoire" title="Préhistoire">Préhistoire</a> et l'<a href="/wiki/Antiquit%C3%A9" title="Antiquité">Antiquité</a> : <a href="/wiki/N%C3%A9cropole_de_Varna" title="Nécropole de Varna">Varna</a>, <a href="/wiki/Anatolie" title="Anatolie">Asie Mineure</a>, <a href="/wiki/Ur_(M%C3%A9sopotamie)" title="Ur (Mésopotamie)">Ur</a>, <a href="/wiki/%C3%89gypte" title="Égypte">Égypte</a>, entre autres. Au <a href="/wiki/IIIe_si%C3%A8cle_av._J.-C." title="IIIe siècle av. J.-C."><abbr class="abbr" title="3ᵉ siècle"><span class="romain">III</span><sup style="font-size:72%">e</sup></abbr> siècle <abbr class="abbr nowrap" title="avant Jésus-Christ">av. J.-C.</abbr></a>, dans un ouvrage <i><a class="new" href="/w/index.php?title=Sur_les_Pierres&amp;action=edit&amp;redlink=1" title="Sur les Pierres (page inexistante)">Sur les Pierres</a></i><sup class="reference" id="cite_ref-3"><a href="#cite_note-3"><span class="cite_crochet">[</span>2<span class="cite_crochet">]</span></a></sup>, le philosophe <a href="/wiki/Th%C3%A9ophraste" title="Théophraste">Théophraste</a> explique le <a href="/wiki/Titre_(m%C3%A9tal)" title="Titre (métal)">titre</a> d’un alliage est déterminé en <i>collybos</i><sup class="reference" id="cite_ref-4"><a href="#cite_note-4"><span class="cite_crochet">[</span>3<span class="cite_crochet">]</span></a></sup>, drachmes ou demi-drachme - ou encore en grains d'<a href="/wiki/Orge_commune" title="Orge commune">orge</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5"><span class="cite_crochet">[</span>4<span class="cite_crochet">]</span></a></sup>.
</p>
<h2><span class="mw-headline" id="Alliage_binaire">Alliage binaire</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=2" title="Modifier la section : Alliage binaire">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=2" title="Modifier la section : Alliage binaire">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="Alliage_binaire_.C3.A0_une_seule_phase"></span><span class="mw-headline" id="Alliage_binaire_à_une_seule_phase">Alliage binaire à une seule phase</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=3" title="Modifier la section : Alliage binaire à une seule phase">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=3" title="Modifier la section : Alliage binaire à une seule phase">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Un alliage homogène est constitué d'une seule <a href="/wiki/Phase_(thermodynamique)" title="Phase (thermodynamique)">phase</a> solide homogène. Pour obtenir un alliage homogène, il faut qu'il y ait <a href="/wiki/Miscibilit%C3%A9" title="Miscibilité">miscibilité</a> totale entre les éléments d'alliage. Il y a deux possibilités :
</p>
<ol><li>Les deux éléments d'alliage sont solubles l'un dans l'autre quelles que soient leurs proportions respectives.</li>
<li>La concentration de l'élément d'alliage est inférieure à la limite de solubilité.</li></ol>
<p>Les <a href="/wiki/R%C3%A8gles_de_Hume-Rothery" title="Règles de Hume-Rothery">règles de Hume-Rothery</a> indiquent les conditions pour obtenir un alliage homogène avec solubilité totale à l'état solide.
</p>
<div class="thumb tright"><div class="thumbinner" style="width:232px;"><a class="image" href="/wiki/Fichier:Diagramme_phase_Bi_Sb.svg"><img alt="" class="thumbimage" data-file-height="500" data-file-width="740" decoding="async" height="155" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1c/Diagramme_phase_Bi_Sb.svg/230px-Diagramme_phase_Bi_Sb.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1c/Diagramme_phase_Bi_Sb.svg/345px-Diagramme_phase_Bi_Sb.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1c/Diagramme_phase_Bi_Sb.svg/460px-Diagramme_phase_Bi_Sb.svg.png 2x" width="230"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Diagramme_phase_Bi_Sb.svg" title="Agrandir"></a></div>Diagramme de phases Bi-Sb.</div></div></div>

<p>Le <a href="/wiki/Bismuth" title="Bismuth">bismuth</a> et l'<a href="/wiki/Antimoine" title="Antimoine">antimoine</a> sont totalement solubles l'un dans l'autre. Ils forment donc une solution solide quelles que soient la composition chimique et la température (à la condition de rester dans l'état solide c'est-à-dire en dessous de la température de <a href="/wiki/Solidus_(diagramme_de_phases)" title="Solidus (diagramme de phases)">solidus</a>). Le <a href="/wiki/Diagramme_de_phase" title="Diagramme de phase">diagramme de phase</a> qui en résulte est dans ce cas un diagramme à un fuseau.
</p>
<div class="thumb tright"><div class="thumbinner" style="width:232px;"><a class="image" href="/wiki/Fichier:Diagramme_phase_Cu_Ni.svg"><img alt="" class="thumbimage" data-file-height="500" data-file-width="740" decoding="async" height="155" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/32/Diagramme_phase_Cu_Ni.svg/230px-Diagramme_phase_Cu_Ni.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/32/Diagramme_phase_Cu_Ni.svg/345px-Diagramme_phase_Cu_Ni.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/32/Diagramme_phase_Cu_Ni.svg/460px-Diagramme_phase_Cu_Ni.svg.png 2x" width="230"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Diagramme_phase_Cu_Ni.svg" title="Agrandir"></a></div>Diagramme de phases Cu-Ni</div></div></div>
<p>D'autres couples de métaux présentent une bonne miscibilité permettant d’obtenir des solutions solides homogènes à certaines températures : <a href="/wiki/Cuivre" title="Cuivre">cuivre</a>-<a href="/wiki/Nickel" title="Nickel">nickel</a>, cuivre-<a href="/wiki/Palladium" title="Palladium">palladium</a>, <a href="/wiki/Argent" title="Argent">argent</a>-<a href="/wiki/Or" title="Or">or</a>, argent-palladium, <a href="/wiki/Molybd%C3%A8ne" title="Molybdène">molybdène</a>-<a href="/wiki/Vanadium" title="Vanadium">vanadium</a>, molybdène-<a href="/wiki/Tungst%C3%A8ne" title="Tungstène">tungstène</a>, etc.
</p><p>Certains alliages binaires présentent un défaut de solubilité à basses températures. Il apparaît sur le diagramme de phase un secteur où cohabitent deux phases, la première étant constituée d'une solution solide saturée de B dans A, et inversement la deuxième phase étant constituée d'une solution solide saturée de A dans B. C'est le cas par exemple du système cuivre-nickel qui présente en dessous de <span title="611,6 °F ou 595,2 K">322 </span><abbr class="abbr" title="degré Celsius">°C</abbr> une zone avec deux phases.
</p>
<h3><span id="Alliage_binaire_.C3.A0_plusieurs_phases"></span><span class="mw-headline" id="Alliage_binaire_à_plusieurs_phases">Alliage binaire à plusieurs phases</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=4" title="Modifier la section : Alliage binaire à plusieurs phases">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=4" title="Modifier la section : Alliage binaire à plusieurs phases">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Un autre type de diagramme de phases relativement commun dans les alliages binaires est un diagramme de phase eutectique. Ce type de diagramme de phase présente un certain nombre de caractéristiques importantes méritant d'être signalées. Tout d'abord, il existe trois régions monophasées qui sont visibles sur le schéma : α, β et liquide. Sur la figure à gauche, la phase α est une solution solide riche en cuivre, qui a pour soluté l'argent et présente une maille de structure CFC (<a href="/wiki/Cubique_%C3%A0_faces_centr%C3%A9es" title="Cubique à faces centrées">cubique à faces centrées</a>). La phase β (solution solide) est également caractérisée par une structure CFC, mais pour laquelle le cuivre est le soluté.
</p>
<div class="thumb tleft"><div class="thumbinner" style="width:332px;"><a class="image" href="/wiki/Fichier:Cooper-Silver-diagram.JPG"><img alt="Diagramme Cuivre Argent" class="thumbimage" data-file-height="831" data-file-width="882" decoding="async" height="311" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/86/Cooper-Silver-diagram.JPG/330px-Cooper-Silver-diagram.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/86/Cooper-Silver-diagram.JPG/495px-Cooper-Silver-diagram.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/86/Cooper-Silver-diagram.JPG/660px-Cooper-Silver-diagram.JPG 2x" width="330"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/Fichier:Cooper-Silver-diagram.JPG" title="Agrandir"></a></div>Diagramme de phases Cu-Ag</div></div></div>
<p>Ainsi, la solubilité de l'élément d'addition dans chacune de ces phases solides est limitée. Autrement dit la concentration d'argent qui peut se dissoudre dans le cuivre (pour la phase α) sans en modifier la structure cristallographique est limitée. Pour la même raison l'addition de cuivre dans l'argent (phase β) est limitée. La limite de solubilité de la phase α correspond à la ligne de démarcation, marquée « CBA ».
</p><p>Pour des températures inférieures à <span title="1 434,2°F ou 1 052,2 K">779</span> <abbr class="abbr" title="degré Celsius">°C</abbr> (<span title="778,9°C ou 1 052 K">1 434</span> <abbr class="abbr" title="degré Fahrenheit">°F</abbr>), la ligne correspondant à la limite de solubilité solide qui sépare la région de la phase α et la région de coexistence des phases α + β est appelée une ligne de solvus. La frontière séparant la phase α et la région α + L est la ligne de <a href="/wiki/Solidus_(diagramme_de_phases)" title="Solidus (diagramme de phases)">solidus</a> (AB), tandis que la ligne séparant la région α + L et le domaine liquide est la ligne de <a href="/wiki/Liquidus" title="Liquidus">liquidus</a> (AE). Pour la partie riche en argent du diagramme de phases, trois lignes existent également : solvus (HG), solidus (GF) et liquidus (EF). La ligne horizontale BEG, qui est parallèle à l'axe des abscisses s'étend entre les maxima de solubilité respectifs des phases α et β. Elle est appelée palier <a href="/wiki/Eutectique" title="Eutectique">eutectique</a> et peut aussi être considérée comme une ligne de solidus, représentant la température la plus basse à laquelle une phase liquide peut exister à l'équilibre thermodynamique pour tout alliage de cuivre et d'argent.
</p><p>Il y a aussi trois régions de deux phases trouvées dans le système cuivre-argent. Comme l'argent est ajouté au cuivre, la température à laquelle les alliages deviennent totalement liquides diminue au long de la ligne liquidus (ligne AE); ainsi, la température de fusion du cuivre est réduite par l'ajout d'argent. C'est le même principe pour les alliages dont le composé majoritaire est l'argent : l'introduction de cuivre réduit la température de fusion complète au long de la ligne liquidus FE. Ces lignes liquidus répondent au point E sur le diagramme de phase, par le biais de qui passe également la ligne horizontale isotherme BEG. Point E est le point eutectique, qui est désigné par la composition CE et de la température TE; pour le système cuivre-argent, les valeurs de la CE et TE sont 71,9 wt% Ag et <span title="1 434,2°F ou 1 052,2 K">779</span> <abbr class="abbr" title="degré Celsius">°C</abbr> (<span title="778,9°C ou 1 052 K">1 434</span> <abbr class="abbr" title="degré Fahrenheit">°F</abbr>), respectivement.
</p><p>Une importante réaction a lieu dans un alliage de composition « CE » cependant elle change la température en passant par TE. Sur le refroidissement, une phase liquide est transformée en deux phases solides (α et β) à la température TE, la réaction inverse se produit sur l'échauffement. C'est ce qu'on appelle une réaction eutectique (eutectique signifie facilement fondu), et CE et TE représentent les composition et température eutectiques, respectivement. Souvent, la ligne solidus horizontale à TE est appelé isotherme eutectique. La réaction eutectique, sur le refroidissement, est similaire à la solidification des composants purs en ce que la réaction à terme à une température constante, ou isométriquement, à TE. Toutefois, le produit solide de la solidification eutectique est toujours deux phases solides, alors que pour un simple composant, une seule phase se forme. À cause de cette réaction eutectique, les diagrammes de phase similaires à ceux de la figure du diagramme Ag-Cu sont qualifiées de diagrammes de phase eutectiques.
</p><p>Dans la construction de diagrammes de phases binaires, il est important de comprendre qu'un ou au maximum deux phases peuvent être en équilibre dans une région de phase. Pour un système eutectique, trois phases (α, β et L) peuvent être en équilibre, mais seulement à points au long de la ligne eutectique<sup class="reference" id="cite_ref-7"><a href="#cite_note-7"><span class="cite_crochet">[</span>6<span class="cite_crochet">]</span></a></sup>.
</p><p>Il y a des milliers de combinaisons possibles pour diagrammes de phases avec plusieurs phases. Certaines des principales caractéristiques des diagrammes de phases comprennent points congrus, où une phase solide se transforme directement en liquide. Il y a aussi le point péritectoïde, pour lequel une phase solide se transforme en deux phases solides différentes de la phase solide initiale, lors du chauffage. À l'inverse, si la transformation a lieu lors du refroidissement, on parle de point eutectoïde.
</p><p>Un diagramme de phase complexe d'une grande importance technologique est celle de la fer-carbone système de moins de 7 % de carbone.
</p><p>L'axe des X d'un tel schéma correspond à la concentration variable du mélange. Comme les mélanges sont généralement loin d'affaiblir et leur densité en fonction de la température est généralement inconnu, la mesure préférée est la concentration molaire. Un schéma fondé sur le volume de mesure comme molarité serait déconseillé.
</p>
<h2><span class="mw-headline" id="Structure">Structure</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=5" title="Modifier la section : Structure">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=5" title="Modifier la section : Structure">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="Alliage_homog.C3.A8ne"></span><span class="mw-headline" id="Alliage_homogène">Alliage homogène</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=6" title="Modifier la section : Alliage homogène">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=6" title="Modifier la section : Alliage homogène">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Un élément d'addition qui forme une solution solide avec le métal de base peut être localisé soit entre les atomes de l'élément majoritaire (on parle alors d'« insertion »), soit à la place des atomes du métal majoritaire (on parle alors de « substitution »).
</p><p><a class="image" href="/wiki/Fichier:Alliage_solution_solide.png"><img alt="Alliage solution solide.png" data-file-height="111" data-file-width="241" decoding="async" height="111" src="//upload.wikimedia.org/wikipedia/commons/6/68/Alliage_solution_solide.png" width="241"/></a>
</p><p>Une substitution peut conduire, soit à un alliage <i>désordonné</i>, où les différents atomes occupent des positions aléatoires, soit à un alliage <i>ordonné</i>, où les atomes de différentes natures suivent une alternance régulière.
</p><p><a class="image" href="/wiki/Fichier:Alliage_ordonne_desordonne.png"><img alt="Alliage ordonne desordonne.png" data-file-height="94" data-file-width="248" decoding="async" height="94" src="//upload.wikimedia.org/wikipedia/commons/1/1c/Alliage_ordonne_desordonne.png" width="248"/></a>
</p>
<h3><span id="Alliage_h.C3.A9t.C3.A9rog.C3.A8ne"></span><span class="mw-headline" id="Alliage_hétérogène">Alliage hétérogène</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=7" title="Modifier la section : Alliage hétérogène">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=7" title="Modifier la section : Alliage hétérogène">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Lorsque la teneur en élément d'alliage augmente, on peut avoir formation de deux phases : une phase contenant peu d'éléments d'alliage, et une phase à forte teneur en éléments d'alliage. Les <a href="/wiki/Cristallite" title="Cristallite">cristallites</a> à forte teneur sont appelés « précipités ».
</p><p><a class="image" href="/wiki/Fichier:Alliage_precipite.png"><img alt="Alliage precipite.png" data-file-height="142" data-file-width="371" decoding="async" height="142" src="//upload.wikimedia.org/wikipedia/commons/8/86/Alliage_precipite.png" width="371"/></a>
</p><p>Les <a href="/wiki/Pr%C3%A9cipit%C3%A9" title="Précipité">précipités</a> sont souvent des alliages ordonnés, que l'on appelle « intermétalliques ». Les intermétalliques ainsi formés sont parfois par la suite étudiés en tant qu'alliages propres, comme un nouveau matériau, et on essaie d'en produire en tant que tel et non plus en tant que précipités.
</p>

<h4><span class="mw-headline" id="Alliages_de_fer">Alliages de <a href="/wiki/Fer" title="Fer">fer</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=10" title="Modifier la section : Alliages de fer">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=10" title="Modifier la section : Alliages de fer">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul><li><a href="/wiki/Acier" title="Acier">acier</a> : <a href="/wiki/Fer" title="Fer">fer</a> + <a href="/wiki/Carbone" title="Carbone">carbone</a> (&lt; 2,1 % en masse) + optionnellement <a href="/wiki/Nickel" title="Nickel">nickel</a>, <a href="/wiki/Chrome" title="Chrome">chrome</a>, <a href="/wiki/Molybd%C3%A8ne" title="Molybdène">molybdène</a> (&lt; 4 %)</li>
<li><a href="/wiki/Acier_inoxydable" title="Acier inoxydable">acier inoxydable</a> : fer + carbone (&lt; 2,1 % en masse) + nickel + chrome + optionnellement molybdène, <a href="/wiki/Vanadium" title="Vanadium">vanadium</a></li>
<li><a href="/wiki/Fonte_(m%C3%A9tallurgie)" title="Fonte (métallurgie)">fonte</a> : fer + carbone (&gt; 2 % en masse)</li></ul>
<h4><span class="mw-headline" id="Alliages_de_cuivre"><a href="/wiki/Alliage_de_cuivre" title="Alliage de cuivre">Alliages de cuivre</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=11" title="Modifier la section : Alliages de cuivre">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=11" title="Modifier la section : Alliages de cuivre">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<ul><li><a href="/wiki/Bronze" title="Bronze">bronze</a> : <a href="/wiki/Cuivre" title="Cuivre">cuivre</a> + <a href="/wiki/%C3%89tain" title="Étain">étain</a> ; l'« <a href="/wiki/Airain" title="Airain">airain</a> » est l'ancien nom du bronze</li>
<li><a href="/wiki/Laiton" title="Laiton">laiton</a> : cuivre + <a href="/wiki/Zinc" title="Zinc">zinc</a></li>
<li><a href="/wiki/Billon_(alliage)" title="Billon (alliage)">billon</a> : cuivre + <a href="/wiki/Argent" title="Argent">argent</a> ; utilisé principalement pour frapper des monnaies de faible valeur</li></ul>
<h4><span id="Alliages_d.27aluminium"></span><span class="mw-headline" id="Alliages_d'aluminium">Alliages d'<a href="/wiki/Aluminium" title="Aluminium">aluminium</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=12" title="Modifier la section : Alliages d'aluminium">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=12" title="Modifier la section : Alliages d'aluminium">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Ils sont aussi appelés <i>alliages légers</i> compte tenu de la <a href="/wiki/Masse_volumique#Métaux_et_alliages" title="Masse volumique">masse volumique</a> de l'aluminium comparée à celle des autres métaux.

<h3><span class="mw-headline" id="Alliages_moins_connus">Alliages moins connus</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=13" title="Modifier la section : Alliages moins connus">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=13" title="Modifier la section : Alliages moins connus">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li><a href="/wiki/Alliage_plomb-%C3%A9tain" title="Alliage plomb-étain">alliage plomb-étain</a> : pour la <a class="mw-redirect" href="/wiki/Brasure" title="Brasure">brasure</a></li>
<li><a href="/wiki/Amalgame_(m%C3%A9tallurgie)" title="Amalgame (métallurgie)">amalgame</a> : <a href="/wiki/Mercure_(chimie)" title="Mercure (chimie)">mercure</a> + un autre métal, par exemple <a href="/wiki/Or" title="Or">or</a> ou cuivre ; le terme désigne également un mélange de métaux utilisé pour les soins dentaires (« plombage »)</li>
<li><a href="/wiki/%C3%89lectrum" title="Électrum">électrum</a> : <a href="/wiki/Or" title="Or">or</a> + <a href="/wiki/Argent" title="Argent">argent</a></li>

<li><a href="/wiki/Monel" title="Monel">monel</a> (nom commercial) : nickel + cuivre</li>
<li><a href="/wiki/R%C3%A9gule" title="Régule">régule</a> : <a href="/wiki/%C3%89tain" title="Étain">étain</a> ou <a href="/wiki/Plomb" title="Plomb">plomb</a> + <a href="/wiki/Antimoine" title="Antimoine">antimoine</a></li>
<li><a href="/wiki/Ruolz" title="Ruolz">ruolz</a> : nickel + argent + cuivre</li>
<li><a href="/wiki/Tumbaga" title="Tumbaga">tumbaga</a> : alliage d'or et de cuivre utilisé par les civilisations précolombiennes d'<a href="/wiki/Am%C3%A9rique_du_Sud" title="Amérique du Sud">Amérique du Sud</a> et de <a href="/wiki/M%C3%A9soam%C3%A9rique" title="Mésoamérique">Mésoamérique</a>.</li>
<li><a href="/wiki/Virenium" title="Virenium">virenium</a> : <a href="/wiki/Cuivre" title="Cuivre">cuivre</a> + <a href="/wiki/Zinc" title="Zinc">zinc</a> + <a href="/wiki/Nickel" title="Nickel">nickel</a></li>
<li><a href="/wiki/Zamak" title="Zamak">zamak</a> : zinc + aluminium + magnésium + cuivre et autres « <a href="/wiki/D%C3%A9signation_des_m%C3%A9taux_et_alliages#Zinc_et_alliages_de_zinc" title="Désignation des métaux et alliages">ZL</a> » (composant principaux : <a href="/wiki/Zinc" title="Zinc">zinc</a> et <a href="/wiki/Aluminium" title="Aluminium">aluminium</a>)</li></ul>
<h3><span id="Alliages_pour_des_applications_sp.C3.A9cifiques"></span><span class="mw-headline" id="Alliages_pour_des_applications_spécifiques">Alliages pour des applications spécifiques</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Alliage&amp;veaction=edit&amp;section=14" title="Modifier la section : Alliages pour des applications spécifiques">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Alliage&amp;action=edit&amp;section=14" title="Modifier la section : Alliages pour des applications spécifiques">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h3>
<ul><li><a href="/wiki/Ferrotitane" title="Ferrotitane">ferrotitanes</a> : fer + 25 à 70 % de Ti + 4 à 10 % d'aluminium</li>
<li>TA6V : <a href="/wiki/Titane" title="Titane">titane</a> + 6 % <a href="/wiki/Aluminium" title="Aluminium">aluminium</a> + 4 % vanadium, très utilisé dans l'<a class="mw-redirect" href="/wiki/Industrie_a%C3%A9ronautique" title="Industrie aéronautique">industrie aéronautique</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8"><span class="cite_crochet">[</span>note 2<span class="cite_crochet">]</span></a></sup></li>
<li>MCrAl : métal + chrome + aluminium + parfois de l'<a href="/wiki/Yttrium" title="Yttrium">yttrium</a> (MCrAlY), alliages réputés pour leur bonne tenue mécanique et résistance à la <a href="/wiki/Corrosion_%C3%A0_haute_temp%C3%A9rature" title="Corrosion à haute température">corrosion à haute température</a>
<ul><li>FeCrAl : fer + chrome + aluminium</li>
<li>NiCrAl : nickel + chrome + aluminium</li></ul></li>
<li><a href="/wiki/Superalliage" title="Superalliage">superalliages</a> à base nickel (par exemple les <a href="/wiki/Inconel" title="Inconel">inconels</a>) : bonne tenue mécanique et résistance à la corrosion à haute température</li>
<li><a href="/wiki/Compos%C3%A9_interm%C3%A9tallique" title="Composé intermétallique">intermétalliques</a> : alliages ordonnés, respectant une <a href="/wiki/St%C5%93chiom%C3%A9trie" title="Stœchiométrie">stœchiométrie</a> précise (mais des écarts à la stœchiométrie sont tolérés)
<ul><li>NiAl β : 50 % nickel + 50 % aluminium</li>
<li>FeAl B2 : 50 % fer + 50 % aluminium</li>
<li>TiAl : 50 % titane + 50 % aluminium</li></ul></li></ul>
<ul><li>Les alliages présentant de faibles <a href="/wiki/Dilatation_thermique" title="Dilatation thermique">coefficients de dilatation</a> (créés par <a href="/wiki/Charles_%C3%89douard_Guillaume" title="Charles Édouard Guillaume">Charles Édouard Guillaume</a>)
<ul><li><a href="/wiki/Invar" title="Invar">Invar</a> (36 % de nickel, 0,4 % de manganèse, 0,1 % de carbone, 63,5 % de fer), dix fois moins dilatable que le fer</li>
<li><a href="/wiki/%C3%89linvar" title="Élinvar">Élinvar</a> (Nivarox, Métélinvar, Isoval) (nickel, chrome, fer)</li></ul></li></ul>
<ul><li><a href="/wiki/Kovar" title="Kovar">FeNiCo</a> un alliage 54 % fer + 29 % nickel + 17 % cobalt destiné au scellement verre/métal ou céramique/métal</li></ul>
