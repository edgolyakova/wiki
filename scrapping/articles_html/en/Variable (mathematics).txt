<h1 class="firstHeading mw-first-heading" id="firstHeading"><span class="mw-page-title-main">Variable (mathematics)</span></h1>
<div class="vector-body" id="bodyContent">
<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div id="contentSub2"></div>
<div id="jump-to-nav"></div>

<div class="mw-body-content mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><div class="shortdescription nomobile noexcerpt noprint searchaux" style="display:none">Symbol representing a mathematical object</div>
<style data-mw-deduplicate="TemplateStyles:r1033289096">.mw-parser-output .hatnote{font-style:italic}.mw-parser-output div.hatnote{padding-left:1.6em;margin-bottom:0.5em}.mw-parser-output .hatnote i{font-style:normal}.mw-parser-output .hatnote+link+.hatnote{margin-top:-0.5em}</style><div class="hatnote navigation-not-searchable" role="note">For variables in computer science, see <a href="/wiki/Variable_(computer_science)" title="Variable (computer science)">Variable (computer science)</a>. For other uses, see <a class="mw-redirect mw-disambig" href="/wiki/Variable_(disambiguation)" title="Variable (disambiguation)">Variable (disambiguation)</a>.</div>
<p>
In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, a <b>variable</b> (from <a class="mw-redirect" href="/wiki/Latin_language" title="Latin language">Latin</a> <i><a class="extiw" href="https://en.wiktionary.org/wiki/variabilis" title="wikt:variabilis">variabilis</a></i>, "changeable") is a <a class="mw-redirect" href="/wiki/Mathematical_symbol" title="Mathematical symbol">symbol</a> and placeholder for any <a href="/wiki/Mathematical_object" title="Mathematical object">mathematical object</a>. In particular, a variable may represent a <a href="/wiki/Number" title="Number">number</a>, a <a class="mw-redirect" href="/wiki/Vector_(mathematics)" title="Vector (mathematics)">vector</a>, a <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">matrix</a>, a <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a>, the <a href="/wiki/Argument_of_a_function" title="Argument of a function">argument of a function</a>, a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a>, or an <a href="/wiki/Element_(mathematics)" title="Element (mathematics)">element</a> of a set.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>
</p><p><a href="/wiki/Algebra#Algebra_as_a_branch_of_mathematics" title="Algebra">Algebraic computations</a> with variables as if they were explicit numbers solve a range of problems in a single computation. For example, the <a href="/wiki/Quadratic_formula" title="Quadratic formula">quadratic formula</a> solves any <a href="/wiki/Quadratic_equation" title="Quadratic equation">quadratic equation</a> by substituting the numeric values of the coefficients of that <a href="/wiki/Equation" title="Equation">equation</a> for the variables that represent them in the quadratic formula. In <a href="/wiki/Mathematical_logic" title="Mathematical logic">mathematical logic</a>, a <i>variable</i> is either a symbol representing an unspecified <a href="/wiki/Term_(logic)" title="Term (logic)">term</a> of the theory (a <a href="/wiki/Metavariable" title="Metavariable">meta-variable</a>), or a basic object of the theory that is manipulated without referring to its possible intuitive interpretation.
</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Variable_(mathematics)&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In ancient works such as <a href="/wiki/Euclid" title="Euclid">Euclid</a>'s <a href="/wiki/Euclid%27s_Elements" title="Euclid's Elements"><i>Elements</i></a>, single letters refer to geometric points and shapes. In the 7th century, <a href="/wiki/Brahmagupta" title="Brahmagupta">Brahmagupta</a> used different colours to represent the unknowns in algebraic equations in the <i><a href="/wiki/Br%C4%81hmasphu%E1%B9%ADasiddh%C4%81nta" title="Brāhmasphuṭasiddhānta">Brāhmasphuṭasiddhānta</a></i>. One section of this book is called "Equations of Several Colours".<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>
</p><p>At the end of the 16th century, <a href="/wiki/Fran%C3%A7ois_Vi%C3%A8te" title="François Viète">François Viète</a> introduced the idea of representing known and unknown numbers by letters, nowadays called variables, and the idea of computing with them as if they were numbers—in order to obtain the result by a simple replacement. Viète's convention was to use consonants for known values, and vowels for unknowns.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>
</p><p>In 1637, <a href="/wiki/Ren%C3%A9_Descartes" title="René Descartes">René Descartes</a> "invented the convention of representing unknowns in equations by <i>x</i>, <i>y</i>, and <i>z</i>, and knowns by <i>a</i>, <i>b</i>, and <i>c</i>".<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Contrarily to Viète's convention, Descartes' is still commonly in use. The history of the letter x in math was discussed in a 1887 <a href="/wiki/Scientific_American" title="Scientific American">Scientific American</a> article.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>
</p><p>Starting in the 1660s, <a href="/wiki/Isaac_Newton" title="Isaac Newton">Isaac Newton</a> and <a href="/wiki/Gottfried_Wilhelm_Leibniz" title="Gottfried Wilhelm Leibniz">Gottfried Wilhelm Leibniz</a> independently developed the <a class="mw-redirect" href="/wiki/Infinitesimal_calculus" title="Infinitesimal calculus">infinitesimal calculus</a>, which essentially consists of studying how an <a href="/wiki/Infinitesimal" title="Infinitesimal">infinitesimal</a> variation of a <i>variable quantity</i> induces a corresponding variation of another quantity which is a <i><a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a></i> of the first variable. Almost a century later, <a href="/wiki/Leonhard_Euler" title="Leonhard Euler">Leonhard Euler</a> fixed the terminology of infinitesimal calculus, and introduced the notation <span class="texhtml"><i>y</i> = <i>f</i>(<i>x</i>)</span> for a function <span class="texhtml"><i>f</i></span>, its <b>variable</b> <span class="texhtml"><i>x</i></span> and its value <span class="texhtml"><i>y</i></span>. Until the end of the 19th century, the word <i>variable</i> referred almost exclusively to the <a href="/wiki/Argument_of_a_function" title="Argument of a function">arguments</a> and the <a href="/wiki/Value_(mathematics)" title="Value (mathematics)">values</a> of functions.
</p><p>In the second half of the 19th century, it appeared that the foundation of infinitesimal calculus was not formalized enough to deal with apparent paradoxes such as a nowhere <a href="/wiki/Differentiable_function" title="Differentiable function">differentiable</a> <a href="/wiki/Continuous_function" title="Continuous function">continuous function</a>. To solve this problem, <a href="/wiki/Karl_Weierstrass" title="Karl Weierstrass">Karl Weierstrass</a> introduced a new formalism consisting of replacing the intuitive notion of <a href="/wiki/Limit_(mathematics)" title="Limit (mathematics)">limit</a> by a formal definition. The older notion of limit was "when the <i>variable</i> <span class="texhtml"><i>x</i></span> varies and tends toward <span class="texhtml"><i>a</i></span>, then <span class="texhtml"><i>f</i>(<i>x</i>)</span> tends toward <span class="texhtml"><i>L</i></span>", without any accurate definition of "tends". Weierstrass replaced this sentence by the formula

</p><p>This static formulation led to the modern notion of variable, which is simply a symbol representing a <a href="/wiki/Mathematical_object" title="Mathematical object">mathematical object</a> that either is unknown, or may be replaced by any element of a given <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> (e.g., the set of <a href="/wiki/Real_number" title="Real number">real numbers</a>).
</p>
<h2><span class="mw-headline" id="Notation">Notation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Variable_(mathematics)&amp;action=edit&amp;section=2" title="Edit section: Notation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Variables are generally denoted by a single letter, most often from the <a href="/wiki/Latin_alphabet" title="Latin alphabet">Latin alphabet</a> and less often from the <a href="/wiki/Greek_alphabet" title="Greek alphabet">Greek</a>, which may be lowercase or capitalized. The letter may be followed by a subscript: a number (as in <span class="texhtml"><i>x</i><sub>2</sub></span>), another variable (<span class="texhtml"><i>x</i><sub><i>i</i></sub></span>), a word or abbreviation of a word (<span class="texhtml"><i>x</i><sub>total</sub></span>) or a <a class="mw-redirect" href="/wiki/Mathematical_expression" title="Mathematical expression">mathematical expression</a> (<span class="texhtml"><i>x</i><sub>2<i>i</i> + 1</sub></span>). Under the influence of <a href="/wiki/Variable_(computer_science)" title="Variable (computer science)">computer science</a>, some variable names in pure mathematics consist of several letters and digits. Following <a href="/wiki/Ren%C3%A9_Descartes" title="René Descartes">René Descartes</a> (1596–1650), letters at the beginning of the alphabet such as <i>a</i>, <i>b</i>, <i>c</i> are commonly used for known values and parameters, and letters at the end of the alphabet such as (<i>x</i>, <i>y</i>, <i>z</i>) are commonly used for unknowns and variables of functions.<sup class="reference" id="cite_ref-E004_6-0"><a href="#cite_note-E004-6">[6]</a></sup> In printed mathematics, the norm is to set variables and constants in an italic typeface.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>

</p><p>Specific branches and applications of mathematics have specific <a class="mw-redirect" href="/wiki/Naming_conventions" title="Naming conventions">naming conventions</a> for variables. Variables with similar roles or meanings are often assigned consecutive letters or the same letter with different subscripts. For example, the three axes in 3D <a class="mw-redirect" href="/wiki/Coordinate_space" title="Coordinate space">coordinate space</a> are conventionally called <i>x</i>, <i>y</i>, and <i>z</i>. In physics, the names of variables are largely determined by the <a href="/wiki/Physical_quantity" title="Physical quantity">physical quantity</a> they describe, but various naming conventions exist. A convention often followed in <a href="/wiki/Probability" title="Probability">probability</a> and <a href="/wiki/Statistics" title="Statistics">statistics</a> is to use <i>X</i>, <i>Y</i>, <i>Z</i> for the names of <a href="/wiki/Random_variable" title="Random variable">random variables</a>, keeping <i>x</i>, <i>y</i>, <i>z</i> for variables representing corresponding better-defined values.
</p>

<p>is interpreted as having five variables: four, <span class="texhtml"><i>a</i>, <i>b</i>, <i>c</i>, <i>d</i></span>, which are taken to be given numbers and the fifth variable, <span class="texhtml"><i>x</i>,</span> is understood to be an <i>unknown</i> number. To distinguish them, the variable <span class="texhtml"><i>x</i></span> is called <i>an unknown</i>, and the other variables are called <i>parameters</i> or <i><a href="/wiki/Coefficient" title="Coefficient">coefficients</a></i>, or sometimes <i>constants</i>, although this last terminology is incorrect for an equation, and should be reserved for the <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> defined by the left-hand side of this equation.
</p><p>In the context of functions, the term <i>variable</i> refers commonly to the arguments of the functions. This is typically the case in sentences like "<a href="/wiki/Function_of_a_real_variable" title="Function of a real variable">function of a real variable</a>", "<span class="texhtml"><i>x</i></span> is the variable of the function <span class="texhtml"><i>f</i>: <i>x</i> ↦ <i>f</i>(<i>x</i>)</span>", "<span class="texhtml"><i>f</i></span> is a function of the variable <span class="texhtml"><i>x</i></span>" (meaning that the argument of the function is referred to by the variable <span class="texhtml"><i>x</i></span>).
</p><p>In the same context, variables that are independent of <span class="texhtml"><i>x</i></span> define <a href="/wiki/Constant_function" title="Constant function">constant functions</a> and are therefore called <i>constant</i>. For example, a <i><a href="/wiki/Constant_of_integration" title="Constant of integration">constant of integration</a></i> is an arbitrary constant function that is added to a particular <a href="/wiki/Antiderivative" title="Antiderivative">antiderivative</a> to obtain the other antiderivatives. Because the strong relationship between <a href="/wiki/Polynomial" title="Polynomial">polynomials</a> and <a class="mw-redirect" href="/wiki/Polynomial_function" title="Polynomial function">polynomial function</a>, the term "constant" is often used to denote the coefficients of a polynomial, which are constant functions of the indeterminates.
</p><p>This use of "constant" as an abbreviation of "constant function" must be distinguished from the normal meaning of the word in mathematics. A <b>constant</b>, or <b><a href="/wiki/Mathematical_constant" title="Mathematical constant">mathematical constant</a></b> is a well and unambiguously defined number or other mathematical object, as, for example, the numbers 0, 1, <span class="texhtml"><a href="/wiki/Pi" title="Pi"><i>π</i></a></span> and the <a href="/wiki/Identity_element" title="Identity element">identity element</a> of a <a href="/wiki/Group_(mathematics)" title="Group (mathematics)">group</a>. Since a variable may represent any mathematical object, a letter that represents a constant is often called a variable. This is, in particular, the case of <span class="texhtml mvar" style="font-style:italic;">e</span> and <span class="texhtml mvar" style="font-style:italic;">π</span>, even when they represents <a class="mw-redirect" href="/wiki/Euler%27s_number" title="Euler's number">Euler's number</a> and <span class="texhtml">3.14159...</span>
</p><p>Other specific names for variables are:
</p>
<ul><li>An <b>unknown</b> is a variable in an <a href="/wiki/Equation" title="Equation">equation</a> which has to be solved for.</li>
<li>An <b><a href="/wiki/Indeterminate_(variable)" title="Indeterminate (variable)">indeterminate</a></b> is a symbol, commonly called variable, that appears in a <a href="/wiki/Polynomial" title="Polynomial">polynomial</a> or a <a href="/wiki/Formal_power_series" title="Formal power series">formal power series</a>. Formally speaking, an indeterminate is not a variable, but a <a href="/wiki/Constant_(mathematics)" title="Constant (mathematics)">constant</a> in the <a href="/wiki/Polynomial_ring" title="Polynomial ring">polynomial ring</a> or the ring of <a href="/wiki/Formal_power_series" title="Formal power series">formal power series</a>. However, because of the strong relationship between polynomials or power series and the <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">functions</a> that they define, many authors consider indeterminates as a special kind of variables.</li>
<li>A <b><a href="/wiki/Parameter" title="Parameter">parameter</a></b> is a quantity (usually a number) which is a part of the input of a problem, and remains constant during the whole solution of this problem. For example, in <a href="/wiki/Mechanics" title="Mechanics">mechanics</a> the mass and the size of a solid body are <i>parameters</i> for the study of its movement. In <a href="/wiki/Computer_science" title="Computer science">computer science</a>, <i>parameter</i> has a different meaning and denotes an argument of a function.</li>
<li><b><a href="/wiki/Free_variables_and_bound_variables" title="Free variables and bound variables">Free variables and bound variables</a></b></li>
<li>A <b><a href="/wiki/Random_variable" title="Random variable">random variable</a></b> is a kind of variable that is used in <a href="/wiki/Probability_theory" title="Probability theory">probability theory</a> and its applications.</li></ul>
<p>All these denominations of variables are of <a href="/wiki/Semantics" title="Semantics">semantic</a> nature, and the way of computing with them (<a href="/wiki/Syntax_(logic)" title="Syntax (logic)">syntax</a>) is the same for all.
</p>
<h3><span class="mw-headline" id="Dependent_and_independent_variables">Dependent and independent variables</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Variable_(mathematics)&amp;action=edit&amp;section=4" title="Edit section: Dependent and independent variables">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<link href="mw-data:TemplateStyles:r1033289096" rel="mw-deduplicated-inline-style"/><div class="hatnote navigation-not-searchable" role="note">Main article: <a href="/wiki/Dependent_and_independent_variables" title="Dependent and independent variables">Dependent and independent variables</a></div>
<p>In <a href="/wiki/Calculus" title="Calculus">calculus</a> and its application to <a href="/wiki/Physics" title="Physics">physics</a> and other sciences, it is rather common to consider a variable, say <span class="texhtml"><i>y</i></span>, whose possible values depend on the value of another variable, say <span class="texhtml"><i>x</i></span>. In mathematical terms, the <i>dependent</i> variable <span class="texhtml"><i>y</i></span> represents the value of a <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> of <span class="texhtml"><i>x</i></span>. To simplify formulas, it is often useful to use the same symbol for the dependent variable <span class="texhtml"><i>y</i></span> and the function mapping <span class="texhtml"><i>x</i></span> onto <span class="texhtml"><i>y</i></span>. For example, the state of a physical system depends on measurable quantities such as the <a href="/wiki/Pressure" title="Pressure">pressure</a>, the <a href="/wiki/Temperature" title="Temperature">temperature</a>, the spatial position, ..., and all these quantities vary when the system evolves, that is, they are function of the time. In the formulas describing the system, these quantities are represented by variables which are dependent on the time, and thus considered implicitly as functions of the time.
</p><p>Therefore, in a formula, a <b>dependent variable</b> is a variable that is implicitly a function of another (or several other) variables. An <b>independent variable</b> is a variable that is not dependent.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>
</p><p>The property of a variable to be dependent or independent depends often of the point of view and is not intrinsic. For example, in the notation <span class="texhtml"><i>f</i>(<i>x</i>, <i>y</i>, <i>z</i>)</span>, the three variables may be all independent and the notation represents a function of three variables. On the other hand, if <span class="texhtml"><i>y</i></span> and <span class="texhtml"><i>z</i></span> depend on <span class="texhtml"><i>x</i></span> (are <i>dependent variables</i>) then the notation represents a function of the single <i>independent variable</i> <span class="texhtml"><i>x</i></span>.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>
</p>

</math></span><img alt="f(x)" aria-hidden="true" class="mwe-math-fallback-image-inline" src="https://wikimedia.org/api/rest_v1/media/math/render/svg/202945cce41ecebb6f643f31d119c514bec7a074" style="vertical-align: -0.838ex; width:4.418ex; height:2.843ex;"/></span>)</li>
<li><i>i</i> for the <a href="/wiki/Imaginary_unit" title="Imaginary unit">imaginary unit</a></li>
<li><i>i</i>, <i>j</i>, <i>k</i> (sometimes <i>l</i> or <i>h</i>) for varying <a class="mw-redirect" href="/wiki/Integers" title="Integers">integers</a> or indices in an <a href="/wiki/Indexed_family" title="Indexed family">indexed family</a>, or <a class="mw-redirect" href="/wiki/Unit_vectors" title="Unit vectors">unit vectors</a></li>
<li><i>l</i> and <i>w</i> for the length and width of a figure</li>
<li><i>l</i>  also for a line, or in number theory for a prime number not equal to <i>p</i></li>
<li><i>n</i> (with <i>m</i> as a second choice)  for a fixed integer, such as a count of objects or the <a class="mw-redirect mw-disambig" href="/wiki/Degree_(disambiguation)" title="Degree (disambiguation)">degree</a> of an <a href="/wiki/Equation" title="Equation">equation</a></li>
<li><i>p</i> for a <a href="/wiki/Prime_number" title="Prime number">prime number</a> or a <a href="/wiki/Probability" title="Probability">probability</a></li>
<li><i>q</i> for a <a href="/wiki/Prime_power" title="Prime power">prime power</a> or a <a href="/wiki/Quotient" title="Quotient">quotient</a></li>
<li><i>r</i> for a <a href="/wiki/Radius" title="Radius">radius</a>, a <a href="/wiki/Remainder" title="Remainder">remainder</a> or a <a href="/wiki/Correlation_coefficient" title="Correlation coefficient">correlation coefficient</a></li>
<li><i>t</i> for <a href="/wiki/Time" title="Time">time</a></li>
<li><i>x</i>, <i>y</i>, <i>z</i> for the three <a class="mw-redirect" href="/wiki/Cartesian_coordinates" title="Cartesian coordinates">Cartesian coordinates</a> of a point in <a href="/wiki/Euclidean_geometry" title="Euclidean geometry">Euclidean geometry</a> or the corresponding <a class="mw-redirect" href="/wiki/Axis_(mathematics)" title="Axis (mathematics)">axes</a></li>
<li><i>z</i> for a <a href="/wiki/Complex_number" title="Complex number">complex number</a>, or in statistics a <a href="/wiki/Normal_distribution" title="Normal distribution">normal random variable</a></li>
<li><i>α</i>, <i>β</i>, <i>γ</i>, <i>θ</i>, <i>φ</i> for <a class="mw-redirect" href="/wiki/Angle_measure" title="Angle measure">angle measures</a></li>
<li><i>ε</i> (with <i>δ</i> as a second choice) for an arbitrarily small positive number</li>
<li><i>λ</i> for an <a class="mw-redirect" href="/wiki/Eigenvalue" title="Eigenvalue">eigenvalue</a></li>
<li>Σ (capital sigma) for a sum, or σ (lowercase sigma) in statistics for the <a href="/wiki/Standard_deviation" title="Standard deviation">standard deviation</a><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup></li>
<li><i>μ</i> for a <a href="/wiki/Mean" title="Mean">mean</a></li></ul>
